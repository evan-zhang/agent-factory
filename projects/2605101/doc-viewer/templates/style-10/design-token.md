# Style 10 — Stripe（开发者平台风）

## 风格定位
Stripe 风格，Fintech、开发者优先、优雅渐变、分层卡片设计。

## 核心气质
- Developer-first, elegant gradients
- 分层卡片布局
- API 风格区块
- 优雅渐变（暖橙色系）
- 大圆角、柔和阴影
- 专业且现代

## 颜色体系（暖色渐变）

| Token | 值 | 用途 |
|-------|-----|------|
| `--bg` | `#FFFFFF` | 主背景（纯白） |
| `--bg-alt` | `#F5F0EB` | 次背景（暖白） |
| `--bg-dark` | `#1E1715` | 深色区背景（暖棕黑） |
| `--accent` | `#C97842` | 强调色（暖橙） |
| `--accent-light` | `#D98B52` | 次强调色（暖橙浅） |
| `--gradient-start` | `#C97842` | 渐变起始色 |
| `--gradient-end` | `#D98B52` | 渐变结束色 |
| `--text` | `#1A1918` | 主文字（深灰棕） |
| `--text-secondary` | `#9C8F84` | 次文字（暖灰棕） |
| `--border` | `#E6E0DA` | 边框色 |

## 字体

中文："PingFang SC", -apple-system, "MiSans", sans-serif
英文：-apple-system, "Inter", "SF Pro Text", sans-serif

标题：font-weight 700/800，大字号
正文：font-size 14-15pt
代码：monospace, "Menlo", "Consolas", monospace

## 布局特征
- 分层卡片（大圆角 16-24px）
- 优雅渐变背景（橙色系）
- 功能卡片网格布局
- 深色代码展示区（#1E1715）
- 柔和阴影（box-shadow: 0 12px 40px rgba(0,0,0,0.08)）
- 充足间距（padding: 80-100px）

## 元素样式
- 卡片：大圆角、柔和阴影、悬停上浮效果
- 代码块：深色背景、暖色语法高亮
- 渐变：从 #C97842 到 #D98B52（暖橙渐变）
- 按钮：渐变背景、大圆角、悬停效果
- Hero 区：大标题 + 渐变背景

## 适用场景
- 开发者平台
- API 文档
- 技术产品官网
- 开源项目
- 金融科技产品
