# Style 07 — Warm Minimal Editorial（日系暖色极简插画风）

## 风格定位
AI 创业产品官网风格，混合 Claude / Arc / Notion / Linear 的视觉气质。

## 核心气质
- 温暖、极简、高级、柔和
- 编辑部风格
- 轻插画、设计师产品感
- AI 工具宣传页风格

## 与其他风格的区别
- 非 Google Cloud 那种企业级数据报告风
- 更像 AI 创业产品官网（Claude / Arc / Perplexity / Notion AI / Cursor）
- 更偏设计感、创作者工具、更轻更年轻

## 颜色体系（低饱和暖色系）

| Token | 值 | 用途 |
|-------|-----|------|
| `--bg` | `#EFE4D6` | 主背景（米白/奶咖） |
| `--bg-alt` | `#E6D9CA` | 次背景（羊皮纸） |
| `--bg-card` | `#F7EFE7` | 卡片背景 |
| `--accent` | `#C97842` | 强调色（暖橙） |
| `--accent-light` | `#D98B52` | 次强调色 |
| `--text` | `#1E1715` | 主文字（棕黑，非纯黑） |
| `--text-secondary` | `#2B211D` | 次文字 |
| `--muted` | `#9C8F84` | 辅助灰 |
| `--border` | `#C8BCB2` | 边框色 |
| `--border-light` | `#D9D0C6` | 轻边框 |

## 字体

中文："MiSans", "PingFang SC", "HarmonyOS Sans SC", sans-serif
英文："Inter", "SF Pro Display", "Helvetica Neue", sans-serif

标题：粗重（font-weight: 800）、大字号、letter-spacing: -2px
正文：极简、轻量
标签：全大写、letter-spacing: 2px、font-weight: 700

## 布局特征
- 超大留白（padding: 80px 120px）
- 左右分区（左文案、右插画/UI mockup）
- 视觉重心偏右
- 柔和阴影（box-shadow: 0 10px 30px rgba(0,0,0,0.05)）
- 大圆角（border-radius: 24px）

## 插画风格
- 半立体极简 UI 插画
- 柔和阴影、低对比、大圆角、几何体
- 避免：科技线、粒子、发光

## 适用场景
- AI 产品官网 / 宣传页
- AI Agent 产品介绍
- 创作者工具页面
- Markdown → HTML 工具展示页
- Landing Page
