第1章 执行摘要

本报告为极早期初步筛查报告。MB-001未在任何国家上市，A-1技能的"海外已上市"前提不满足。评估深度低于A-1标准要求，所有结论仅供初步筛查参考，不构成投资建议。

项目内容
产品名称MB-001（Mage Biologics）— 口服抗TNF-α单克隆抗体
当前最高研发阶段Phase 1/2（I期健康志愿者完成，I/II期UC患者启动中）C级-待验证
核心差异化全球首创口服抗TNF-α单抗 — 结肠靶向释放，最小全身暴露
目标适应症中重度溃疡性结肠炎（UC）
业务主体深康（消化肝病领域）
合作方Mage Biologics Inc.（SPC）/ Tillotts Pharma AG（Zeria Group子公司）
推荐结论附条件推进（Conditional Go）

核心结论

MB-001作为全球首创口服抗TNF-α单抗，差异化概念强（口服vs注射），但处于极早期（Phase 1/2），几乎所有评估维度均为低置信度。建议在I/II期PoC数据读出前，仅保持关注和初步接触。

主要风险

临床失败风险 — Phase 1/2极早期，尚未获得PoC数据。口服大分子抗体的肠道吸收和组织渗透存在技术不确定性。
竞争格局恶化 — 口服JAK抑制剂（乌帕替尼）已在中国上市，面临"口服选项之一"的风险。
毛利率不达标 — 进口模式毛利率29-49%，低于创新药≥60%门槛。本地化生产是改善关键。

关键决策节点

P0 近期（2026H2-2027）：I/II期PoC数据读出
P1 中期（2027-2028）：CDE Pre-IND沟通结果
P2 远期（2028+）：如PoC阳性，重新跑完整A-1评估

终局五问

1. 成功定义

MB-001在中国获批上市后第5年达到峰值销售额4-8亿元人民币，成为中国中重度UC口服生物制剂的首选/次选治疗方案。

2. 核心价值主张

全球首创口服抗TNF-α单抗 — 靶向结肠炎症黏膜释放，最小化全身暴露，为UC患者提供首个口服生物制剂选择。

3. 最大风险

临床失败风险 — MB-001处于Phase 1/2极早期，尚未在任何适应症获得PoC数据。I/II期UC患者试验若未能证明疗效信号，项目价值归零。

4. 止损边界

  • 注册/临床失败：首付款控制在前期投入<500万美元，里程碑与临床进展绑定
  • I/II期未达主要终点：终止后续里程碑支付，退出项目
  • 中国注册路径不可行（需独立III期且成本>$50M）：暂缓推进

5. 决策人与时限

阶段A 决策：投前总监 + 深康消化领域负责人
阶段B/C 决策：投前委员会 + CFO
最晚决策时限：I/II期PoC数据读出后6个月内

异议详情

审查层异议 1

Gate 2 临床证据评分1.65/5仍给出"有条件通过"，标准过于宽松

A-1技能定义中Gate 2通过标准为"临床评分≥3.0/5"。MB-001评分为1.65/5，远低于标准。虽然报告中注明"产品处于极早期，非产品缺陷"，但体系规则是"Gate即止损"，不应以"潜力"代替"证据"。

建议：将Gate 2结论改为"未达标——等待I/II期PoC数据后重新评估"。

执行层回应

接受。同意审查层判断。Gate 2结论从"有条件通过"改为"未达标——等待I/II期PoC数据后重新评估"。1.65/5远低于A-1标准门槛3.0/5，按照"Gate即止损"原则，本Gate应判定为未达标。

审查层异议 2

Gate 5 毛利率未达60%门槛但仅标注"需改善"，未执行止损

体系总规则§5明确规定"毛利率低于本体系第5节硬编码下限时，当轮停止，不允许以'战略意义'为由豁免"。

执行层回应

接受。Gate 5结论从"继续推进至Gate 6"改为"未达标——暂停推进，待本地化生产方案确认后重新评估"。

审查层异议 3

Gate 6财务模型几乎全部为D级估算，无法支持投资决策

在如此低置信度下给出"IRR 15-25%"的方向性估算可能误导投委会决策。

执行层回应

部分接受。同意删除具体IRR区间数字，改为"现阶段无法可靠估算财务回报"。但保留三情景框架和假设说明以助后续快速迭代。
