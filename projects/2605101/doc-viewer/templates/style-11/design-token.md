# Style 11 — Linear Dark（暗色极简风）

## 风格定位
Linear 风格，暗色极简、精准间距、微妙渐变、精英软件气质。

## 核心气质
- Dark minimalism, elite software
- 暗色主题但不用蓝紫
- 精准间距
- 微妙暗色渐变
- 高对比度文字（暖白）
- 极简边框

## 颜色体系（暖色暗色）

| Token | 值 | 用途 |
|-------|-----|------|
| `--bg` | `#08090A` | 主背景（极深黑） |
| `--bg-alt` | `#191B1F` | 次背景（深灰） |
| `--bg-card` | `#121416` | 卡片背景（中灰） |
| `--accent` | `#C97842` | 强调色（暖橙，替代蓝紫） |
| `--text` | `#E8E4DF` | 主文字（暖白） |
| `--text-secondary` | `#6B5E54` | 次文字（暖灰棕） |
| `--border` | `#2A2520` | 边框色（深暖灰棕） |
| `--shadow` | `rgba(0,0,0,0.4)` | 阴影 |

## 字体

中文："PingFang SC", -apple-system, "MiSans", sans-serif
英文：-apple-system, "Inter", "SF Pro Text", sans-serif

标题：font-weight 700/800，精准字号
正文：font-size 14pt，line-height 1.6
代码：monospace, "Menlo", "Consolas", monospace

## 布局特征
- 暗色主题（#08090A, #191B1F）
- 精准间距（Linear 的关键特征）
- 极简边框（border: 1px solid #2A2520）
- 微妙的暗色渐变（不用蓝色/紫色）
- 卡片小圆角（8-12px）
- 高对比度文字（#E8E4DF 暖白）

## 元素样式
- 卡片：深色背景、极简边框、微妙阴影
- 强调元素：暖橙（#C97842）
- 按钮：圆角、悬停效果
- 分隔线：细线、深灰色
- 数据指标：清晰排版

## 适用场景
- 开发者工具
- SaaS 产品
- 技术平台
- 暗色主题界面
- 精英软件产品
