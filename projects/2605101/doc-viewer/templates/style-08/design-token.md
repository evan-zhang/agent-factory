# Style 08 — Apple Minimal（苹果极简风）

## 风格定位
苹果发布会 Keynote 风格，极致简约、电影感、高端产品展示。

## 核心气质
- Minimal, premium, cinematic
- 超大留白、巨型排版
- 电影感产品展示区
- 极简边框、几乎无装饰
- 高端质感、注意力聚焦

## 颜色体系

| Token | 值 | 用途 |
|-------|-----|------|
| `--bg` | `#FFFFFF` | 主背景（纯白） |
| `--bg-alt` | `#F5F5F7` | 次背景（苹果浅灰） |
| `--accent` | `#C97842` | 强调色（暖橙，替代苹果蓝） |
| `--accent-hover` | `#D98B52` | 强调色悬停态 |
| `--text` | `#1D1D1F` | 主文字（非纯黑） |
| `--text-secondary` | `#86868B` | 次文字（辅助灰） |
| `--border` | `#D2D2D7` | 边框色 |
| `--shadow` | `rgba(0,0,0,0.04)` | 柔和阴影 |

## 字体

中文："SF Pro Display", "PingFang SC", -apple-system, sans-serif
英文："SF Pro Display", -apple-system, "Helvetica Neue", sans-serif

标题：font-weight 800, 巨型字号（48-72pt），letter-spacing: -1.5px
副标题：font-weight 600, 中等字号
正文：font-weight 400/500，轻量极简

## 布局特征
- 超大留白（padding: 120px 160px）
- 巨型排版（标题 60-72pt）
- 电影感 Hero 区域（全屏大图展示）
- 居中或左右对称布局
- 极简边框，border-radius: 12-20px
- 微妙阴影（box-shadow: 0 4px 20px rgba(0,0,0,0.04)）

## 适用场景
- 产品发布会页面
- 高端产品官网
- 影片式产品展示
- 品牌宣传页
- 简洁的产品介绍
