## 一句话定位

将复杂的业务知识体系，转化为一套结构清晰、多人协作可用的方案报告。

## 核心方法论

五阶段工作流 + 五个批判性节点 + Best-Minds 专家思维协议

### 五阶段
Phase 0: 理解吸收 → Phase 1: 框架搭建 → Phase 2: 逐章起草 → Phase 3: 审核迭代 → Phase 4: 合并交付

### 六个批判性节点
🔴 CP1: 认知纠偏 → 🔴 CP2: 框架批判 → 🔴 CP3: 目录预审 → 🔴 CP4: 初稿自判 → 🔴 CP5: 修订回溯 → 🔴 CP6: 交付终审

## 实战来源

从 xmt-sop 项目（院外体系 1+6+2+3 框架）提炼，该项目产出 13 卷、约 17000 行 Markdown。
