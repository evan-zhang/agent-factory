# 项目 2605121 — html-page

## 概述

通用型内容展示页生成器。根据用户提供的内容和素材，生成风格化的 HTML 单页面，自动上传 Doc Viewer 并返回预览链接。

## 用途

- 文章/观点展示宣传页
- 企业报告/白皮书展示页
- 数据可视化展示页
- 活动宣传页

## 技术栈

- HTML + TailwindCSS（CDN）
- 纯前端动效（Intersection Observer + CSS transition）
- 交付：单文件 HTML → Doc Viewer 上传 → 预览链接
