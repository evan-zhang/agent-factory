# 故事生成框架

## 用途
在直接模式 Step 2 中，基于结构化概念（来自 concept-template.md），按此框架生成 1500-2500 字的隐喻故事。

---

## 1. 世界观选择原则

故事需要一个自洽的虚构世界，概念隐藏在世界的运行规则中。

### 可选世界观类型

| 类型 | 适合的概念类型 | 示例 |
|------|---------------|------|
| 封闭系统 | 系统论、控制论、均衡类 | 一个与世隔绝的岛 |
| 未来社会 | 社会学、经济学、制度类 | 22 世纪的人类聚落 |
| 异星文明 | 生物学、演化论、生态类 | 外星物种的社会组织 |
| 虚拟世界 | 信息论、计算机科学类 | 生活在代码中的意识 |
| 职场/组织 | 管理学、组织行为学类 | 一个特殊的企业/团队 |
| 奇幻规则 | 物理学、数学、逻辑类 | 有特殊物理法则的世界 |

### 选择标准
- 世界观必须能自然地嵌入概念的运行机制
- 不选择过于常见的设定（如"魔法学校"）
- 世界观本身要有新鲜感，能独立吸引读者

---

## 2. 角色构造方法

### 角色类型

| 角色 | 功能 | 设计要点 |
|------|------|----------|
| 探索者 | 代替读者发现规则 | 好奇、有逻辑、不完全理解系统 |
| 守护者 | 维护现有规则 | 知道部分真相、有立场、不全是反派 |
| 变异者 | 打破规则的人/物 | 代表概念中的异常或边界条件 |
| 系统 | 世界本身的运行逻辑 | 通过结果和反馈展现，不是角色 |

### 角色设计约束
- 每个故事 2-4 个角色（含系统）
- 角色不需要名字，可以用代号或身份
- 角色行为必须服务于概念的展现
- 避免脸谱化：守护者也可以有善意，变异者也可以有私心

---

## 3. 规则设计

### 核心原则
- 世界中存在 1-3 条明确的运行规则
- 规则是概念机制的"翻译"
- 规则必须在故事早期建立，让读者可以验证

### 规则暴露节奏
1. **前 1/4**：通过角色行为暗示规则存在
2. **中段**：规则被明确表述（但概念名仍隐藏）
3. **后段**：规则被挑战或出现意外
4. **结尾**：揭示规则背后的真正概念

### 规则设计检查清单
- [ ] 规则是概念的直接隐喻吗？
- [ ] 规则足够具体，读者能在脑中模拟吗？
- [ ] 规则不会提前暴露概念名吗？
- [ ] 违反规则的后果是否明确？

---

## 4. 系统失控设计

### 为什么要"失控"
好的隐喻故事需要一个"系统被推到极限"的时刻，这对应概念中边界条件、反例或极端情况。

### 失控方式

| 方式 | 适合的概念 | 描述 |
|------|-----------|------|
| 正反馈爆炸 | 反馈回路类 | 小变化导致连锁放大 |
| 规则冲突 | 博弈/均衡类 | 多条规则同时作用产生矛盾 |
| 临界突破 | 相变/阈值类 | 超过临界点后质变 |
| 信息崩溃 | 信息论/熵类 | 信息丢失或过载导致系统失效 |
| 演化分叉 | 演化/涌现类 | 系统走向不可预测的分支 |

### 失控场景要求
- 必须让读者感到"这不对劲"
- 失控是故事的高潮
- 失控后必须有解释（对应概念的边界条件）

---

## 5. 揭晓方式

### 标准揭晓结构

```
[故事高潮结束]

[一个短暂的沉默/空白]

---

[概念名称]

[核心定义]

[隐喻映射说明]

[理论对应分析]
```

### 揭晓设计要点
1. 揭晓前要有一个"顿悟时刻"——故事中某个角色或读者自己发现规律
2. 概念名出现在独立段落，加粗或特殊排版
3. 揭晓后不要急于解释，先让读者消化 2-3 秒（页面留白）
4. 解释分层展开：先说"是什么"，再说"刚才故事里哪对应了"

---

## 6. 三层解释框架

在揭晓概念后，使用 interpretation-template.md 的框架进行分层解释：

### 第一层：概念本身
- 正式定义
- 核心要点（3-5 条）
- 适用范围

### 第二层：隐喻映射
- 故事世界观 ↔ 概念的领域
- 规则系统 ↔ 概念的机制
- 角色 ↔ 概念中的参与者/要素
- 失控事件 ↔ 概念的边界条件

### 第三层：理论对应
- 故事中哪些机制精确对应了理论
- 哪些地方做了简化或艺术化处理
- 现实世界中这个概念的应用实例

---

## 7. 质量检查清单

生成故事后，按此清单自检：

- [ ] 故事长度 1500-2500 字
- [ ] 世界观自洽，无明显逻辑漏洞
- [ ] 概念名未在揭晓前出现
- [ ] 读者能在故事中感受到"某种规律"
- [ ] 揭晓有顿悟感，不是突兀的
- [ ] 三层解释完整且准确
- [ ] 故事本身可读，即使不知道概念也能享受
- [ ] 不是童话风格，是科幻/黑镜/哲学寓言风格
- [ ] 没有说教语气
- [ ] 失控场景是故事高潮
