# 三层解释模板

## 用途
在故事揭晓概念后，按此模板生成分层解释，帮助读者理解隐喻映射关系和理论对应。

---

## 模板结构

### 第一层：概念本身

```markdown
## 💡 概念：[概念名称]

**定义**：[一句话正式定义]

**核心要点**：
1. [要点 1]
2. [要点 2]
3. [要点 3]

**适用范围**：[概念在哪些领域/场景下有意义]
```

### 第二层：隐喻映射

```markdown
## 🔗 隐喻映射

| 故事元素 | ↔ | 概念要素 |
|----------|---|----------|
| [世界观中的 X] | ↔ | [概念中的 A] |
| [故事中的规则 Y] | ↔ | [概念中的机制 B] |
| [角色 Z 的行为] | ↔ | [概念中的参与者/要素 C] |
| [失控事件 W] | ↔ | [概念中的边界条件 D] |

**关键映射说明**：
- [最重要的一个映射，用 2-3 句话解释为什么这样映射]
- [第二个重要映射]
```

### 第三层：理论对应

```markdown
## 🔬 理论对应

**精确对应**：
- [故事中哪些机制精确反映了理论]
- [另一点精确对应]

**简化/艺术化处理**：
- [哪些地方为了故事性做了简化]
- [简化的原因]

**现实应用**：
- [这个概念在现实世界中的一个应用实例]
- [另一个应用实例]
```

---

## 格式示例（以"纳什均衡"为例）

### 第一层：概念本身

```markdown
## 💡 概念：纳什均衡（Nash Equilibrium）

**定义**：博弈中所有参与者都无法通过单方面改变策略来获得更好结果的状态。

**核心要点**：
1. 多方参与，每方都有两个或更多策略可选
2. 每个人的最优策略取决于其他人的选择
3. 均衡状态下，没有人有动力单独改变

**适用范围**：博弈论、经济学、国际关系、演化生物学
```

### 第二层：隐喻映射

```markdown
## 🔗 隐喻映射

| 故事元素 | ↔ | 概念要素 |
|----------|---|----------|
| 岛上的三条通道 | ↔ | 三个可选策略 |
| 每天选择通道的村民 | ↔ | 博弈参与者 |
| 通道的拥挤程度取决于所有人的选择 | ↔ | 收益函数取决于策略组合 |
| 所有人都走同一条路的那天 | ↔ | 达到纳什均衡 |

**关键映射说明**：
- 故事中"所有人都不想换路"的那一刻，就是纳什均衡的核心——每个人都在做"给定别人选择下的最优选择"，虽然整体结果不是最优的。
```

### 第三层：理论对应

```markdown
## 🔬 理论对应

**精确对应**：
- 村民选路 = 策略选择
- 通行时间 = 收益函数
- "没人想换" = 均衡条件

**简化/艺术化处理**：
- 真实的纳什均衡需要理性假设，故事中用直觉代替了理性计算
- 均衡的数学证明被简化为"大家都不动了"

**现实应用**：
- OPEC 成员国的产量博弈
- 商业中的价格战——所有参与者降价，利润都下降，但没人敢先涨价
```

---

## 使用说明

1. 每层解释必须与故事内容一一对应，不能凭空引入故事中没有的元素
2. 隐喻映射表至少 3 行，覆盖世界观、规则、角色、失控四个维度
3. 理论对应中"简化/艺术化处理"部分至少 1 条，诚实说明哪些地方不够精确
4. 现实应用至少 1 个，帮助读者将概念落地
