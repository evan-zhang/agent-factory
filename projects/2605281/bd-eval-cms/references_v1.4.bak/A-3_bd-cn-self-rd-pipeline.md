## 模块三：评估阶段映射（三阶段模型）

```
阶段A（公开信息 + 内部立项材料）：S-1 + S-2 + S-3
  └─ 核心问题：靶点有因果验证吗？候选化合物有差异化吗？合作方能交付吗？
  └─ 信息来源：内部研究报告、CRO尽调报告、文献数据库
  └─ 输出物：立项科学性评估报告（One-pager终局先立 + Gate 1~3结论）

阶段B（双轨并行）：S-4 + S-5 + S-6
  └─ 核心问题：研发里程碑能达到吗？IP能保护吗？License-out有市场吗？
  └─ 双轨：科学推进轨（R&D执行）+ 商业开发轨（License-out市场探索）
  └─ 输出物：中期管线价值评估报告（Gate 4~6结论）

阶段C（终审）：S-7 + S-8
  └─ 核心问题：自有商业化价值有多大？合并财务回报达标吗？
  └─ 输出物：管线投资决策建议书（含一票否决核查、三级验收标准、rNPV（风险调整净现值）模型）
```

### ▶ Gate 1（S-1）：靶点科学性评估（前提门）

**核心评估维度**：

**S-1.1 靶点因果验证**
- 人类遗传学证据（GWAS（全基因组关联研究）/孟德尔随机化）是否支持靶点与疾病因果关系？
- 功能性证据（动物模型/细胞实验/器官芯片）与临床表现的一致性？
- 靶点是否已有临床验证（同靶点竞品有临床数据，即使失败也提供验证信息）？

**S-1.2 靶点可成药性**
- 靶点蛋白结构是否已解析（PDB数据库收录状态）？
- 已知药物结合口袋质量（Druggability评分）？
- 小分子/生物药/核酸药哪种模式更适合该靶点？

**S-1.3 靶点竞争格局**
- 全球同靶点在研药物数量及阶段分布？
- 是否存在「靶点独占者」（单一公司持有核心IP并已推进至III期）？
- 进入时机窗口：先驱者/快速跟随者/差异化后进入？

**Gate 1结论模板**：
```
靶点因果验证等级：[强/中/弱/无]
靶点可成药性：[高/中/低]
竞争格局分类：[蓝海/蓝红海/红海]
置信度：[高/中/低]
需补证据Top5：
  1. 靶点人类遗传学文献综述（近5年）
  2. 同靶点所有临床试验汇总（成功+失败均需分析）
  3. 竞品靶点IP全景扫描（PCT专利检索）
  4. 国际KOL（关键意见领袖）对该靶点的学术评价
  5. CDE（药品审评中心）对同类靶点药物的审评历史
```

| 10年回报倍数 | 4~9倍 | Gate 7（S-8） |
| 毛利率（毛利润/营业收入）（自有商业化） | ≥60% | Gate 7（S-7） |
| R&D合作方跑道 | ≥18个月 | Gate 3（S-3） |

## 模块六：一票否决清单（7条）

1. **靶点无因果验证**：人类遗传学和功能性证据均不支持靶点与疾病的因果关系
2. **ADMET不可克服**：候选化合物存在无法通过结构优化解决的毒性或代谢缺陷
3. **R&D合作方跑道不足**：合作方可用资金跑道<12个月且无明确融资计划
4. **IP无法保护**：化合物专利申请存在根本性障碍（现有技术完全覆盖），或无法防止合作方对IP的侵蚀
5. **合并IRR（内部收益率）不达标**：License-out + 自有商业化双路并行情景下，合并IRR（内部收益率）仍低于15%
6. **IP转让无法约定**：合作合同无法明确约定全球IP归属CMS，存在IP所有权争议风险
7. **无商业化主体承接**：五大TA中无任何业务主体（深康/德镁/维盛）具备承接该适应症商业化的能力
