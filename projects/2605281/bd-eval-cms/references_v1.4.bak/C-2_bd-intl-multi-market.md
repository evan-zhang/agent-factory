## 模块三：评估阶段映射

C-2采用"宽进严出"的两阶段漏斗模型：

```
候选市场池（用户指定，最多15个）
          │
          ▼
阶段A：快速筛选（1周）
七维评分 → 排除<40分市场 → 保留TOP候选市场
          │
          ▼
阶段B：精排与路线图（1周）
精细评分 → 市场分层 → 进入路线图设计
          │
          ▼
阶段C：TOP市场触发C-1（滚动执行）
优先级最高市场 → 逐一触发C-1深度评估
```

| 阶段 | 核心产出 | 负责人 | 时限 |
|------|----------|--------|------|
| **A·快筛** | 候选市场评分初稿 | 投前分析师 | 5天 |
| **B·精排** | 最终优先级矩阵+进入路线图 | 投前经理+商务 | 5天 |
| **C·触发** | TOP市场C-1评估启动 | 投前团队 | 滚动 |

## 模块四：评估章节结构

### ▶ 全景扫描门：候选市场概览

**目标**：在正式评分前，建立各候选市场的基础档案，确保评分口径统一。

每个候选市场必须完成：
1. **监管机构确认**：明确对应监管机构名称
2. **市场规模初步估计**：含当地货币+人民币双币标注
3. **康联达资源就绪度**：现有团队/产品/合规许可证

---

### ▶ 核心七维评分体系

**七维权重分配（总计100%）**：

| 维度编号 | 维度名称 | 权重 | 评分说明（1-10分） |
|----------|----------|------|-----------------|
| D-1 | **市场规模** | 20% | 10=SAM（可服务市场）>CNY 5亿；1=<CNY 500万 |
| D-2 | **注册难度**（反向） | 15% | 10=注册路径简单/<12个月；1=注册>36个月或明确拒绝 |
| D-3 | **竞争强度**（反向） | 15% | 10=市场空白无竞品；1=≥5家强竞品已占据主要份额 |
| D-4 | **准入可行性** | 15% | 10=有明确报销路径+高概率；1=纯自费且WTP（支付意愿）极低 |
| D-5 | **渠道就绪度** | 15% | 10=康联达已有成熟渠道+合规证；1=从零建设 |
| D-6 | **战略价值** | 10% | 10=旗舰市场/对组合有标杆意义；1=孤立市场/无协同 |
| D-7 | **财务吸引力** | 10% | 10=IRR（内部收益率）预估>40%；1=IRR<15% |

**综合得分计算**：
```
综合得分 = Σ(维度得分 × 权重)

调整项（+/-）：
+ 2分：该市场为康联达2030北极星目标中的旗舰市场
- 2分：合作方对该市场设有排他性竞争安排
- 3分：市场处于政治/汇率高风险状态（近12个月）
```

---

### ▶ 市场分层输出

**综合得分分层**：

| 层级 | 得分范围 | 标签 | 后续行动 |
|------|---------|------|---------|
| **优先进入** | ≥75分 | 🔴 P1 | 立即触发C-1深度评估；6个月内启动注册 |
| **次序推进** | 55-74分 | 🟡 P2 | 6-18个月内触发C-1；待P1市场进入稳定后 |
| **机会储备** | 40-54分 | 🟢 P3 | 纳入康联达3年投前储备库；每年复评 |
| **暂不推进** | <40分 | ⚪ 排除 | 记录排除原因；定期复核市场变化 |

### ▶ SPBP市场分级对应

康联达SPBP（Strategic Priority Based Planning，战略优先级规划）市场分级与七维评分对应关系：

| SPBP级别 | 定义 | 典型市场 | 七维评分参考范围 | 资源投入比例 |
|----------|------|----------|---------------|------------|
| **S级（战略核心）** | 组合最重要市场，承载2030收入目标主体 | 新加坡、台湾、沙特 | ≥80分 | 50% |
| **P级（优先推进）** | 高增长潜力，短期可实现的市场 | 香港、阿联酋、澳大利亚 | 65-79分 | 30% |
| **B级（基础布局）** | 战略补充，长期价值市场 | 马来西亚、泰国、墨西哥 | 50-64分 | 15% |
| **P级（管线储备）** | 暂不投入，持续监控 | 越南、印尼、哥伦比亚 | <50分 | 5% |

> 注：SPBP分级每年复评一次，市场环境重大变化时可触发临时复评。

### ▶ 市场进入路线图设计

**序贯进入 vs 并行进入决策框架**：

| 因素 | 建议序贯进入 | 建议并行进入 |
|------|-----------|-----------|
| 资源约束 | 紧张（预算<CNY 5000万/年） | 充裕 |
| 市场相似度 | 市场间差异大（如港台 vs GCC） | 市场相似（如新马泰同一监管参考） |
| 注册依赖性 | 后续市场参考前续市场注册文件 | 各市场独立申报 |
| 康联达团队经验 | 新进入区域（需积累经验） | 已有成熟区域团队 |

**推荐路线图模板**：

```
第一波（0-12个月）：P1市场（≥75分）
        新加坡 / 香港 / 台湾 / 沙特（视具体评分）
              ↓
第二波（12-24个月）：P2市场（55-74分）
        阿联酋 / 澳大利亚 / 马来西亚
              ↓
第三波（24-36个月）：P3市场（40-54分）
        泰国 / 墨西哥 / 巴西（视P1/P2进展）
```

## 模块五：财务内控对照表

> C-2为排序工具，财务内控仅用于D-7维度评分参考；具体财务门槛在C-1中执行。

| 财务指标 | 中文全称 | C-2评分参考（D-7维度） |
|----------|----------|---------------------|
| IRR | 内部收益率 | ≥40%=10分 / 25-40%=7分 / 15-25%=4分 / <15%=1分 |
| Peak Revenue | 峰值收入 | >CNY 5亿=10分 / 1-5亿=7分 / 2500-1亿万=4分 / <2500万=1分 |
| Payback Period | 回收期 | ≤2年=10分 / 2-3年=7分 / 3-4年=4分 / >4年=1分 |
| Gross Margin | 毛利率 | ≥60%=10分 / 50-60%=7分 / 40-50%=4分 / <40%=1分 |

## 模块六：一票否决清单

以下情形使对应市场直接归入"暂不推进"层，不参与评分排序：

- [ ] **X-01** 目标市场属于康联达明确排除名单（以色列/印度/韩国）
- [ ] **X-02** 监管机构明确拒绝该类产品或存在进口禁令
- [ ] **X-03** 市场当前处于战争/制裁/金融危机高危状态（参考外交部风险提示）
- [ ] **X-04** 合作方在该市场已签署排他性协议，康联达无进入权利
- [ ] **X-05** 七维综合得分+调整项<30分（绝对低分，资源不值得评估）

```
用户提交候选市场清单（M-02）
          │
          ▼

          │
          ▼

          │
          ▼

          ▼

          │
          ▼

          │
   ┌──────┴──────────────┐
   ▼                      ▼
P1市场立即触发C-1      P2市场纳入12-24月计划
```

## 模块九：输出规范

### 9.1 报告结构

```
封面页（含目标市场地图）
├── 执行摘要（优先级排序结论，300字）
├── 评估基础信息（M-01~M-10）
├── 候选市场概览档案（逐市场简档）
├── 七维评分矩阵（完整表格+雷达图）
├── 市场分层结果（P1/P2/P3/排除，含气泡图）
├── SPBP市场分级对应分析
├── 序贯 vs 并行进入建议
├── 市场进入路线图（3年甘特图）
├── 资源分配建议（预算/人力分配比）
├── 一票否决核查结果（排除市场清单）
├── 下一步行动（P1市场C-1触发计划）
└── 附录（各市场评分详细依据/数据来源）
```

### 9.2 格式要求
- **语言**：简体中文，财务英文缩写须标注中文全称
- **双币标注**：涉及具体金额均需双币（当地货币+人民币）
- **格式**：麦肯锡风格HTML，含市场气泡图（规模×吸引力坐标系）
- **特色图表**：七维雷达图（单市场）、评分矩阵热力图、3年进入路线图甘特图
- **字数**：≥4500字（市场数量越多字数越长）

### 9.3 三级验收标准

| 级别 | 必达项 | 参考项 | 否决项 |
|------|--------|--------|--------|
| L1 | 所有候选市场均完成七维评分；P1市场已触发C-1计划 | SPBP分级与集团战略目标对齐 | 含排除市场（以色列/印度/韩国） |
| L2 | 市场进入路线图覆盖36个月 | 资源分配建议具体可执行 | 七维评分缺失超过2个维度 |
| L3 | 结论与C-3组合战略目标一致 | 外部数据来源注明 | P1市场无C-1跟进计划 |
