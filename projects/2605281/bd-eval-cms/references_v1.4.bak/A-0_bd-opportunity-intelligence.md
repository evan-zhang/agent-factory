| **技能名称** | 商机情报扫描（投前 Opportunity Intelligence Scanner） |

## 模块三：评估阶段映射（三阶段模型）

```
阶段A（公开信息）：五层漏斗全部在阶段A内完成
  └─ 扫描层1：信号捕获（输入门）
  └─ 扫描层2：七项否决快扫（过滤门）
  └─ 扫描层3：赛道分析（价值门）
  └─ 扫描层4：A/B/C分级（输出门）
  └─ 扫描层5：分拨与输出（分发门）

阶段B（双轨并行）：A-0不进入阶段B，由下游技能（A-1/A-2等）承接
阶段C（终审）：A-0不进入阶段C，由下游技能承接
```

**三种扫描模式详述**：

### 模式①：适应症维度扫描
- **输入**：指定适应症（如2型糖尿病肾病、慢性乙型肝炎、特应性皮炎）
- **产出**：该适应症全球在研/已上市产品地图、机制分类、开发阶段分布、竞争热度评估
- **工具**：ClinicalTrials.gov / PubMed / IQVIA / EvaluatePharma公开数据

### 模式②：靶点/机制维度扫描
- **输入**：指定靶点（如SGLT-2、IL-4Rα、VEGF-A）或机制类别
- **产出**：该靶点/机制全球布局图、各阶段代表性候选药物、IP障碍点、可切入窗口分析
- **工具**：专利数据库（PCT/CNIPA）、学术文献、公司管线公告

### 模式③：公司维度扫描
- **输入**：指定目标公司（Biotech/Pharma）或公司特征（如「日本中型药企」「韩国皮肤科Biotech」）
- **产出**：目标公司全管线梳理、战略方向分析、投前交易历史、与集团协同点识别
- **工具**：公司官网、年报、投资者公告、投前数据库

## 模块四：评估章节结构（Gate结构）

### ▶ 输入层：信号捕获（扫描层1）

**触发信号来源清单**：
1. 学术会议/重磅数据披露（NEJM/Lancet/JAMA/NEJM Evidence等顶刊）
2. 监管机构公告（FDA突破性疗法/快速通道/孤儿药认定；NMPA优先审评）
3. 大型药企投前交易公告（合作/授权/收购披露）
4. 行业数据库预警（EvaluatePharma管线变化、ClinicalTrials新注册）
5. KOL/学术合作方主动推荐
6. 竞品公司公开管线更新
7. 会议/展会（BIO-Asia/JP Morgan/ESMO/ADA等）展台信息

**层1输出物**：原始信号清单（候选标的名称、来源、捕获日期、推送方）

---

### ▶ 过滤门：七项否决快扫（扫描层2）

以下七项任一命中即终止推进（第7项允许降级为Watch）：

| 否决编号 | 否决项 | 判断标准 |
|----------|--------|----------|
| V-1 | 治疗领域完全不匹配 | 与集团六大TA（心肾代谢/消化肝病/CNS/皮肤健康/眼科/仿制药）均无交集 |
| V-2 | 中国注册路径明确不可行 | 临床数据完全不可桥接且无中国患者入组计划，或涉及明确监管禁止品类 |
| V-3 | 原研方已与竞争对手独家签约 | 中国市场已签署独家License-in协议且无子授权可能 |
| V-4 | 核心IP障碍不可绕过 | 中国专利保护期≥10年且原研方不授权且无等效绕道路径 |
| V-5 | 安全性/合规红线 | 存在FDA/EMA黑框警告且适应症人群与集团目标重叠度≥80% |
| V-6 | 绝对规模天花板过低 | 按适应症规模分级判断（依据体系总规则增补条款§5.5）：大适应症（>50万患者）乐观峰值<8亿元；中适应症（5-50万患者）乐观峰值<5亿元；罕见病/小适应症（<5万患者）乐观峰值<2.5亿元；未明确规模时沿用<1亿元标准 |
| V-7 | 合作可能性明显不足 | 命中以下三类高风险场景之一且无例外条件：**CP-1** 全球Top20药企核心品种（全球销售≥10亿美元或战略重点）+无公开合作/外授信号+无战略收缩；**CP-2** 持有方已建成熟销售队伍（≥200人或目标科室覆盖率≥60%）+核心利润品种（占收入≥20%）+无合作/转让意向；**CP-3** 上市公司/大型集团控股+核心利润贡献（板块收入≥15%或净利润Top3）+近期增持/并表/战略加码（非收缩）。**处理方式**：降级为Watch或关闭（非即时关闭）。**例外条件**（满足任一即突破）：①对方主动接触；②公开BD信号；③战略调整事件；④竞争格局突变；⑤股权结构变动；⑥BD团队已有接触基础且对方未明确拒绝 |

**层2输出物**：过滤后候选标的清单（注明通过/Watch/关闭及原因）

**"轻触探意愿"机制（V-7专用）**：
- 对于信号模糊的标的（无法确定是否命中CP-1/CP-2/CP-3），BD团队通过非正式渠道探寻合作意愿
- 时限：≤2周，极低资源投入（邮件/会议边际接触/中间人询问）
- 结果处理：对方有意→激活正式评估（进入层3）；对方无意/无回应→进入Watch清单
- Watch清单季度回顾，监测例外条件是否触发

---

### ▶ 价值门：赛道分析（扫描层3）

**3.1 市场规模初判**
- 患者总数/目标患者数（粗估）
- 现有治疗方案渗透率
- 市场增速（近3年CAGR（复合年均增长率））
- 峰值销售额（PeakSales）预估区间（乐观/基准/保守）

**3.2 竞争格局评分（1-5分）**
- 已上市同类药物数量及市占率
- 在研竞品数量及阶段分布
- 集团进入时机窗口（先行优势/跟随优势/差异化空间）

**3.3 注册可行性初判**
- 桥接路径（海外数据+中国桥接 / 全球同步 / 中国独立研究）
- 预计注册时间线（至中国首批上市）
- 政策加速通道可能性（突破性治疗品种/优先审评/附条件批准）

**3.4 集团协同度评分（1-5分）**
- 与现有管线TA匹配度
- 与现有商业化团队重叠度
- 与现有客户/医院网络适配度

**3.5 合作可行性评分（1-5分）**
- 合作方主动释放合作信号的强度（公开声明/BIO展台/中间人传话/无信号）
- 合作方历史BD交易活跃度（近3年外授/合作数量）
- 合作方当前战略方向与CMS合作的契合度
- 是否存在V-7例外条件触发的迹象
- 评分标准：5分=对方主动接触且明确意向；4分=公开BD信号+历史活跃；3分=中间人传话+历史有交易；2分=无明确信号但无排斥迹象；1分=命中CP-1/CP-2/CP-3且无例外条件

**层3输出物**：赛道分析摘要表（每个候选标的一行，含市场/竞争/注册/协同/合作可行性五维评分）

### ▶ 输出门：A/B/C/Watch分级（扫描层4）

| 等级 | 定义 | 行动指令 | 时限 |
|------|------|----------|------|
| **A级** | 五维评分均≥3.5，且无重大不确定性；具备立即推进条件 | 立即启动正式评估（分拨至对应A-1/A-2/A-3/A-4/A-6/A-8技能） | 0~30天立项 |
| **B级** | 五维整体评分≥3.0，存在1~2个可监测的不确定性 | 纳入季度跟踪清单，设置触发条件（如「II期数据披露后重评」） | 季度跟踪 |
| **C级** | 五维整体评分2.0~3.0，或存在重大不确定性但潜力可观 | 年度回顾，可在集团战略调整时重新激活 | 年度回顾 |
| **Watch** | 五维评分<2.0，或合作可行性评分≤2且其余四维≥3.0（赛道好但合作可能性低），或V-7命中后降级| 低强度监测，不占用评估资源 | 持续监测 |
| **关闭** | 命中层2六项否决任一，或层3赛道分析结论为不可行 | 正式关闭，记录关闭原因存档 | 即时关闭 |

## 模块六：一票否决清单

> 以下任一条件命中，A-0扫描即刻关闭该标的，无需进入下游技能。

1. **TA完全不匹配**：与集团六大TA（心肾代谢/消化肝病/CNS与精神/皮肤健康/眼科/化学仿制药）均无任何交集
2. **中国注册路径完全封闭**：监管政策明确禁止该品类在中国注册上市
3. **独家竞争封锁**：中国市场已被强竞争对手获得独家授权且条款明确无子授权
4. **峰值规模过低**：即便乐观情景下中国峰值销售额（PeakSales）<1亿元人民币（医美<3000万元人民币）
5. **不可绕过的IP障碍**：核心专利保护期覆盖≥10年且原研方无意向授权
6. **严重安全信号**：已知不可克服的黑框警告或致死性安全事件且无法通过患者筛选管理
7. **合规红线**：涉及集团明确禁止的品类（如麻醉药品、强管制一类精神药品主力品种）
8. **合作可能性明显不足（V-7）**：命中CP-1/CP-2/CP-3三类高风险场景之一且无例外条件触发——处理方式为降级Watch或关闭（非即时关闭），区别于前7条的即时关闭

## 模块七：Gate阶段说明

| 扫描层 | 门控名称 | 核心问题 | 通过标准 | 负责角色 |
|--------|----------|----------|----------|----------|
| 层1 | 输入门 | 这个信号值得花时间看吗？ | 来自可信来源，基本信息可获取 | 投前情报分析师 |
| 层2 | 过滤门 | 六项否决是否命中？ | 六项全部通过（无命中） | 投前分析师（15分钟快扫） |
| 层3 | 价值门 | 赛道有没有价值？ | 四维评分均≥2.5 | 投前总监+医学团队 |
| 层4 | 输出门 | 分级是什么？该怎么处理？ | A/B/C分级明确，行动指令清晰 | 投前委员会 |
| 层5 | 分发门 | 推给谁评？ | 正确分拨至对应下游技能 | 投前主管 |

**时间效率标准**：
- 层1→层2：≤1个工作日
- 层2→层3：≤3个工作日
- 层3→层4→层5：≤5个工作日
- **全流程（层1~层5）目标完成时间：≤10个工作日**

## 模块八：串接接口

### 上游接口（输入来源）
- 外部：学术会议/监管公告/行业数据库/KOL/竞品情报
- 内部：集团战略规划部优先TA清单、业务单元专项扫描需求

### 下游接口（输出去向）

```
A-0输出
  ├─ A级标的 → A-1（海外已上市/中国未上市）
  │           → A-2（代理权引进）
  │           → A-3（自主研发管线）
  │           → A-4（生物类似药）
  │           → A-6（院外延伸）
  │           → A-8（高壁垒仿制药，天津康哲）
  │           → medical-aesthetics-product-evaluator（医美单标的）
  ├─ 多个A级 → A-7（多标的横向筛选）
  ├─ B/C级   → 跟踪池（季度/年度回顾）
  └─ 关闭    → 关闭档案库（记录原因，备查）
```

### 数据流规范
- 每个扫描标的须生成**唯一标的ID**（格式：PT-YYYYMMDD-XXX）
- A-0输出摘要表须包含：标的ID、名称、TA、扫描模式、分级、推荐技能、置信度、关键待验证信息
- 输出表格须同步至投前项目管理系统（由使用方自行对接）

## 模块九：输出规范

### 视觉与格式规范
- **文件格式**：HTML（主报告）+ Markdown摘要表（可单独提取）
- **配色**：深蓝 `#1a3a5c`（标题/关键结论）；白色背景；禁用灰色字体；禁用任何元描述文字
- **风格**：麦肯锡顾问风格——结论先行，数据支撑，结构清晰，无废话
- **财务缩写规范**：
  - IRR → IRR（内部收益率）
  - CAGR → CAGR（复合年均增长率）
  - PeakSales → 峰值销售额（PeakSales）
  - NPV → NPV（净现值）
  - EBITDA → EBITDA（息税折旧摊销前利润）
  - COGS → COGS（销售商品成本）
- **非人民币双币标注**：凡出现美元/欧元金额，须同步标注人民币（按当期汇率，注明换算日期）
- **禁止灰色字体**：所有正文字体颜色须清晰可辨，禁用 `#999999` 以下浅色
- **禁止元描述**：不得在报告内出现「本报告旨在……」「本技能用于……」等自我描述文字

### 报告结构模板
```
封面：标的名称 / 扫描日期 / 扫描维度 / 发起单元
第1章：执行摘要（扫描发现 + 分级结论）
第2章：信号来源与背景
第3章：六项否决快扫结果
第4章：赛道分析（市场/竞争/注册/协同）
第5章：A/B/C分级详述与行动建议
附录A：原始信号清单
附录B：同赛道竞品列表
参考文献
```

### 置信度标注规范
- **高**：关键结论均有公开数据支撑，无重大信息缺口
- **中**：核心结论有部分依据，存在1~2个重要待验证假设
- **低**：结论主要基于推断，待验证信息超过3项
