## 模块三：评估阶段映射

C-1采用三阶段线性评估，必须在市场分级确认后启动：

```
市场分级确认（来自C-2或用户直接指定）
          ↓
阶段A（注册可行性）    阶段B（商业价值）    阶段C（交易结构）
Gate 1注册路径确认  →  Gate 2~4市场评估  →  Gate 5~6财务/交易
      ↓                    ↓                    ↓
   注册团队（5天）     商业化团队（20天）     财务+法务（10天）
```

**市场分级优先级**（影响评估深度与时间投入）：

| 梯队 | 覆盖市场 | 评估深度 | 评估周期 |
|------|----------|---------|---------|
| **第一梯队** | 香港、台湾、新加坡 | 完整6-Gate | 35天 |
| **第二梯队** | GCC（沙特/阿联酋为主）、澳大利亚/新西兰、拉美（墨西哥/巴西） | 完整6-Gate | 40天 |
| **第三梯队** | 东南亚其他（马来西亚/泰国/越南/印尼/菲律宾） | 简化4-Gate | 25天 |

## 模块四：评估章节结构（Gate结构）

### ▶ One-pager：市场与产品匹配先立

**目标**：在进入数据核查前，先确认产品与目标市场的战略匹配度。

必填要素：
1. **产品市场匹配陈述**：`[产品] 在 [目标市场] 解决 [当地患者/消费者] 的 [需求]，优于当地现有 [竞品] 之处在于 [X]`
2. **康联达战略价值**：该市场/该产品对康联达的组合意义（填补TA缺口/强化既有优势）
3. **5年收入预测**（双币）：峰值收入 ___（当地货币）/ ___（人民币）
4. **核心注册障碍**：最主要的监管挑战是什么
5. **One-pager决策**：通过 → Gate 1 ｜ 暂缓 → 补充信息 ｜ 否决 → 记录归档

---

### ▶ Gate 1：注册路径与监管环境（前提门）

**核心问题**：产品能否在目标市场获得上市许可？时间和成本是否可接受？

**各市场主要监管机构及注册路径**：

| 市场 | 监管机构 | 注册路径（优先） | 典型审批周期 | 特殊要求 |
|------|----------|--------------|------------|---------|
| 新加坡 | HSA（卫生科学局） | 变体申请/完整NDA | 12-18个月 | 良好分销规范（GDP）认证 |
| 香港 | 香港药物管制处（DH） | 注册药剂制品 | 6-12个月 | 需本地负责人 |
| 台湾 | TFDA（台湾食品药物管理署） | 药品许可证 | 18-24个月 | 中文标签强制 |
| 沙特 | SFDA（沙特食药局） | 本地化申请 | 12-24个月 | 阿拉伯语标签+Halal认证 |
| 阿联酋 | MOH（卫生部）/DHA | 联邦注册 | 12-18个月 | 进口许可证 |
| 澳大利亚 | TGA（治疗商品管理局） | 注册/标注路径 | 12-18个月 | ARTG收录 |
| 新西兰 | Medsafe | 评估申请 | 12-24个月 | — |
| 马来西亚 | NPRA（国家药品管理局） | MAL注册 | 12-18个月 | BM标签 |
| 泰国 | FDA Thailand | 药品注册 | 18-24个月 | 泰文标签 |

**评估维度**：

| 维度 | 评估内容 | 权重 |
|------|----------|------|
| 1-1 注册路径可行性 | 是否存在快速通道（加速批准/参考认可） | 35% |
| 1-2 注册时间线 | 预计获批时间（月数）；是否影响市场窗口 | 25% |
| 1-3 注册成本（双币） | 政府费用+技术申报费+本地测试要求 | 20% |
| 1-4 上市后义务 | 不良事件报告/定期安全性更新报告（PSUR）/本地药物警戒要求 | 20% |

**Gate 1 一票否决**：
- [ ] 目标市场监管机构明确拒绝该类产品注册
- [ ] 注册周期>36个月，错过产品生命周期最佳窗口
- [ ] 存在中国大陆被禁原料/成分，目标市场同样禁用

### ▶ Gate 2：临床价值与本地化需求（证据门）

**核心问题**：产品的临床价值能否满足目标市场的本地化需求？是否需要本地临床研究？

| 种族差异研究 | [是/否/已有数据] | — | — |

| 包装本地化（语言/规格） | 必须 | 中 | 2-3个月 |

### ▶ Gate 6：交易结构与财务回报（可做门）

**核心问题**：交易条款是否合理？综合财务回报是否满足康联达国际业务投资门槛？

**财务内控标准（强制）**：

| 指标 | 中文全称 | 康联达国际门槛 | 说明 |
|------|----------|-------------|------|
| IRR | 内部收益率 | ≥25%（创新药）/ ≥50%（仿制/消费） | 5年现金流，双币计算 |
| 回收期 | — | ≤3年（第一梯队）/ ≤4年（第二/三梯队） | |
| 毛利率 | Gross Margin | ≥55%（第一梯队）/ ≥50%（其他） | |
| Peak Revenue | 峰值收入 | ≥当地货币500万（约CNY 2500万） | 最低规模阈值 |
| NPV | 净现值 | >0（折现率12%） | 双币NPV均须为正 |

**交易条款核查**：

| 条款项 | 评估内容 | 风险等级 |
|--------|----------|---------|
| 独家授权范围 | 地域/适应症/期限 | 高 |
| 首付款（当地货币+人民币） | 绝对金额合理性 | 高 |
| 销售提成率（Royalty Rate） | 比例+基数定义 | 中 |
| 最低销售承诺（双币） | 是否存在对赌 | 高 |
| 汇率风险对冲 | 收付款货币约定；汇率波动保护 | 中 |

## 模块六：一票否决清单

- [ ] **R-01** 目标市场在康联达明确排除名单（以色列/印度/韩国）
- [ ] **R-02** 产品在目标市场监管机构明确拒绝注册或存在上市禁令
- [ ] **R-03** 注册周期>36个月且无加速路径
- [ ] **R-04** 基准情景IRR（内部收益率）<15%，无条款优化空间
- [ ] **R-05** 目标市场存在已确认的独家竞争对手协议，康联达无进入机会
- [ ] **R-06** 峰值收入预测<CNY 500万（规模不经济，资源不应投入）
- [ ] **R-07** 汇率风险无法对冲且波动超过±20%，导致IRR（内部收益率）转负
- [ ] **R-08** 合作方在目标市场存在未披露的重大法律纠纷

## 模块七：Gate阶段说明

### Gate流程路径

```
One-pager（市场/产品匹配确认）
          │
          ▼
Gate 1（前提门）── 注册不可行 → [终止]
          │
          ▼
Gate 2（证据门）── 本地化成本过高 → [重评优先级]
          │
          ▼
Gate 3（支付门）── 市场太小 → [考虑捆绑多市场降低固定成本]
          │
          ▼
Gate 4（定调门）── 定价受限/报销无望 → [重谈条款]
          │
          ▼
Gate 5（成本门）── 毛利率<50% → [重谈COGS或终止]
          │
          ▼
Gate 6（可做门）── IRR<20% → [终止或降级资源投入]
          │
          ▼
[推送至C-3组合评审 + 提交康联达管理层]
```

### 市场分级与Gate执行差异

| Gate | 第一梯队（港台新） | 第二梯队（GCC/澳新/拉美） | 第三梯队（东南亚其他） |
|------|-----------------|----------------------|-------------------|
| Gate 1 | 完整执行 | 完整执行 | 简化（仅合规过滤） |
| Gate 2 | 必须KOL访谈 | KOL访谈（≥1位） | 文献研究替代 |
| Gate 3 | 完整TAM/SAM/SOM | 完整 | 简化（仅SOM） |
| Gate 4 | 必须评估报销 | 评估报销 | 自费市场，可简化 |
| Gate 5 | 完整成本建模 | 完整 | 简化 |
| Gate 6 | 完整IRR模型 | 完整 | 简化（定性判断） |

## 模块九：输出规范

### 9.1 报告结构

```
封面页（含目标市场旗帜/地图标注）
├── 执行摘要（双币，300字）
├── 产品×市场前置信息（M-01~M-10）
├── One-pager市场/产品匹配画像
├── Gate 1-6逐门评估结果（含双币数据）
├── 注册路径甘特图（月为单位，含关键里程碑）
├── 竞品定价矩阵（双币热力图）
├── 准入/报销可行性评分表
├── 财务模型（三情景，双币）
│   ├── 5年P&L（当地货币+人民币并列）
│   ├── IRR/NPV计算（双币）
│   └── 汇率敏感性分析
├── 核心条款评估要点
├── 一票否决核查表
├── 三级验收标准
├── 市场进入100天计划
└── 附录（监管文件清单/KOL访谈摘要/竞品资料）
```

### 9.2 格式要求
- **语言**：简体中文，财务英文缩写须标注中文全称
- **双币标注**：所有金额数字均需同时标注当地货币和人民币（注明汇率取值日期）
- **格式**：麦肯锡风格HTML，支持PDF导出
