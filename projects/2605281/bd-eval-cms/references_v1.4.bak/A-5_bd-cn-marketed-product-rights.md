### ▶ One-pager：终局先立（A-5专项锚定）

**A-5专项终局五问**：
1. 接管后，产品销售额能在第几年超越现推广方的历史峰值？
2. 渠道资产（经销商/医院覆盖/处方医生关系）中，真正能迁移过来的有多少比例？
3. 如果渠道迁移比例只有50%，财务是否仍可接受？
4. 原推广方离场后，竞争格局会对我们有利还是不利？
5. 产品是否面临近期集采/医保续约降价风险？如是，降价后财务是否仍可接受？

**One-pager模板（半页纸，禁止超过450字）**：
```
产品：[通用名] | 批文号：[XXXXXXX] | 持有人：[公司名称]
已批适应症：[清单] | 当前医保身份：[甲类/乙类/自费/限定报销]
历史峰值销售额：[X亿元人民币（YYYY年）] | 最近12个月销售额：[X亿元人民币]
现推广方：[公司名称] | 转让/合作原因：[简述]
目标承接主体：[深康/德镁/维盛]
基准渠道迁移比例：[X%] | 基准峰值销售额（PeakSales（峰值销售额））:[X亿元人民币]
首年净现金流：[X万元人民币] | IRR（内部收益率）预估区间：[X%~Y%]
推荐结论：[立项推进 / 附条件推进 / 暂缓 / 关闭]
One-pager负责人：[姓名] | 完成日期：[YYYY-MM-DD]
```

**One-pager必答补充说明**：
- 必须同步给出“历史峰值 vs 接管后峰值”的对比逻辑，不能仅给单点峰值预测。
- 必须给出“迁移比例下降10个百分点”的敏感性提示，作为后续Gate 6压力测试输入。
- 必须标记产品属于“进口原研代理切换”“国产原厂推广外包”“老品种二次盘活”中的哪一类。
- 必须初判该产品的独特价值来源：**即时现金流贡献、低注册风险、存量保护策略、渠道协同扩张**中的主导项。

### ▶ Gate 1：产品力现状评估（前提门）

**核心评估问题**：
- 产品在目标适应症中的临床地位是否仍然稳固（未被新一代药物边缘化）？
- 专利状态：核心化合物专利是否已到期或即将到期（决定仿制药竞争压力）？
- 说明书适应症与现实推广的差距（超适应症风险评估）？
- 是否存在药物安全性风险（ADR（不良反应）警示、黑框风险、限制用药提醒）？
- 产品是否被纳入重点监控目录或医院限用目录（影响院内使用政策）？
- 若为进口原研产品，其原研品牌力是否仍显著强于同类国产替代？
- 若为国产产品，其生产质量、供应稳定性和一致性评价地位是否构成持续护城河？

**Gate 1专项判断逻辑**：
- A-5不讨论注册可行性，但必须判断“产品力是否足以支撑接管后继续投入”。
- 对老品种，不能只看当前收入规模，必须判断“衰退是因产品力减弱，还是因原推广方资源投放不足”。
- 对进口原研品种，必须拆分“品牌红利”与“临床真实优势”，防止将历史高销售错误外推到接管后。

**产品力评分矩阵（1-5分）**：

| 评分维度 | 权重 | 评分（1-5） | 加权分 | 评分说明 |
|---------|------|-------------|--------|---------|
| 临床价值持久性（vs. 新一代替代品） | 35% | | | 指南地位、处方黏性、医生认知 |
| 专利保护剩余年限 | 20% | | | 包括化合物、制剂、用途、市场独占 |
| 安全性风险可控程度 | 25% | | | ADR（不良反应）频率、黑框警示、真实世界安全事件 |
| 政策风险 | 20% | | | 重点监控、辅助用药属性、医保支付限制、集采可及性 |
| **总分** | 100% | | | |

**产品力评级建议**：
- 4.0~5.0分：强，具备接管后继续放量或稳盘基础
- 3.0~3.9分：中，可做但需依赖渠道或准入策略修复
- 2.0~2.9分：弱，仅在交易条款极优或战略协同极强时考虑
- <2.0分：衰退，不建议继续推进

**重点核查模板：产品力衰退归因拆解**：
```
近3年销售变化：[增长 / 平稳 / 下滑]
核心归因1：[产品力下降 / 渠道投入不足 / 竞争加剧 / 支付受限 / 供应问题]
核心归因2：[补充]
若更换推广方后可逆转部分：[是/否]，可逆转比例预估：[X%]
不可逆结构性压力：[列举Top3]
```

**Gate 1结论模板**：
```
产品力评级：[强/中/弱/衰退]
产品力综合评分：[X/5]
核心专利/市场独占到期年份：[YYYY年/已到期]
主要替代竞品威胁：[Top3]
安全性与政策风险摘要：[一句话概括]
产品衰退属性判断：[资源投入不足型 / 竞争替代型 / 支付受损型 / 结构性衰退型]
接管后产品力延续判断：[可延续 / 有条件延续 / 难以延续]
置信度：[高/中/低]
需补证据Top5：
  1. 最新版说明书及历次修订记录
  2. 近3年ADR（不良反应）监测与药物警戒数据
  3. 国内外同类新上市/在研竞品对比分析
  4. 重点监控目录/医院限用政策核查
  5. 专利到期与仿制药竞争时间线
```

### ▶ Gate 2：渠道资产与迁移可行性（定调门）

**核心评估问题**：
- 现有渠道资产（医院覆盖、科室渗透、KOL（关键意见领袖）、经销商、处方医生）中，哪些是真正属于产品，哪些依赖原推广方组织能力？
- 流向数据真实性是否可信（月度销量、库存周转、终端消耗、价格体系、DSO（应收账款周转天数））？
- 销售团队随行比例和关键岗位交接可行性如何？
- 原推广方离场后，品牌认知是否会断层，是否存在“医生认品牌不认产品”的风险？
- 若为进口产品，原研方对新代理商的合作意愿和供应稳定性是否明确？
- 若为国产产品，MAH（上市许可持有人）与推广方权责边界是否清晰，是否存在灰色推广历史遗留风险？

#### 2.1 渠道资产盘点
- 医院覆盖数量（三级/二级/基层；院内/院外/DTP（直接面向患者药房））
- 覆盖科室结构与重点处方科室占比
- 核心KOL（关键意见领袖）与区域带头人名单、合作深度、学术活动频次
- 一级经销商与重点配送商名录、历史合作年限、回款习惯
- 活跃处方医生数量（建议定义：近6个月有处方或稳定采购记录）
- 关键省份或核心城市贡献度（Top 10省份/Top 30医院）

#### 2.2 流向数据真实性核查

| 核查维度 | 核查方式 | 风险信号 | 结论记录 |
|---------|---------|---------|---------|
| 月度销量波动 | 近24~36个月趋势分析，识别是否异常冲量 | 季末/年末异常拉货 | |
| 终端发货与消耗匹配度 | 库存周转与终端消耗对比 | 长期压货、退货异常 | |
| 价格体系一致性 | 出厂价/批发价/终端价核对 | 价格倒挂、跨区窜货 | |
| DSO（应收账款周转天数） | 对比行业均值与合同账期 | 回款拉长、账龄恶化 | |
| 流向归属清晰度 | 医院/经销商/配送商三方口径交叉验证 | 数据口径不一致 | |

**流向数据真实性评级**：
- 可信：多来源交叉验证一致，库存与终端消耗闭环清晰
- 存疑：存在部分异常，但可通过追加尽调解释
- 高风险：核心数据无法对齐，存在造假或压货嫌疑
- 无法评估：对方不提供足够数据

#### 2.3 团队交接可行性
- 现有销售团队人数、地区分布、层级结构
- 可随行比例（销售代表/大区经理/MSL（医学科学联络官）/市场人员）
- 关键人员意向访谈结果：是否愿意转入CMS体系
- 薪酬对标差距：接管后是否需要显著加薪或签字费
- 区域重叠度：与现有深康/德镁/维盛团队的医院覆盖重合程度
- 交接时间窗：合同签署到实质接管之间是否存在真空期

**团队交接评分模板（1-5分）**：

| 维度 | 权重 | 评分 | 说明 |
|------|------|------|------|
| 关键人员可随行比例 | 30% | | |
| 区域重叠与整合效率 | 25% | | |
| 薪酬对标可承受性 | 20% | | |
| 交接时间窗可控性 | 15% | | |
| 合规历史清洁度 | 10% | | |

#### 2.4 品牌认知迁移风险
- 医生对产品的认知主体是“原研品牌/原厂品牌/推广团队品牌/产品通用名”中的哪一类？
- 原推广方是否在同科室有其他重磅品种，迁移后是否会优先扶持竞品？
- 是否存在学术活动、患者教育、随访体系高度绑定原推广方品牌的情况？
- 更换代理商后，患者端和医生端是否会产生供应、价格、报销、配送认知混乱？

**品牌迁移风险判定**：
```
品牌资产主要归属：[产品本身 / 原研方品牌 / 原推广团队 / 混合]
迁移后品牌断层风险：[高/中/低]
原推广方竞品干扰风险：[高/中/低]
需额外品牌重建投入：[低/中/高]
```

#### 2.5 进口产品特有：原研方合作意愿与供应稳定性
- 原研方是否认可CMS作为新推广方/代理商
- 中国供应配额与全球产能是否稳定
- 原研方是否保留中国市场战略调整权
- 合同中是否存在供应中断免责、最低采购量（MBQ（最低采购量））刚性约束或不合理召回责任

### ▶ Gate 3：市场现状与增长空间（证据门）

**核心评估问题**：
- 当前销售额及市占率处于什么水平，接管后能守住多少、提升多少？
- 接管后峰值销售额（PeakSales（峰值销售额））提升空间来自哪里？
- 增量来源是现有医院深耕、新医院开发、新适应症、新渠道、院外延伸还是价格优化？
- 竞争格局在交接窗口期（6~24个月）会如何变化？
- 如果全国集采/VBP（带量采购）进入窗口逼近，存量盘子会受损多少？

**A-5市场分析原则**：
- A-5不是“从0到1”的上市模型，而是“存量盘子 + 增量修复 + 渠道协同扩张”模型。
- 必须区分“历史峰值可否重回”与“接管后新增峰值能否实现”。
- 对成熟老品种，增量逻辑往往不来自大市场新开拓，而来自精细化运营、院外转移、重点省份重做与品牌稳盘。

#### 3.1 当前市场现状核查
- 当前滚动12个月销售额、近3年复合增长率CAGR（复合年均增长率）
- 当前市场份额与核心竞争对手排名
- 医院端与院外端销售结构占比
- Top省份/Top城市/Top医院集中度
- 进口产品与国产竞品之间的份额迁移趋势

#### 3.2 接管后增长空间拆解

```
增量驱动因素              预估贡献（万元人民币）    起效时间       风险备注
现有医院深耕               [X]                    第1~2年        依赖团队交接顺畅
新医院开发                 [X]                    第2~3年        受准入壁垒影响
```

## 模块六：一票否决清单

1. **产品力严重衰退**：被新一代药物实质性替代，市场份额年降幅≥30%连续2年
2. **批文合规性存疑**：批准文号存在法律瑕疵或正在被NMPA复查
3. **流向数据严重失真**：发现流向数据造假迹象（库存倒打、发票异常、终端口径无法闭环）
4. **渠道迁移比例<30%**：基准情景下商业化基础不存在
5. **集采纳入窗口期<12个月**：预计接管后12个月内即被纳入全国集采
6. **IRR（内部收益率）<15%**：基准情景IRR（内部收益率）低于最低门槛
7. **现推广方并发竞品**：交接期间同步推同适应症竞品，形成直接市场干扰
8. **独家性无法锁定**：合同无法保障独家权，存在多头代理或权利冲突风险
9. **进口产品原研方供应中断风险**：原研方明确表示可能停止中国供应，或供货条款存在重大不确定性

| Gate 6 | 可做门 | 整体交易合算吗？ | IRR（内部收益率）≥20%，回收期≤3年，首年净现金流>0 | CFO（首席财务官）+投前委员会 | C |

## 模块八：串接接口

### 上游接口
- A-0 A级分拨 → A-5
- A-7 A级（已上市类）→ A-5
- 原推广方主动接触 → A-5

### 下游接口
```
A-5评估完成
  ├─ Gate 6通过 → 投前委员会 → 代理权合同签署 → 商务团队接管
  ├─ 品种适合院外延伸 → A-6并行
  ├─ 中国+国际双市场 → C-1/C-2/C-3并行
  ├─ Gate 6条件通过 → 补充流向数据/价格压力测试后重回Gate 5/6
  └─ 任一Gate否决 → 暂缓/关闭档案
注：A-5对手方为已有稳定收入的大型药企，不适用E-1股权跟投逻辑（E-1仅适用于产品未在中国上市的Biotech）
```

## 模块九：输出规范

- **主报告格式**：HTML（深蓝 `#1a3a5c`，白底，无灰色字，无元描述）
- **麦肯锡风格**：每章节顶部采用结论句（Action Title），正文承接数据与论证
- **财务缩写标注**：IRR（内部收益率）、COGS（销售商品成本）、SG&A（销售、一般及行政费用）、EBIT（息税前利润）、CAGR（复合年均增长率）、DSO（应收账款周转天数）、MBQ（最低采购量）、PeakSales（峰值销售额）等均须括号标注中文全称
- **双币标注**：非人民币金额须同步标注人民币，并注明汇率换算日期
- **置信度等级**：高（主要结论有公开或可验证经营数据支撑）/ 中（有部分依据但关键假设待验证）/ 低（结论主要基于推断或单方信息）
- **内容来源标注**：遵循CMS投前评估体系总规则及增补条款中的来源标记规范

### 报告章节顺序

```
封面
执行摘要
目录
第1章 产品概述与市场背景（Gate 1）
第2章 渠道资产与迁移可行性（Gate 2）
第3章 接管后市场增长空间（Gate 3）
第4章 价格·医保·准入与存量保护（Gate 4）

附录C 竞品对比

参考文献
```

### 章节写作要求
- 第1章必须回答“产品是否值得接管”，而非重复说明书信息。
- 第2章必须用数据拆分“可迁移资产”与“不可迁移资产”。
- 第3章必须同时呈现“守盘”与“增量”两套逻辑。
- 第4章必须单列“存量保护策略”，不可与医保/价格章节混写后弱化。
- 第5章必须写清过渡期成本与稳定期成本的差异。
- 第6章必须突出“首年正现金流”与“迁移比例50%压力测试”结果。
