## 模块三：评估阶段映射

消费健康产品评估遵循"消费品逻辑优先"的三阶段推进：

```
阶段A（消费品终局锚定）   阶段B（市场与成本验证）   阶段C（规模化可行性）
One-pager终局先立      →  Gate 1~4品类/消费者/渠道  →  Gate 5~6 CAC/IRR决策
      ↓                          ↓                         ↓
  投前经理（2天）             院外业务委员会（15天）       财务+法务（7天）
```

| 阶段 | 核心问题 | 决策人 |
|------|----------|--------|
| **A·锚定** | 这个产品的消费者终局画像是谁？5年后的品牌位置在哪里？ | 投前经理 |
| **B·验证** | 消费者真的有这个需求吗？渠道能触达他们吗？ | 院外业务委员会 |
| **C·决策** | CAC/LTV模型是否健康？IRR（内部收益率）是否满足门槛？ | 财务+法务 |

## 模块四：评估章节结构（Gate结构）

### ▶ One-pager：消费品终局先立（锚定门）

**目标**：在进入任何数据分析之前，先锚定产品的消费品终局图景。消费品评估必须以"消费者终局"而非"临床数据"为起点。

必填要素：
1. **消费者一句话定位**：`[人群X] 在 [场景Y] 下有 [痛点Z]，[产品] 解决它，比 [竞品] 更好的原因是 [差异化W]`
2. **5年品牌愿景**：在目标消费者心中占据什么位置？参考品牌：___（如养元/汤臣倍健/修正）
3. **收入峰值预测**：第3年/第5年峰值收入（万元人民币）
4. **核心渠道押注**：主渠道（电商/药店/商超）+ 辅助渠道
5. **One-pager决策**：通过 → Gate 1 ｜ 暂缓 → 重新定位 ｜ 否决 → 归档

---

### ▶ Gate 1：品类合规准入过滤（前提门）

**核心问题**：产品能否合法在中国市场销售？是否触碰品类重叠红线？

**评估维度**：

| 维度 | 评估内容 | 权重 |
|------|----------|------|
| 1-1 监管分类确认 | OTC药品/保健品/普通食品/化妆品；分类影响广告允许内容 | 35% |
| 1-2 品类重叠检查 | 是否与B-1医美或A群处方药管线重叠 | 30% |
| 1-3 进口合规 | 进口食品/化妆品备案；跨境电商合规路径 | 20% |
| 1-4 广告法合规预审 | 禁止使用"最""第一""治愈"等绝对化用语 | 15% |

**Gate 1 一票否决触发项**：
- [ ] 产品监管分类属于处方药（Rx），不在OTC/消费品范围内
- [ ] 与B-1医美或A群处方药管线存在直接品类重叠
- [ ] 含中国禁用原料或成分
- [ ] 无合法进口渠道（跨境/一般贸易均不可行）

---

### ▶ Gate 2：品牌差异化定位（定调门）

**核心问题**：在拥挤的消费健康市场中，这个产品凭什么让消费者记住并复购？

**评估维度**：

| 维度 | 评估内容 | 权重 |
|------|----------|------|
| 2-1 差异化来源 | 成分专利/效果可感知性/包装设计/品牌故事 | 35% |
| 2-2 竞品景观密度 | 同类竞品数量/TOP3份额集中度/可进入空间 | 30% |
| 2-3 品牌可传播性 | 名字是否好记/故事是否适合社交媒体传播 | 20% |
| 2-4 价格段位清晰度 | 高端/中端/大众；价格段是否有空间 | 15% |

**定位输出模板**：
```
产品定位陈述：
[产品名] 是面向 [人群] 的 [品类] 产品。
与 [竞品A/B] 不同，我们的核心差异是 [X]。
我们的定价策略是 [价格段]，核心渠道是 [渠道]。
5年品牌目标：在 [细分市场] 成为 TOP [N] 品牌。
```

---

### ▶ Gate 3：消费者需求验证（证据门）

**核心问题**：消费者真的想要这个产品吗？愿意支付多少？会复购吗？

**验证方式优先级**：

| 方式 | 可信度 | 成本 | 周期 |
|------|--------|------|------|
| 线上真实销售数据（竞品参考） | 极高 | 低 | 1周 |
| 消费者焦点小组（FGD） | 高 | 中 | 2-3周 |
| 第三方问卷调研（N≥200） | 中 | 低 | 1-2周 |
| 社交媒体聆听（小红书/抖音） | 中 | 低 | 1周 |
| 专家访谈 | 低（供参考） | 低 | 1周 |

**必须完成的验证**（至少2项）：
- [ ] 竞品线上销售数据分析（月销量/评价分析）
- [ ] 消费者调研（N≥200，覆盖目标人群）
- [ ] 社交媒体舆情分析（小红书/抖音相关内容量和情感倾向）

**输出指标**：
- 消费者NPS（净推荐值）：目标≥50
- 复购意愿：目标≥60%表示"肯定会/可能会"复购
- 价格接受度：目标≥70%在目标定价区间表示"愿意购买"

---

### ▶ Gate 4：渠道适配与市场容量（支付门）

**核心问题**：目标渠道的容量够大吗？产品能在渠道中存活吗？

**渠道适配矩阵**：

| 渠道 | 市场规模（亿元） | 进入难度 | 毛利率影响 | 适配评分 |
|------|---------------|---------|-----------|---------|
| 天猫旗舰店 | [填写] | 中 | -10%~-15%（佣金） | |
| 京东自营 | [填写] | 高 | -8%~-12% | |
| 抖音电商 | [填写] | 低-中 | -20%~-30%（达人佣金） | |
| 连锁药店（国大/大参林/益丰） | [填写] | 高 | -30%~-40%（进场费+陈列） | |
| 商超（大润发/盒马） | [填写] | 高 | -35%~-45% | |
| 社区团购 | [填写] | 低 | -25%~-35% | |
| 私域（微信/小程序） | [填写] | 低 | -5%~-10% | |

**TAM（总可寻址市场）/SAM（可服务市场）/SOM（可获得市场）测算**：
- TAM：整体品类市场规模（行业报告来源）
- SAM：德镁渠道可覆盖的细分市场
- SOM：基于渠道就绪度的3年可实现目标

---

### ▶ Gate 5：CAC（客户获取成本）/COGS（销售商品成本）结构（成本门）

**核心问题**：单位经济模型是否健康？CAC能否被LTV（用户生命周期价值）覆盖？

**单位经济模型（Unit Economics）**：

| 指标 | 中文全称 | 计算公式 | 目标值 |
|------|----------|----------|--------|
| CAC | 客户获取成本 | 总营销费用 ÷ 新增客户数 | ≤首单收入的50% |
| LTV | 用户生命周期价值 | 客单价 × 年复购次数 × 客户生命周期年数 × 毛利率 | ≥3×CAC |
| LTV/CAC | 用户价值获客比 | LTV ÷ CAC | ≥3倍 |
| Payback Period | 回收期 | CAC ÷ （月均毛利贡献） | ≤12个月 |
| Gross Margin | 毛利率 | （收入-COGS）÷ 收入 | ≥55%（消费品门槛低于医美） |
| COGS | 销售商品成本 | 含采购/运输/关税/仓储 | ≤45%收入占比 |

**营销费用结构**：

| 营销项目 | 预算占比 | 说明 |
|----------|---------|------|
| 达人/KOL种草 | [填写]% | 小红书/抖音 |
| 平台投流（信息流广告） | [填写]% | ROI目标≥3 |
| 私域运营 | [填写]% | 微信/企业微信 |
| 线下地推/体验活动 | [填写]% | |
| 品牌公关/媒体 | [填写]% | |

---

### ▶ Gate 6：IRR（内部收益率）与规模化可行性（可做门）

**核心问题**：综合财务回报是否满足院外业务投资门槛？规模化路径是否清晰？

**消费健康财务门槛**（与医美不同，因消费品特性适当调整）：

| 指标 | 中文全称 | 院外消费健康门槛 | 说明 |
|------|----------|---------------|------|
| IRR | 内部收益率 | ≥40%（低于医美，因规模效应更强） | 5年现金流 |
| MOIC | 投资回报倍数 | ≥8-15倍 | 低于医美，规模补偿 |
| Payback Period | 回收期 | ≤3年 | 消费品周期略长 |
| 首年盈利 | — | 可接受小额亏损（≤初始投入20%） | 品牌建设期 |
| 3年峰值收入 | — | ≥500万元（否则规模不经济） | 最低商业规模阈值 |

**规模化路径评估**：
- 渠道复制性：从1个渠道复制到3+渠道的可行性
- 品类延伸性：未来能否推出SKU系列（产品矩阵）
- 数据资产积累：复购数据能否支撑私域运营

## 模块六：一票否决清单

- [ ] **R-01** 产品监管分类为处方药（Rx），不在OTC/消费品范围
- [ ] **R-02** 与B-1医美或A群处方药管线存在直接品类重叠（品类纯洁性原则）
- [ ] **R-03** Gate 3消费者验证NPS<30，复购意愿<40%
- [ ] **R-04** LTV/CAC比值<2（单位经济模型不可持续）
- [ ] **R-05** 基准情景IRR（内部收益率）<25%，且无可优化路径
- [ ] **R-06** 3年峰值收入预测<300万元（规模不经济）
- [ ] **R-07** 主渠道进入壁垒不可逾越（如连锁药店拒绝陈列）
- [ ] **R-08** 含虚假宣传硬伤（无法通过物料调整修复）

## 模块七：Gate阶段说明

### Gate流程路径

```
One-pager（锚定门）
       │
       ▼
Gate 1（前提门）── 监管/品类红线 → [触发任一否决 → 立即终止]
       │
       ▼
Gate 2（定调门）── 差异化评分 < 60 → [建议放弃，可申诉]
       │
       ▼
Gate 3（证据门）── NPS < 30 → [终止；NPS 30-50 → 附条件推进]
       │
       ▼
Gate 4（支付门）── SOM < 300万 → [规模不经济，终止]
       │
       ▼

       │
       ▼
Gate 6（可做门）── IRR < 25% → [终止]
       │
       ▼
[提交院外业务委员会终审]
```

## 模块九：输出规范

### 9.1 报告结构

```
封面页
├── 执行摘要（消费者视角，300字）
├── 产品基本信息（M-01~M-10汇总）
├── One-pager消费品终局画像
├── Gate 1-6逐门评估结果
├── 单位经济模型（CAC/LTV/LTV:CAC）
│   ├── 基准/乐观/悲观三情景
│   └── 渠道拆分（电商/药店/其他）
├── 财务模型（5年P&L预测）
│   ├── 收入驱动因子分解
│   ├── 营销费用预算
│   └── IRR/MOIC/回收期
├── 渠道适配矩阵（热力图）
├── 竞品定位图（价格×差异化二维坐标）
├── 一票否决核查表
├── 三级验收标准
├── 规模化路径建议（100天启动计划）
└── 附录
```

### 9.2 格式要求
- **语言**：简体中文，财务英文缩写须标注中文全称
- **格式**：麦肯锡风格HTML报告
- **特色图表**：消费者旅程图、LTV/CAC健康仪表盘、渠道适配热力图
- **字数**：≥5000字

### 9.3 三级验收标准

| 级别 | 必达项（Must-have） | 参考项（Nice-to-have） | 否决项（Deal-breaker） |
|------|-------------------|---------------------|---------------------|
| L1 | NPS≥50；LTV/CAC≥3 | 社媒自然流量潜力 | 监管分类为处方药 |
| L2 | 3年峰值≥500万 | 私域运营数据积累 | 品类重叠B-1/A群 |
| L3 | IRR≥40% | KOL合作意向 | IRR<25% |
