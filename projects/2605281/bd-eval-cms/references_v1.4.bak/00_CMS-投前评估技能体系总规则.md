## 1. 体系概述

### 1.1 体系规模与构成

本体系由 **20个独立技能** 与 **1个战略合作治理增效工具包**（非独立挂载）共同构成，覆盖CMS集团全业务条线的投前评估全流程。

```
总体结构
├── D群 × 4   —— 基座与工作流（横向底座，支撑所有技能）
├── A群 × 9   —— 中国市场药品评估（核心业务主干）
├── B群 × 3   —— 消费属性产品评估（医美+消费健康）
├── E群 × 1   —— 股权投资评估
├── C群 × 3   —— 国际市场评估（港澳台/东南亚/GCC/澳新/拉美）
└── 战略合作治理增效工具包（非独立）
```

### 1.2 三框架融合原则

所有19个技能均在三框架融合的底层架构上运行，不允许任何技能绕过任一层：

| 框架层级 | 名称 | 核心功能 |
|---|---|---|
| **骨架层** | 终审模板 v6.2 | 定义输出结构、必输字段、一票否决项、Gate结论格式 |
| **内容层** | 投前尽调14维度 | 规定评估维度的覆盖完整性与深度标准 |
| **方法论层** | 麦肯锡8步法 | 问题结构化→假设树→数据驱动→叙事线→交付物 |

> **强制规则**：三层框架缺一不可。任何单技能如仅满足其中一层或两层，视为不合规输出，须在当轮标注"框架缺失项"并终止推进。

### 1.3 体系设计原则

1. **以终为始**：每项评估必须先定义成功终局（One-pager），再倒推验证路径。
2. **Gate即止损**：任一Gate未达标，当轮停止推进，不允许"带着疑点往下走"。
3. **财务硬编码**：第5节财务内控标准为硬编码基准，不可在单技能层面自行降低门槛。
4. **模块化串接**：每个技能暴露标准串接接口，支持技能链组合调用。
5. **输出一致性**：全体系统一采用第10节输出规范，不允许各技能自定义风格。

## 3. Gate底层逻辑（以终为始·6-Gate框架）

### 3.1 框架来源

以 **A-8（天津康哲仿制药/生物类似药评估技能）林总框架**为蓝本，抽象为体系级通用Gate逻辑，所有19个技能统一采用。

### 3.2 评估起点：One-pager（终局先立）

在任何Gate启动前，必须完成One-pager，明确以下五项：

| 字段 | 内容要求 |
|---|---|
| 成功终局定义 | 上市后第X年峰值销售额、市占率目标、对集团贡献 |
| 核心价值主张 | 凭什么在目标市场胜出（1-2句话，不允许罗列） |
| 最大风险单项 | 若只有一个风险能击垮本项目，是什么 |
| 止损边界 | 在何种条件下应主动退出或终止 |
| 决策人与时限 | 谁来拍板，最晚何时须给出结论 |

### 3.3 六道Gate详解

#### Gate 1 — 前提门（Premise Gate）
**核心问题**：项目前提是否成立？  
**必审项**：
- 合作方主体资质（注册、监管记录、诉讼史）
- 产品在中国/目标市场的法律可行性
- CMS集团是否存在利益冲突（现有管线重叠）
- NDA保密协议状态

**输出**：前提通过 / 前提未达（列明缺失项）  
**未达处理**：当轮终止，不得进入Gate 2。

---

#### Gate 2 — 定调门（Positioning Gate）
**核心问题**：产品定调是什么？定调决定整个商业模式。  
**必审项**：
- 产品属性（首创/同类最优/跟随/仿制/改良/医美/消费健康）
- 目标适应症与患者群体精准划定
- 竞争格局初判（现有竞品数量、市场集中度）
- 拟议合作模式（独家代理 / 联合开发 / 股权投资 / 4.0战略）

**输出**：定调结论 + 置信度（高/中/低）+ 需补证据Top 5  
**未达处理**：定调存在根本分歧时，当轮停止，提交决策人裁定。

---

#### Gate 3 — 证据门（Evidence Gate）
**核心问题**：价值主张有无硬数据支撑？  
**必审项**：
- 临床数据质量评级（A/B/C）
- 与SOC（标准治疗方案）的差异化证据
- 中国/目标市场相关研究或bridging数据
- KOL意见（至少2位三甲医院专家背书）
- 真实世界数据（RWE）可获取性

**输出**：证据强度评级 + 置信度 + 需补证据Top 5  
**未达处理**：证据D级或低置信度，当轮停止，要求合作方补充资料。

---

#### Gate 4 — 支付门（Market Access Gate）
**核心问题**：目标市场能否实现有效支付？  
**必审项**：
- 医保准入路径（国谈 / 省增补 / 自费 / 商保）
- 定价策略（参考价 / 自主定价空间）
- 患者支付能力与意愿调研
- VBP（带量采购）风险敞口评估
- 国际市场：当地医保/HTA评审体系对标

**输出**：支付可行性结论 + 置信度 + 需补证据Top 5  
**未达处理**：支付路径不可行时，当轮停止并评估改变商业模式可能性。

---

#### Gate 5 — 成本门（Cost Gate）
**核心问题**：成本结构是否支撑目标利润率？  
**必审项**：
- BOM级成本拆解（原料药 / 辅料 / 制造 / QC / 包装）
- 注册成本（CDE / FDA / EMA / 各市场监管费用）
- 市场推广费用预算（SG&A占比）
- 供应链单点失效风险
- 汇率敞口（非人民币市场）

**输出**：成本结构验证 + 毛利率预估 + 置信度 + 需补证据Top 5  
**未达处理**：毛利率低于本体系第5节硬编码下限时，当轮停止，不允许以"战略意义"为由豁免。

---

#### Gate 6 — 可做门（Dealability Gate）
**核心问题**：交易在法律、财务、退出机制上是否可执行？  
**必审项**：
- 交易结构（TS/LOI 核心条款）
- 财务回报测算（IRR / 回收期 / 倍数，对照第5节标准）
- 止损条款完整性（里程碑对赌 / 退出触发条件）
- 知识产权归属与排他条款
- 交割条件与时间表

**输出**：可行性最终结论 + 置信度 + 推进建议  
**全通过处理**：生成投委会决策包，进入阶段C终审。

### 3.4 Gate通用输出格式

```
【Gate X — XXX门 结论卡】
结论：通过 / 有条件通过 / 未达标
置信度：高 / 中 / 低
主要支撑依据（≤3条）：
1. ...
2. ...
3. ...
需补证据 Top 5：
1. ...
2. ...
3. ...
4. ...
5. ...
当轮处理决定：推进至Gate X+1 / 停止当轮 / 提交决策人裁定
```

## 4. 投前评估三阶段流程模型

### 4.1 阶段A — 商机发现与立项决策

```
商机捕获
    └─► Gate 0 初筛（A-0技能：商机雷达与初筛）
            └─► 具体产品初审评估报告（A-1至A-8 / B-1 / B-3 / C-1 / E-1 按产品类型选取）
                    └─► 立项会决策（Go / No-Go / 附条件Go）
```

**阶段A关键节点说明**：
- Gate 0初筛须在72小时内完成，输出"初筛一页纸"（不超过500字）。
- 初审评估报告覆盖M-01~M-10（第6节公司通用模块）+ Gate 1~Gate 3。
- 立项会须有投前负责人、对应业务主体负责人、财务代表三方同时在场。

### 4.2 NDA弹性签署节点

> **重要**：NDA（保密协议）签署节点**不固定于特定阶段**，遵循以下弹性规则：

| 情形 | NDA签署时机 |
|---|---|
| 合作方主动提供完整数据包 | 数据交换前签署 |
| 合作方有内部数据库可授权访问 | 签署NDA后即可获取，无需等待阶段B |
| 非竞争性公开数据 | 可先行研究，NDA在进入实质谈判前签署 |
| 公开上市公司信息 | 无需NDA |

签署NDA并不代表项目正式立项，亦不构成对合作的任何承诺。

### 4.3 阶段B — 立项后双轨并行

```
立项批准
    ├─► 技术深化评估线（Gate 3~Gate 5）
    │       ├── 临床与注册深度尽调
    │       ├── 生产与供应链验证
    │       └── 市场与准入精细化建模
    │
    └─► 商务谈判线（TS/LOI草案拟定）
            ├── 财务进场节点：TS阶段（不早于TS草案出具）
            ├── 法务进场节点：LOI签署前
            └── 双线互动：技术评估结论实时反哺谈判条款
```

**双轨并行原则**：
- 两条线独立推进，但须在每周例会中同步进展。
- 技术线重大发现（如临床数据造假、专利障碍）须立即暂停谈判线，等待技术结论更新。
- 商务线重大让步（如独家条款放弃、里程碑大幅降低）须触发技术线重新评估财务模型。

### 4.4 阶段C — 终审决策

```
双线收敛
    └─► D-3技能：生成投委会决策包
            └─► Gate 6 投委会审议
                    ├─► 批准：进入协议谈判与签署
                    ├─► 条件批准：列明附加条件，设定条件达成时限
                    └─► 否决：归档，说明否决理由，可设置"重新提交窗口期"
```

**投委会决策包必备模块**：
1. One-pager终局摘要
2. Gate 1~Gate 6结论卡汇总
3. 财务回报敏感性分析（基准/乐观/悲观三情景）
4. 一票否决项最终核查表
5. 交易结构草案
6. 推荐意见（Go / No-Go / 有条件Go）

## 5. 财务评估内控标准

> **硬编码声明**：本节所有财务门槛为**体系级硬编码**，不可在单技能层面自行调整。如确有特殊情形需要豁免，须经集团CFO及投前委员会联合书面审批，并在评估报告中明确标注豁免原因。

> **财务缩写注释规范**（适用全文件）：IRR（内部收益率）/ EBITDA（税息折旧摊销前利润）/ FCF（自由现金流）/ CAPEX（资本支出）/ NPV（净现值）/ rNPV（风险调整后净现值）/ BOM（物料清单）/ SG&A（销售费用及管理费用）/ ROI（投资回报率）/ ROIC（投入资本回报率）/ VBP（带量采购）/ HTA（卫生技术评估）/ SOC（标准治疗方案）/ EPS（每股收益）/ DSO（应收账款周转天数）/ TS（条款清单）/ LOI（意向书）/ NDA（保密协议）/ CDE（国家药品审评中心）。

### 5.1 创新药财务内控标准（含A-3自研项目）

| 风险投资额 | 上市剩余年限 | IRR（内部收益率）| 回收期 | 5年倍数 | 10年倍数 | 毛利率 | 净利润率 | 盈亏平衡 |
|---|---|---|---|---|---|---|---|---|
| ≤3亿元 | 上市≤3年 | ≥15% | ≤5年 | 2倍 | 4倍 | ≥60% | 10年均≥20% | 上市后3年累计 |
| ≤3亿元 | 上市3-5年 | ≥20% | ≤5年 | 2.5倍 | 6倍 | ≥60% | 10年均≥20% | 上市后3年累计 |
| ≤3亿元 | 上市5年以上 | ≥22% | ≤5年 | 2.7倍 | 7倍 | ≥60% | 10年均≥20% | 上市后3年累计 |
| >3亿元 | 上市≤3年 | ≥20% | ≤5年 | 2.5倍 | 6倍 | ≥60% | 10年均≥20% | 上市后3年累计 |
| >3亿元 | 上市3-5年 | ≥22% | ≤5年 | 2.7倍 | 7倍 | ≥60% | 10年均≥20% | 上市后3年累计 |
| >3亿元 | 上市5年以上 | ≥25% | ≤5年 | 3.1倍 | 9倍 | ≥60% | 10年均≥20% | 上市后3年累计 |

**创新药补充说明**：
- "风险投资额"指从签署合作协议至产品首次在目标市场获批上市所需的全部投入（含里程碑付款、注册费用、临床投入、上市准备费用），不含上市后营销费用。
- 上市剩余年限从协议签署日起算至预计获批上市日。
- IRR（内部收益率）以税后、扣除全部投入为基础计算，折现率基准为集团加权平均资本成本（WACC）。
- 毛利率指稳定商业化阶段（上市后第3年起）的毛利率下限。

### 5.2 仿制药财务内控标准

| 品种类型 | IRR（内部收益率）| 回收期 | 5年倍数 | 净利润率 | 盈亏平衡 |
|---|---|---|---|---|---|
| 一般化学仿制药 | ≥50% | ≤3年 | 3倍 | 5年均≥20% | 上市首年盈利 |
| 短期不纳入集采品种 | ≥35% | ≤3.5年 | 4倍 | 5年均≥20% | — |
| 改良型新药（4类）| ≥30% | ≤4年 | 3.5倍 | 5年均≥20% | 上市2年内盈利 |
| 生物类似药 | ≥25% | ≤5年 | 3倍 | 10年均≥20% | 上市3年累计 |

**仿制药补充说明**：
- VBP（带量采购）风险必须在Gate 4中单独量化，建议敏感性测试覆盖"当期不集采/3年内集采/上市即集采"三个情景。
- 短期不集采品种须附专家论证意见，说明判断依据及结论有效期（通常不超过3年）。
- 生物类似药适用创新药毛利率标准（≥60%），而非仿制药宽松标准。

### 5.3 医美产品财务内控标准

| 总投资额 | 至上市/上线年限 | IRR（内部收益率）| 回收期 | 5年倍数 | 毛利率 |
|---|---|---|---|---|---|
| ≤1亿元 | ≤2年 | ≥50% | ≤2年 | 15倍 | ≥75% |
| ≤1亿元 | >2年 | ≥55% | ≤2年 | 20倍 | ≥75% |
| >1亿元 | ≤2年 | ≥55% | ≤2年 | 25倍 | ≥75% |
| >1亿元 | >2年 | ≥60% | ≤2年 | 30倍 | ≥75% |

**医美产品补充说明**：
- 医美产品毛利率门槛（≥75%）显著高于药品类，体现医美赛道的高溢价属性。
- 5年倍数要求远高于药品类，反映医美行业对快速资本回报的高预期。
- 对于器械类医美产品，"总投资额"包含注册检测费、样机采购/引进费用及推广培训费。
- 单纯消费健康类（非医疗器械/药品）参照B-3技能的专项财务基准。

### 5.4 财务模型通用规则

1. **双情景基准**：所有财务模型须输出"基准情景"和"压力情景"，压力情景假设销售峰值降低30%、成本上涨15%。
2. **汇率处理**：非人民币市场交易须采用当日中间价作为基准汇率，敏感性测试覆盖±10%汇率波动。
3. **折现率**：创新药项目折现率建议12-15%；仿制药10-12%；医美产品15-20%（视风险水平调整）。
4. **税务假设**：中国大陆项目默认适用高新技术企业15%所得税税率；国际项目须在报告中明确当地税率假设。
5. **rNPV（风险调整后净现值）适用范围**：临床阶段项目（I期/II期/III期）必须采用rNPV模型，不允许直接使用NPV（净现值）。

## 8. 战略合作治理增效工具包（非独立技能）

### 8.1 定位说明

本工具包**非独立技能**，不可单独触发，必须通过"签约后治理链"挂载于任一主技能之上。适用于协议签署后合作执行期的全生命周期治理。

### 8.2 五大工具详述

#### 工具1：Issue Tree（问题树）
- **用途**：结构化拆解合作中的核心问题，确保团队对问题的认知一致
- **格式要求**：MECE原则，3层以上，叶节点可赋责任人
- **触发场景**：合作执行期出现方向性争议、目标偏差或跨部门协同障碍时

#### 工具2：IP归属三维矩阵
- **用途**：厘清合作中产生的知识产权（IP）归属
- **三维度**：资金出资方 × 技术贡献方 × 使用权归属
- **输出格式**：3×3矩阵表 + 合同语言建议
- **触发场景**：联合开发项目、合作研究协议（SRA）签署前后

#### 工具3：JSC-JPT治理框架
- **JSC**（联合指导委员会）：季度战略层级，成员含双方VP级以上
- **JPT**（联合项目团队）：月度执行层级，成员含双方项目负责人
- **输出内容**：会议章程模板 / 决策权限矩阵 / 升级路径 / KPI仪表盘
- **触发场景**：独家合作协议签署后，商业化合作正式启动时

#### 工具4：退出机制模板
- **触发条件列表**（以下任一条件成立即可触发退出）：
  - 连续2个里程碑未达成
  - 合作方关键人员（CEO/CSO）离职且90天内未补充
  - 监管机构发出重大合规警告
  - 双方对下一阶段开发方向存在无法调和的实质分歧
- **退出方式**：全面退出 / 缩减合作范围 / 转让合作权益
- **输出内容**：退出通知模板 / 资产分割方案 / 保密延续义务条款

#### 工具5：投委会审议决策包
- **适用场景**：阶段C终审、追加投资审批、重大合同条款变更审议
- **必备内容**：One-pager摘要 / Gate结论卡汇总 / 三情景财务模型 / 一票否决核查表 / 推荐意见
- **格式**：不超过20页PPT或等效HTML报告
- **时限要求**：投委会召开前48小时完成并送达所有委员
