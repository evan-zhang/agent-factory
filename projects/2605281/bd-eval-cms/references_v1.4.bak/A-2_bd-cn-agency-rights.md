
### ▶ Gate 1：产品力现状评估（前提门）

**核心评估问题**：
- 产品在目标适应症中的临床地位是否仍然稳固（未被新一代药物边缘化）？
- 专利状态：核心化合物专利是否已到期或即将到期（决定仿制药竞争压力）？
- 说明书适应症与现实推广的差距（超适应症风险评估）？
- 是否存在药物安全性风险（未解决的ADR（不良反应）警示、黑框风险）？
- 产品是否被纳入重点监控目录（影响院内使用政策）？

**产品力评分（1-5分）**：

| 评分维度 | 权重 | 评分 | 备注 |
|---------|------|------|------|
| 临床价值持久性（vs. 新一代替代品） | 35% | | |
| 专利保护剩余年限 | 20% | | |
| 安全性风险可控程度 | 25% | | |
| 政策风险（重点监控/一致性评价） | 20% | | |

### ▶ Gate 2：渠道资产与迁移可行性（定调门）

**A-2核心特有评估维度**：

**2.1 渠道资产盘点**
- 医院覆盖数量（一级/二级/三级医院分布）
- 覆盖科室数量及核心KOL（关键意见领袖）名单
- 现有一级经销商名录及回款周期
- 现有处方医生数量（活跃处方医生定义：近6个月≥1次处方）

**2.2 流向数据真实性核查**
```
核查维度              核查方式
月度销量波动是否合理  近24个月流向数据趋势分析（排除季节性异常）
终端发货与消耗匹配度  库存周转天数与行业均值对比
价格体系一致性        出厂价/批发价/终端价是否存在倒挂
经销商回款周期        平均DSO（应收账款周转天数）是否在行业正常范围
```

**2.3 团队交接可行性**
- 现有销售团队可随行比例（协议约定/意愿评估）
- 医学代表区域分布与集团现有团队重叠度
- 随行团队薪酬对标（接管后是否需要大幅调整）
- 关键大区经理/医学联络官（MSL（医学科学联络官））随行意向

**2.4 品牌认知迁移风险**
- 处方医生对「产品品牌」的认知是归属于「原推广方品牌」还是「产品本身」？
- 如果是前者，迁移后品牌混乱风险有多大？
- 原推广方是否会推同类竞品，形成「品牌迁移干扰」？

## 模块六：一票否决清单

1. **产品力严重衰退**：被新一代药物实质性替代，市场份额年降幅≥30%连续2年
2. **批文合规性存疑**：批准文号存在法律瑕疵或正在被NMPA复查
3. **流向数据严重失真**：发现流向数据造假迹象（库存倒打/发票异常）
4. **渠道迁移比例<30%**：基准情景下渠道迁移比例低于30%，商业化基础不存在
5. **集采纳入窗口期<12个月**：预计接管后12个月内即被纳入全国集采
6. **IRR（内部收益率）<15%**：基准情景IRR（内部收益率）低于代理权最低门槛
7. **现推广方并发竞品**：现推广方在交接期间同步推同适应症竞品，形成直接市场干扰
8. **独家性无法锁定**：合同无法保障全国代理独家权，存在多头代理风险

```
A-2评估完成

```

## 模块九：输出规范

- **主报告格式**：HTML（深蓝 `#1a3a5c`，白底，无灰色字，无元描述）
- **麦肯锡风格**：结论先行（Action Title），数据支撑置于正文
- **财务缩写标注**：IRR（内部收益率）、CAGR（复合年均增长率）、PeakSales（峰值销售额）、EBIT（息税前利润）、SG&A（销售、一般及行政费用）、MBQ（最低采购量）、DSO（应收账款周转天数）等均须括号标注中文全称
- **双币标注**：非人民币金额同步标注人民币（注明汇率换算日期）
- **报告章节顺序**：
```
封面
执行摘要（结论 + 推荐行动）
第1章 产品概述与市场背景
第2章 产品力现状评估（Gate 1）
第3章 渠道资产与迁移可行性（Gate 2）
第4章 接管后市场增长空间（Gate 3）
第5章 价格·医保·准入（Gate 4）
第6章 团队与运营成本（Gate 5）
第7章 交易结构与财务回报（Gate 6）
第8章 一票否决核查与综合结论
附录A 流向数据分析附表
附录B 销售团队结构分析
附录C 竞品对比矩阵
参考文献
```
