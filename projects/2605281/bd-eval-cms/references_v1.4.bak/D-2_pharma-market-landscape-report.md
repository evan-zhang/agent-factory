## 模块2：通用前置章节（公司基本信息 M-01 ~ M-10）

> D-2 作为市场分析技能，M-01~M-10 以「市场视角」采集，侧重疾病/适应症背景而非合作方公司信息。

| 编号 | 字段名称 | D-2 采集重点 | 数据来源优先级 |
|------|----------|------------|--------------|
| M-01 | 目标疾病/适应症 | ICD-11 编码 + 流行病学分级（发病率/患病率/死亡率） | WHO/国家卫健委/PubMed |
| M-02 | 目标地理市场 | 具体到省/城市分级（如适用）；国际市场须注明货币基准 | 集团目标市场清单 |

| M-04 | 评估时间基准年 | 报告基准年 + 预测终止年（通常 +10年） | 当前年份确认 |
| M-05 | 竞品范围边界 | 直接竞品（同机制）+ 间接竞品（同适应症）+ 新兴竞品 | 公开数据库 + 用户确认 |

## 模块4：评估章节结构（15章三合一报告）

### 总体结构

```
Part I：市场全景（第1~6章）
  ——回答"这个市场有多大、竞争格局如何"

Part II：患者分布（第7~10章）
  ——回答"目标患者在哪里、如何分层触达"

Part III：专科渠道深度（第11~14章）
  ——回答"如何通过渠道实现商业化"

第15章：战略建议与 X+Y+N 上市策略
  ——整合前14章输出，给出可操作建议
```

### Part I：市场全景

#### 第1章 疾病背景与流行病学

| 内容模块 | 核心产出 |
|---------|---------|
| 疾病定义与分型（ICD-11 编码） | 明确评估范围边界 |
| 发病率/患病率/死亡率（目标市场，最近5年趋势） | 市场规模计算基础 |
| 疾病负担（DALY，失能调整生命年；医疗花费） | 支撑医保准入论据 |
| 未满足临床需求（Unmet Medical Need）分析 | 产品差异化定位基础 |

| 内容模块 | 核心产出 |

### 第15章：战略建议与 X+Y+N 上市策略

#### X+Y+N 框架定义

```
X = 核心定位优势（产品差异化竞争优势，最多3条）
Y = 优先目标细分市场（患者分层 × 渠道交叉，给出TOP 3细分）
N = 上市后扩张节点（适应症扩展/渠道扩张/地域扩张，分阶段规划）
```

| 策略组件 | 内容 | 呈现方式 |
|---------|------|---------|
| 产品差异化定位（X） | 基于竞品格局的差异化卡位（临床/定价/便利性） | 定位矩阵图 |
| 目标细分市场（Y） | 优先攻取的患者群/医院群/城市群 | 优先级热力图 |
| 扩张路线图（N） | 12/24/36个月关键里程碑 | 时间轴 |
| 商业模式推荐 | 自建/合作/联合推广的最优方案 | 决策树 |
| 关键成功要素（KSF，关键成功因素）Top 5 | 决定商业成败的5个核心要素 | 重要性-可控性矩阵 |
| 关键风险与应对预案 | Top 3 风险 + 应对方案 | 风险登记表 |

## 模块6：一票否决清单

> D-2 为市场分析技能，以下市场层面发现须立即上报 D-1 红旗风险台账，并可触发软性否决讨论。

| 编号 | 市场层面红旗 | 触发阈值 | 处理方式 |
|------|------------|---------|---------|
| MK-01 | 目标市场 SAM（可服务市场）预测 < 集团财务门槛 | 低于对应品类门槛 50% | 上报 D-1，触发 Gate 3 否决讨论 |
| MK-02 | 直接竞品已占据 >70% 市场份额，无差异化空白 | 份额集中度 HHI（赫芬达尔指数）>2500 且无细分机会 | 上报 D-1，要求重新定义 Y（目标细分） |
| MK-03 | 产品适应症已纳入或计划纳入 VBP（带量采购），价格崩塌风险高 | VBP 纳入概率 >70%（专家评估） | 上报 D-1，进行 VBP 情景下财务压力测试 |
| MK-04 | 核心 KOL 已被竞品锁定（独家顾问协议），推广渗透成本极高 | Top 10 KOL 中 >7人已有竞品独家关系 | 上报 D-1，制定差异化 KOL 战略 |
| MK-05 | 目标市场医保报销政策存在重大不确定性（改革窗口期） | 政策变动概率 >50%（政策顾问评估） | 列为市场风险，要求政策情景分析 |

## 模块9：输出规范

### HTML 报告视觉规范

**色彩体系：**
- 主色：深蓝 `#1a3a5c`（页眉、章节标题栏、表头）
- 辅色1：浅蓝 `#2980b9`（二级标题、图表系列1）
- 辅色2：橙黄 `#f39c12`（关键数字标注、警示框）
- 辅色3：浅绿 `#27ae60`（正向指标、机会标记）
- 背景色：白 `#ffffff`；交替行：极浅蓝 `#f0f5fb`
- 禁用：任何灰色文字（`#666666` 及以下），灰色表格背景
- 禁用：元描述段落（如"本报告旨在……"、"以下将分析……"）

**排版规范：**
- 正文字号：14px（中文宋体/微软雅黑，英文 Arial/Helvetica）
- 标题层级：H1 = 24px 加粗 / H2 = 20px 加粗 / H3 = 16px 加粗
- 数字字体：等宽字体，右对齐
- 图表：每章至少1张可视化（柱状图/饼图/折线图/热力图），图表标题规范为"图X-X: [内容]"

**内容规范：**
- 所有财务缩写首次出现须标注中文全称：CAGR（复合年均增长率）、TAM（总可及市场）、SAM（可服务市场）、SOM（可获得市场）、ICER（增量成本效益比）、DALY（失能调整生命年）、HTA（健康技术评估）、HEOR（健康经济与结果研究）、NRDL（国家医保目录）、VBP（国家带量采购）、KOL（关键意见领袖）、DTP（直达患者药房）、PAP（患者援助计划）、KSF（关键成功因素）、RWE（真实世界证据）、HHI（赫芬达尔指数）
- 非人民币市场须双币标注：如"USD 1.2B（约 RMB 87亿元，2025年汇率 7.25 折算）"
- 数据须注明来源（脚注格式：数据来源：[机构名称，年份]）
- 预测数据须注明基准假设与情景说明

**章节顺序（15章完整版）：**
```
第1章  疾病背景与流行病学
第2章  总体市场规模与增长
第3章  竞品格局与市场份额
第4章  监管与政策环境
第5章  药物经济学与卫生技术评估（HTA）
第6章  标杆市场对比（如适用）
第7章  患者分层与画像
第8章  患者地理分布
第9章  诊断率与诊疗路径分析
第10章 患者支付与保险覆盖
第11章 KOL 分级体系与影响力地图
第12章 医院渠道分析
第13章 院外与新兴渠道
第14章 竞品商业化策略解码
第15章 战略建议与 X+Y+N 上市策略
附录A  数据来源与方法论说明
附录B  术语表与缩写中文全称对照
附录C  参考文献
```

> 附录须排列于全部正文章节（第1~15章）之后，不得将附录插入任何正文章节之间。
