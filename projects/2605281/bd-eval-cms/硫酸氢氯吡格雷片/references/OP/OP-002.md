# [OP-002] 信立泰2024年报关键信息：代理合作模式与泰嘉定位
- **URL**: https://file.finance.qq.com/finance/hs/pdf/2025/03/29/1222946946.PDF（2024年报摘要）
- **抓取时间**: 2026-05-31 14:06 CST
- **来源类型**: 上市公司年报

## 关键数据点
- 信立泰在抗血小板聚集领域，泰嘉定位描述为成熟品种
- 2024年报明确提到："在部分区域，采用代理合作等模式，借助品牌力量拓展外围市场覆盖"
- 信立泰销售模式分为：学术推广、代理分销及直接销售
- 创新药采用学术推广模式；仿制药部分区域采用代理分销
- 2026年4月信立泰年报收入分类调整：从"直接推广销售"与"代理推广销售"改为"直接销售"与"向经销商销售"

## 对本评估的意义
- 信立泰已在部分区域对泰嘉采用代理合作模式，说明推广权外包/引进的交易结构有现实基础
- 信立泰战略重心已转向创新管线（信立坦、S086等），泰嘉作为成熟仿制药，出让推广权符合其资源优化逻辑
- 分类调整暗示信立泰正在调整渠道策略，可能为更大范围的推广权合作铺路
