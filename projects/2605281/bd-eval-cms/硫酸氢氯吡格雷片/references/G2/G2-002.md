# [G2-002] 信立泰2024年报代理合作模式确认
- **URL**: https://pdf.dfcfw.com/pdf/H2_AN202503281648758021_1.pdf
- **抓取时间**: 2026-05-31 14:11 CST
- **来源类型**: 上市公司年报

## 关键数据点
- 信立泰2024年报明确：在部分区域采用代理合作等模式，借助品牌力量拓展外围市场覆盖
- 创新药采用学术推广模式，仿制药部分区域采用代理分销
- 2026年4月收入分类从"直接推广销售"与"代理推广销售"改为"直接销售"与"向经销商销售"
- 公司依托覆盖全国的循证医学推广团队
- 经销商订购药品，公司运送产品至经销商指定地点

## 对本评估的意义
- 代理分销模式已在泰嘉等仿制药上实际运营，推广权引进有现实基础
- 收入分类调整暗示渠道模式正在变化，可能为更大规模合作铺路
- "向经销商销售"模式意味着经销商承担主要终端推广职责
