# [P1-001] 国产心血管用药龙头，信立泰：走出集采影响，创新引领新增长
- **URL**: https://www.163.com/dy/article/HMHL6N8N05531YV4.html
- **抓取时间**: 2026-05-31 14:00 CST
- **来源类型**: 行业媒体

## 原文内容
华创证券关于信立泰的深度研究报告。信立泰成立于1998年，为中国传统心血管用药龙头企业。2000年公司抗血小板聚集药物硫酸氢氯吡格雷片（泰嘉）抢先在原研赛诺菲上市前获批，享有八年保护期。国家实施药品集采前泰嘉的销售峰值超过30亿元，为公司第一大核心品种。

## 关键数据点
- 信立泰2000年获批硫酸氢氯吡格雷片（泰嘉），为首仿药
- 集采前泰嘉销售峰值超过30亿元，占公司制剂收入近80%
- 2019年集采扩围意外丢标，2020年泰嘉收入下降约60%
- 信立坦2021年医院端+零售端合计约12亿元，同比增长60%
- 2012年公司从艾力斯引进高血压1.1类新药阿利沙坦酯（信立坦）
- 2021年公司营收30.6亿元，归母净利润5.3亿元
