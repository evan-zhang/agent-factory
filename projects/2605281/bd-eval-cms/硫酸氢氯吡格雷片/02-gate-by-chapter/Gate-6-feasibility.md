## 一、交易结构评估 [阶段A]

### 1.1 推荐交易结构

基于公开信息，推荐以下交易结构框架：

```
交易类型：推广权/代理权转让
合同期限：5年+续约选项
独家性：全国独家推广权（氯吡格雷硫酸氢片·信立泰品牌）
区域覆盖：中国大陆（含集采+非集采区域）
进货方式：向信立泰采购成品（非原料药）
分成/代理费：待谈判（建议：按销售额阶梯分成或固定采购价+少量里程碑）
MBQ（最低采购量）：建议不设或设低门槛（因集采品种销量由制度决定）
退出条款：集采丢标可退出、供应中断可退出、迁移比例<40%可减量
```

### 1.2 关键条款建议

| 条款 | 建议内容 | 谈判底线 |
|------|---------|---------|
| 代理费/进货价 | 固定采购价，毛利率≥50% | 毛利率≥45% |
| 独家性 | 全国独家推广权 | 至少主要省份独家 |
| 合同期限 | 5年+自动续约 | 不少于3年 |
| MBQ | 不设（集采品种） | 不超过当前年销量50% |
| 退出条款 | 集采丢标/供应中断可退出 | 至少集采丢标可退出 |
| 竞业禁止 | 信立泰不得在同一适应症推替格瑞洛竞品 | 至少合同期内 |
| 过渡支持 | 信立泰提供6个月团队过渡支持 | 至少3个月 |

### 1.3 独家性判断

**当前评估：有瑕疵**
- 氯吡格雷为多厂家竞争品种，深康仅获得"信立泰泰嘉"品牌的推广权
- 医院可能同时使用多个厂家的氯吡格雷（集采品种）
- "独家推广权"实质是独家代理信立泰产品，而非独占氯吡格雷市场
- **关键风险**：信立泰是否可能同时授权其他代理商？

## 四、即时现金流贡献测算 [阶段A]

### 4.1 首年净现金流测算

```
首年销售回款：82,500万元（基于75%保留率）
- 首年进货成本：39,600万元（毛利率52%）
- 团队与运营重建成本：7,500万元
- 一次性接管成本：1,500万元（过渡期双付等）
= 首年净现金流：33,900万元
```

**首年净现金流判断：正向，且金额可观**
- A-5品种的核心优势就是"接管后即变现"
- 无注册风险、无临床开发成本、无上市等待期
- 首年即可贡献正向净现金流

### 4.2 与A-1/A-2的区隔

- A-1（海外产品License-in）：需注册审批2-3年，首年无现金流
- A-2（国内合作成熟阶段）：需NDA审批1-2年，首年无现金流
- **A-5（已上市推广权接管）：接管后立即产生现金流**
- 这也是A-5评估"低注册风险"价值的具体体现

## 六、止损机制建议 [阶段A]

### 6.1 止损三件套

1. **集采丢标退出权**：若深康在集采续约中丢标，可无条件退出合同
2. **销量保底对赌**：若首年销售额低于基准情景60%（即年销<5亿元），可要求信立泰降价或退出
3. **供应中断保护**：若信立泰供应中断超过60天，可退出合同并索赔

### 6.2 里程碑对赌建议

```
里程碑1（第1年末）：销售额≥7亿元 → 继续执行
里程碑2（第2年末）：销售额≥9亿元 → 触发分成比例优化
里程碑3（第3年末）：销售额≥11亿元 → 触发合同续约选项
如未达里程碑1：信立泰降价10%或CMS有权退出
```
