## 二、集采/VBP风险评估 [阶段A]

| 评估维度 | 判断 | 风险等级 |
|---------|------|---------|
| 全国集采窗口期 | 已进入集采体系多年 | 已落地（低风险） |
| 地方联盟集采风险 | 部分省份可能有补充集采 | 低 |
| 降价幅度敏感性 | 当前价格已接近成本线，再降10-20%影响可控 | 中低 |
| 原研保护空间 | 赛诺菲集采中标后无特殊保护 | 无 |
| 院外转移能力 | 慢病续方可转移部分至院外 | 中 |

### 2.2 降价后利润敏感性

| 降价情景 | 出厂价（元/片） | 毛利率预估 | 财务影响 |
|---------|----------------|-----------|---------|
| 当前价格（不降） | ~2.5 | ~50-60% | 基准 |

**判断**：集采已执行多轮，价格再大幅下降的可能性极低。即使再降10-20%，财务仍可承受。

### 2.3 医保续约风险

- 氯吡格雷为成熟品种，不存在专利期/独占期到期导致的医保谈判降价
- 集采品种的医保支付标准与集采价格同步调整
- 短期（2-3年内）无重大医保政策变化风险

## 三、进院与准入策略 [阶段A]

### 3.1 进院流程

集采品种进院流程极大简化：
- 集采中标品种自动进入医院采购目录
- 不需要单独药事会审议
- 医院有集采用量考核指标，使用动力强

**判断**：氯吡格雷作为集采品种，进院准入几乎无障碍。

### 3.2 DRG/DIP影响

| DRG维度 | 影响 | 说明 |
|---------|------|------|
| PCI相关病组 | 正面 | 氯吡格雷为必用药品，替换风险低 |
| 低价仿制药优势 | 正面 | DRG/DIP下医院倾向使用低价品种 |
| 氯吡格雷vs替格瑞洛 | 正面 | 氯吡格雷价格更低，在DRG下更有优势 |

**判断**：DRG/DIP对氯吡格雷使用反而是正面因素——低价仿制药在成本控制环境下更受青睐。

### 3.3 准入资源可迁移性

| 准入资源 | 可迁移性 | 说明 |
|---------|---------|------|
| 医院进院关系 | 高 | 集采品种自动进院，不依赖关系 |
| 药事会资源 | 高 | 集采品种不需要药事会 |
| 医保支付关系 | 高 | 医保支付标准统一，不依赖个别关系 |
| 医生处方习惯 | 中高 | 氯吡格雷处方习惯根深蒂固 |

**判断**：集采品种的准入资源可迁移性极高，几乎不存在"关系资产"依赖。

## 四、存量保护策略 [阶段A]

### 4.1 氯吡格雷存量保护四件套

**策略1：国产替代路径（核心策略）**
- 赛诺菲波立维虽为集采中标，但在部分医院/省份仍存在使用惯性
- 深康可通过强调"国产一致性评价品种+成本优势"争取更多处方量
- 预计可转移5-10%的原研份额至泰嘉

**策略2：院外转移路径**
- PCI术后DAPT需持续12个月+，部分患者转入长期慢病管理
- 院外渠道（DTP药房、互联网医院、零售药店）续方可保持用量
- 预计院外渠道占比可从20-25%提升至30-35%

**策略3：自费维持路径**
- 氯吡格雷价格极低（日费用约2.5元），自费可负担性高
- 但自费市场对仿制药品种意义有限

**策略4：科室深耕路径**
- 聚焦PCI手术量大、氯吡格雷用量高的核心医院
- 强调氯吡格雷在高出血风险患者、老年患者中的安全性优势
- 深康心内科团队与氯吡格雷目标科室高度匹配

### 4.2 存量保护效果预估

```
若发生集采降价/医保续约冲击（概率：低）：
- 可保留的核心医院比例：≥90%（集采品种不受进院影响）
- 可转移至院外的销量比例：10-15%
- 可维持自费使用的患者比例：≥95%（日费用极低）
- 预计首年收入损失控制在：≤5%
```

## 五、推荐定价与准入策略 [阶段A]

### 5.1 推荐策略

**稳价深耕**——集采品种无定价自主权，策略重心为：
1. 确保集采续约中标（维护基本盘）
2. 在非集采渠道（院外/零售）争取略有溢价
3. 通过推广投入提升处方量，以量补价

### 5.2 准入优先级

| 优先级 | 行动 | 时间 |
|--------|------|------|
| P0 | 确认当前集采批次续约时间表，准备续约投标 | 立即 |
| P1 | 梳理深康在氯吡格雷主要使用省份的准入基础 | 1-3个月 |
| P2 | 建立院外/DTP渠道销售体系 | 3-6个月 |
| P3 | 与信立泰确认集采中标/非中标区域的推广权边界 | 交易前 |
