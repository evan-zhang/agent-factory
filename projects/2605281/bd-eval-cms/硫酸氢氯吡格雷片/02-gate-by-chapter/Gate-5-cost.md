## 一、接管团队规模测算 [阶段A]

### 1.1 团队需求分析

氯吡格雷作为集采品种，推广团队需求远低于创新药：
- **学术推广需求低**：成熟品种，指南地位稳固，无需大量学术投入
- **渠道维护为主**：集采品种配送由制度决定，主要维护医院关系和处方习惯
- **深康可共享资源**：深康已有心血管推广团队，氯吡格雷可与现有品种协同推广

### 1.2 团队规模测算

```
岗位类别                  现有（信立泰估计）  随行预估  新增招聘  CMS承接方案
销售代表                  50-80人           15-25人   20-30人  部分由深康现有团队兼任
大区/地区经理             5-10人            3-5人     2-3人    由深康管理层覆盖
MSL/医学经理             2-3人             1人       1人      共享深康医学部
市场准入/渠道管理         3-5人             2人       1-2人    共享深康准入团队
后勤运营/订单支持         2-3人             1人       0-1人    共享深康中台
合计                      62-101人          22-34人   24-36人  约50-70人最终配置
```

### 1.3 关键判断

- **以随行+新招混合模式为主**：信立泰团队可随行约30-40%，其余需新建或由深康现有团队覆盖
- **与深康现有团队的协同空间大**：氯吡格雷覆盖科室（心内科/神内科）与深康现有品种重叠
- **新建独立团队成本低**：集采品种推广人员薪酬低于创新药推广

## 三、毛利率分析 [阶段A]

### 3.1 毛利率测算

```
氯吡格雷集采中标价（出厂价）：约2.5元/片
原料药/辅料/制造成本（估计）：0.8-1.2元/片
包装/QC成本（估计）：0.1-0.2元/片
出厂成本合计：约0.9-1.4元/片
毛利润/片：约1.1-1.6元/片
毛利率：约44-64%
```

⚠️ **关键不确定性**：毛利率高度依赖于信立泰的进货价格/分成比例
- 若信立泰以成本+合理利润方式供货，毛利率可能在50-60%
- 若信立泰要求高额分成/代理费，毛利率可能降至40-50%
- **必须在与信立泰的谈判中明确进货/分成结构**

### 3.2 毛利率敏感性

| 进货成本（元/片） | 毛利率 | 达标判断 |
|------------------|--------|---------|
| 1.0（成本+少量利润） | 60% | ✅ |
| 1.25（中等分成） | 50% | ✅ 刚好达标 |
| 1.5（高分成） | 40% | ❌ 不达标 |
