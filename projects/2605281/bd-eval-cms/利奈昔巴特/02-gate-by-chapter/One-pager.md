# One-pager：利奈昔巴特（Linerixibat / Lynavoy）

## 终局五问

### 1. 中国峰值销售额预测
- 乐观情景：8~10亿元人民币（渗透率较高+溢价定价）
- 基准情景：4~6亿元人民币
- 保守情景：2~3亿元人民币
- 依据：中国 PBC 患者约10-15万人，瘙痒患者约9万，年治疗费用参考海外约2-3万美元/年，NRDL 谈判后预计降至5-8万元人民币/年 [OP-001]

### 2. 业务主体承接
- 推荐：**深康**（消化肝病为核心治疗领域）
- PBC 属于消化肝病范畴，与深康现有管线高度匹配
- 销售协同预期：高（>70%目标科室重叠）

### 3. 假设减半压力测试
- 若峰值减半至2-3亿元（保守情景），按合理首付款估算，IRR仍有望>15%
- PBC 瘙痒领域目前无任何竞品，市场先发优势显著
- 孤儿药定位，小适应症门槛低（基准≥1.5亿即达标）

### 4. 止损机制
- 注册风险：中国 NDA 已受理且纳入优先审评，获批概率较高
- 注册失败的止损：交易结构中里程碑绑定获批节点，未获批则不触发后续付款
- 竞品风险：PBC 瘙痒领域无直接竞品，IBAT 抑制剂竞品（maralixibat、odevixibat）均针对不同适应症（PFIC/Alagille）

### 5. 先发制人价值
- 评级：**高威胁**
- 替代获得者分析：若 CMS 不引进，最可能替代者包括：
  - ① 恒瑞/豪森等国内创新药企（正在布局自研 IBAT 抑制剂）
  - ② 竺帆/云顶等专注于肝病领域的 BD 公司
- 若替代者获得：直接竞争 CMS 现有消化肝病管线市场地位，且可能抢占 PBC 科室 KOL 资源
