## 一、核心差异化卖点

### Top 3 差异化
1. **首个 PBC 瘙痒靶向治疗**：全球首个且唯一获批专门用于 PBC 胆汁淤积性瘙痒的药物，填补了重要的临床空白
2. **快速起效**：第2周即显著改善瘙痒症状，且疗效持续维持
3. **独特机制**：IBAT 抑制剂通过减少胆汁酸重吸收，从源头降低瘙痒介质，区别于传统抗组胺/胆汁酸螯合剂

## 三、临床证据详细评估

### 3.1 GLISTEN III 期关键数据 [P1-001][P1-003]
- **设计**：双盲、随机、安慰剂对照，1:1分组
- **样本量**：n=238（每组 n=119）
- **主要终点**：24周内每月 WI-NRS 瘙痒评分变化
  - LS均值差：-0.72（利奈昔巴特 vs 安慰剂）
  - 统计学显著
- **关键次要终点**：
  - 第2周瘙痒评分改善（快速起效）
  - 24周瘙痒相关睡眠干扰评分改善
  - 第24周临床意义改善率：56% vs 43%（差异13pp）
- **安全性**：
  - 腹泻：61%（最常见），但大多轻中度
  - 腹痛：18%
  - 因腹泻停药：4% vs <1%（安慰剂）
  - 无严重安全性信号

## 四、临床试验注册法规解读

### 桥接可行性法规核查清单

| 核查维度 | 判断 | 说明 |
|---------|------|------|
| 种族敏感性 | 低风险 | IBAT 靶点种族保守性高，肠道局部作用 |
| 全球多中心入组 | 需确认 | GLISTEN 亚洲受试者比例待确认 |
| 主要终点认可度 | 可接受 | WI-NRS 为 PRO 量表，CDE 可能要求补充硬终点 |
| 样本量充分性 | 充分 | n=238，主要终点统计显著 |
| 安全性数据库 | 待评估 | 暴露人数需确认是否满足 ICH E1 慢性病要求（≥1500人·年） |
| 对照组适用性 | 合理 | 安慰剂对照符合伦理（瘙痒非致命） |

### 注册策略结论
```
桥接试验必要性：免桥接（大概率，NDA已受理暗示CDE接受）
CDE沟通建议：NDA已进入审评，如需补充应通过发补程序
注册分类确认：5.1类
关键注册法规风险：
  1. CDE 可能要求补充中国患者数据
  2. 安全性数据库长期暴露数据可能不足
  3. PRO 终点在中国注册环境下的认可度
```

## 五、KOL 锁定风险

- **评级**：低
- **理由**：PBC 瘙痒领域此前无专门治疗药物，KOL 对 IBAT 抑制剂用于 PBC 瘙痒的认知尚在建立初期
- **突破策略**：首个获批产品本身即为最佳 KOL 教育工具
