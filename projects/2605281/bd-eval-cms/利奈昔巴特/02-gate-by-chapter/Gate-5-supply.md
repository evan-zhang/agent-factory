# Gate 5：供应链与 CMC（成本门）

## 一、生产基地与 GMP 状态

### 1.1 原研生产
- 利奈昔巴特由 GSK 开发，全球权益已于 2026年3月9日转让给 Alfasigma
- Alfasigma 总部：意大利博洛尼亚
- Alfasigma 拥有自己的生产基地（意大利境内）
- 具体生产利奈昔巴特的工厂：需确认

### 1.2 GMP 状态
- GSK 生产基地通常通过 FDA/EMA GMP 认证
- NMPA 境外检查记录：需确认
- 风险评估：GSK/Alfasigma 为大型药企，GMP 合规风险较低

| 风险维度 | 评级 | 说明 |

## 三、COGS 成本结构预估

| 成本组成 | 占出厂价比例（%） | 说明 |
|---------|-----------------|------|
| 原研转让价（CIF到岸价） | 40-50% | 创新药典型区间 |

| 进口质检/报关费 | 3-5% | |
| 仓储/物流 | 2-4% | 口服制剂，无需冷链 |
| 质量控制/放行检验 | 2-3% | |
| 销售前损耗 | 1-2% | |
| **合计 COGS 占净售价比例** | **56-76%** | |
| **毛利率预估** | **24-44%** | |

> ⚠️ **毛利率不达标风险**：按初步估算，如果转让价占 40-50%，毛利率可能低于 60% 门槛。需通过以下方式优化：
> 1. 谈判降低原研转让价
> 2. 规模效应降低单位成本
> 3. 争取更高终端定价

## 四、国产化/本地化生产可能性

- 利奈昔巴特为小分子口服固体制剂
- 技术转让可行性：中等（合成路线需评估复杂度）
- 本地化时间节点：获批后3-5年可考虑
- 本地化后毛利率有望提升至 65-70%

## 五、Gate 5 结论

```
供应链风险等级：中
预计毛利率区间：24-44%（⚠️ 可能不满足60%门槛）
是否满足创新药毛利率≥60%门槛：否（需通过谈判优化转让价或提高定价）
COGS核心风险Top3：
  1. 原研转让价占比过高，可能挤压毛利率
  2. 进口环节税费叠加
  3. 单一生产基地风险
置信度：低（缺乏具体转让价数据）
需补证据Top5：
  1. Alfasigma 利奈昔巴特生产工厂 NMPA 境外检查记录
  2. 原研方报价单（PI单/Quotation）
  3. 产能上限及中国配额政策
  4. 本地化生产技术转让可行性评估

```

> ⚠️ **Gate 5 风险提示**：毛利率是本品种最大的财务风险点。建议在谈判中重点争取较低的转让价，或探索本地化生产方案以提高毛利率。
