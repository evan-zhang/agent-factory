### 2.3 MB-001竞争优势分析

**核心优势**：
1. **唯一口服生物制剂** — 区别于所有已上市TNF-α单抗（均为注射）
2. **局部靶向** — 理论安全性优于全身性TNF-α阻断
3. **生物制剂vs小分子** — 与JAK抑制剂相比，长期安全性可能更优

**核心劣势**：
1. **极早期** — Phase 1/2，距离上市5-8年，市场格局将巨变
2. **口服JAK已上市** — 乌帕替尼已在中国上市且为口服，MB-001并非唯一口服选择
3. **价格劣势** — 生物制剂生产成本远高于小分子JAK抑制剂

### 2.4 治疗路径动态推演

**当前治疗序列**（中国中重度UC）：
```
一线：5-ASA → 
二线：糖皮质激素 → 
三线：抗TNF-α单抗(英夫利昔/阿达木) 或 维得利珠单抗 或 乌司奴单抗
四线：JAK抑制剂(乌帕替尼) 或 手术
```

**3年后推演**（假设MB-001尚未上市）：
```
一线：5-ASA →
二线：糖皮质激素 →
三线：JAK抑制剂(口服) 或 生物制剂(注射) — JAK抑制剂可能升线至三线
四线：生物制剂联合治疗 或 手术
```

**5年后推演**（假设MB-001获批）：
```
一线：5-ASA →
二线：糖皮质激素 →
三线：口服JAK抑制剂 或 口服MB-001（生物制剂口服选项）— MB-001面临与JAK口服制剂直接竞争
四线：注射生物制剂 或 手术
```

**路径锁定风险**：如果JAK抑制剂在未来3-5年内在中国UC治疗指南中巩固三线地位，MB-001上市后可能面临从"唯一口服生物制剂"变成"口服选项之一"的竞争格局。
