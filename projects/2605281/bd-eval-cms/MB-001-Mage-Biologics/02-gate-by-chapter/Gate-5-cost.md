## 1. 生产与供应链分析

### 1.1 原研方GMP生产基地

**Tillotts Pharma AG（瑞士）**：
- 母公司Zeria Group（日本上市公司）
- 拥有欧洲GMP认证
- 产品包括Asacol™(5-ASA)、Entocort™(布地奈德)、DIFICLIR™(非达霉素)等已上市产品 [P1-001]
- 约400名员工，产品销往~65国 [P1-002]
- ⚠️ MB-001的GMP生产是否在Tillotts设施内完成？需确认

**关键CMC问题**：
1. MB-001是口服结肠靶向制剂，需要特殊的延迟释放和持续释放配方技术 [P1-001]
2. 活性成分是人源化单克隆抗体 → 生物制品GMP要求
3. 口服抗体制剂的稳定性、释放度控制是技术难点
4. Tillotts在口服肠溶制剂方面有经验（Asacol使用OPTICORE™技术），但口服抗体是全新挑战

### 1.2 供应链稳定性评估

**供应链风险等级：中-高**

| 风险维度 | 评估 | 说明 |
|---------|------|------|
| 单一生产基地风险 | 高 | 口服抗体制剂生产工艺复杂，可能只有1-2个GMP设施 |
| 产能瓶颈 | 中 | 抗体生产产能有限，口服制剂灌装线可能需要专用设备 |
| 原料药来源 | 中 | 人源化单抗的细胞培养和纯化依赖成熟供应链 |
| 冷链要求 | 低-中 | 口服固体制剂通常不需要严格冷链（vs注射型生物制剂） |
| 技术转移难度 | 高 | 口服结肠靶向抗体制剂技术转移极其复杂 |

### 1.3 国产化/本地化可能性

- 中长期可能：中等
- Tillotts的OPTICORE™结肠靶向技术可能需要技术转移
- 抗体生产部分可考虑在中国委托CDMO
- 但口服抗体制剂的完整技术转移预计需要3-5年
- ⚠️ 本地化生产的可行性高度依赖于Tillotts的合作意愿

## 2. 成本结构估算

### 2.1 COGS结构拆解（基于阶段A公开信息估算）

| 成本组成 | 占出厂价比例（%） | 说明 |
|---------|------------------|------|
| 原研转让价（CIF到岸价） | 35-45% | 单抗原料+口服制剂生产 [D级-基于假设] |
| 关税+增值税 | 8-12% | 生物制品进口关税+13%增值税 |
| 进口质检/报关费 | 2-3% | 生物制品需批批检 |
| 冷链/仓储/物流 | 2-4% | 口服制剂冷链要求低于注射剂 |
| 质量控制/放行检验 | 3-5% | 抗体+特殊释放度测试 |
| 销售前损耗 | 1-2% | 稳定性导致的损耗 |
| **合计COGS占净售价比例** | **51-71%** | — |
| **毛利率预估** | **29-49%** | — |

**⚠️ 关键风险**：毛利率预估29-49%，**可能低于创新药毛利率≥60%门槛**

**原因分析**：
1. 口服抗体制剂生产成本远高于传统口服固体制剂
2. 结肠靶向技术的特殊配方增加了CMC复杂度
3. 进口环节成本（关税+增值税+质检）叠加
4. ⚠️ 如果能实现本地化生产，毛利率可能改善至55-65%

### 2.2 毛利率改善路径

| 改善措施 | 预计毛利率改善 | 可行性 |
|---------|---------------|--------|
| 本地化生产（委托CDMO） | +10-15pp | 中期可行（3-5年） |
| 规模效应（产量翻倍） | +3-5pp | 上市后逐步实现 |
| 转让价谈判 | +5-10pp | 取决于交易条款 |
| **综合改善后毛利率** | **55-65%** | 仍可能低于60%门槛 |
