## 1. 产品定调

**产品属性**：全球首创（First-in-class）口服抗TNF-α单克隆抗体

**定调依据**：
- MB-001是全球唯一进入临床阶段的口服抗TNF-α单抗 [P1-001]

### 2.3 科学价值判断

**强差异化潜力（但尚未验证）**：

MB-001的差异化建立在三个维度：

1. **给药途径创新**：口服 vs 注射/静脉 — 如果成功，将根本改变UC生物制剂的治疗范式
   - 现有TNF-α单抗：英夫利昔单抗(IV)、阿达木单抗(SC)、戈利木单抗(SC)均需注射
   - 口服给药可显著提高患者依从性，减少医疗资源占用（无需输液中心/护士注射）

2. **局部靶向 vs 全身暴露**：结肠黏膜靶向释放 → 局部浓度高，全身暴露低
   - 理论上可减少TNF-α全身阻断带来的感染风险（现有TNF-α单抗的主要安全性顾虑）
   - 改良糖基化模式优化受体相互作用 [P1-001]

3. **平台扩展潜力**：口服抗体递送技术可应用于其他抗体药物
   - UCL School of Pharmacy的结肠靶向技术是平台级技术 [P1-001]
   - 可能扩展至其他IBD适应症（克罗恩病）或其他肠道疾病

**科学价值结论**：强差异化概念，但尚未获得临床验证。科学价值高度依赖于I/II期PoC数据。

## 3. 临床试验注册法规解读（Gate 2必填子模块）

### 3.1 桥接可行性法规核查清单

| 核查维度 | 核查内容 | 判断 | 说明 |
|---------|---------|------|------|
| 种族敏感性 | 口服结肠靶向制剂是否种族敏感 | 可能敏感 | 结肠转运时间、肠道菌群、黏膜通透性存在种族差异 |
| 全球多中心入组 | 关键III期是否包含中国/亚洲受试者 | 否 | 全球III期尚未启动，中国无参与 |
| 主要终点认可度 | UC临床终点是否CDE认可 | 待确认 | Mayo评分/改良Mayo评分是UC国际标准终点 |
| 样本量充分性 | 关键试验样本量 | 不适用 | III期尚未设计 |
| 安全性数据库 | 暴露人数 | 极少 | 仅48名健康志愿者+少数UC患者 |
| 对照组适用性 | 对照组SOC是否符合中国临床实践 | 待确认 | 中国UC SOC与海外可能有差异 |
| 特殊人群数据 | 肝肾功能不全/老年/儿童数据 | 无 | 无任何特殊人群数据 |

### 3.2 注册策略结论

```
桥接试验必要性：需独立中国临床（至少II/III期桥接，可能需完整III期）
预计桥接规模：无法估算（全球III期设计尚未确定）
CDE沟通建议：强烈建议Pre-IND会议，明确口服生物制剂注册要求
注册分类确认：待确认（可能走1类创新药，需与CDE沟通确认）
关键注册法规风险：
  1. NMPA无口服单抗审评先例，审评标准不确定
  2. 结肠靶向制剂可能需额外生物等效性/释放度研究
  3. 全球临床数据有限，中国可能要求独立III期
```

## 4. KOL锁定风险评估

**KOL锁定风险评级：低-中**

- UC/IBD领域中国KOL资源相对开放，未观察到单一竞品深度绑定核心KOL群体的证据
- 但现有TNF-α制剂（英夫利昔单抗/阿达木单抗）在中国已有较长处方历史，处方惯性存在
- MB-001作为口服制剂，差异化定位足够明显，有望突破KOL认知壁垒
- 建议：提前锁定2-3位国内IBD领域核心KOL作为学术顾问
