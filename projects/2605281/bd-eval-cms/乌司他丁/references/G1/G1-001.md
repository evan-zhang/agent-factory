# [G1-001] 乌司他丁指南推荐与临床地位

- **来源类型**: 指南/共识文献检索
- **检索日期**: 2026-05-31

## 关键发现

### 国内指南/共识
1. **《中国急性胰腺炎诊治指南(2021)》**（中华医学会外科学分会胰腺外科学组）——推荐蛋白酶抑制剂用于急性胰腺炎治疗，乌司他丁为主要推荐药物 [G1-001a]
2. **《2013中国急诊急性胰腺炎临床实践指南》**——推荐乌司他丁作为蛋白酶抑制剂用于急性胰腺炎治疗
3. **《乌司他丁用于临床常见急危重症的专家共识》**——总结乌司他丁在急性胰腺炎、休克、脓毒症等急危重症中的应用，强调其抑制蛋白水解酶和调控炎症反应的作用
4. **《急诊科重症急性胰腺炎预防与阻断专家共识(2022)》**——乌司他丁可有效抑制胰蛋白酶和糜蛋白酶活性，也可抑制IL-6和TNF-α等炎症因子的生成
5. **《中国急性胰腺炎诊治指南2025修订》**——2024年12月启动修订，预计2025-2026年发布

### 国际指南
1. **2025 IAP修订版指南**（国际胰腺病学会）——2025年7月发布，基于近10年高质量RCT证据全面修订，涵盖疼痛管理、液体治疗、营养支持、感染性坏死处理、并发症管理等。提及蛋白酶抑制剂在急性胰腺炎中的应用
2. **2024 ACG指南**（美国胃肠病学院）——2024年3月发布，提出11条推荐意见与23个关键概念。该指南对急性胰腺炎药物治疗的推荐偏向于支持治疗为主，对蛋白酶抑制剂的推荐等级有限
3. **2025 APS循证医学临床实践指南**——重症急性胰腺炎的诊断和治疗管理

### 分析要点
- 国内指南对乌司他丁推荐等级较高，为中国急性胰腺炎基础用药
- 国际指南（IAP/ACG）对蛋白酶抑制剂的推荐等级相对有限，更强调支持治疗
- 存在国内/国际指南推荐差异，可能与临床实践差异和证据解读有关
- 乌司他丁应用范围已从急性胰腺炎扩展至脓毒症、ARDS、休克等急危重症领域

## 置信度
- 国内指南地位：高（多部指南/共识推荐）
- 国际指南地位：中低（推荐等级有限）
- 临床处方惯性：高（20+年临床使用历史）
