# [P1-001] 2025年中国乌司他丁行业现状及展望
- **URL**: https://www.163.com/dy/article/K7AJCHQB05387IEF.html
- **抓取时间**: 2026-05-31 14:00 CST
- **来源类型**: 行业媒体

## 原文内容
华经情报网2025年8月发布。乌司他丁为丝氨酸蛋白酶抑制剂，广泛用于急性胰腺炎。近年应用扩展至休克、脓毒症、重症肺炎、ARDS等。2023年中国乌司他丁市场规模约11.6亿元。上游为乌司他丁粗品生产（艾迪药业为主要供应商），中游为纯化与制剂（天普生化为核心），下游为医院与药房。上海医药集团旗下天普生化为主要生产企业，天普洛安®为国内独家品种。上海医药2024年上半年营收1394亿元。

## 关键数据点
- 2023年中国乌司他丁市场规模约11.6亿元
- 天普洛安（注射用乌司他丁）为国内独家品种（国家二类新药）
- 乌司他丁应用从急性胰腺炎扩展至脓毒症、ARDS等急危重症
- 重组乌司他丁仍处于开发阶段
