# G2-002：宝济药业重组乌司他丁（BJ044）竞争威胁

**来源**: 宝济药业公告 / 智通财经 / 腾讯新闻
**获取日期**: 2026-05-31
**置信度**: C级（公开信息）

## 核心信息

### 1. BJ044进展
- 2026年4月：宝济药业自主研发的**重组乌司他丁（BJ044）**IND申请获NMPA受理
- BJ044是**全球首个进入临床申报阶段的重组乌司他丁产品**
- 标志着乌司他丁从"尿液提取"向"生物智造"的技术路径转变

### 2. 竞争逻辑
- 传统天普洛安依赖人尿液提取，存在供应稳定性、病毒污染风险、批间质量差异等局限
- 重组技术可解决上述问题，实现工程化表达与规模化生产
- BJ044完整保留了天然乌司他丁的多靶点抗炎与器官保护功能

### 3. 时间线预估
- 根据宝济药业招股书（2025年7月）：BJ044预计2025年下半年获NDA批准进入I期试验；2026年上半年IND受理已落地
- 从IND到上市，乐观估计需要3-5年（含临床I-III期及上市审批）
- **预计上市时间窗口：2029-2031年**

### 4. 市场影响预判
- 宝济药业上市后，天普洛安的"独家品种"地位将被打破
- 重组产品在纯度、安全性、供应稳定性上具备优势
- 天普洛安可能需要在BJ044上市前完成市场护城河加固

### 5. 市场规模引用
- 2023年中国乌司他丁市场规模约**11.6亿元**（宝济药业引用数据）
