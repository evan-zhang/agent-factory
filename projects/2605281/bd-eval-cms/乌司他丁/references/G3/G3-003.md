# [G3-003] 中国急性胰腺炎发病率与患者基数

- **URL**: https://www.chinagp.net/CN/10.12114/j.issn.1007-9572.2025.0025; 百度健康
- **抓取时间**: 2026-05-31 15:24 CST
- **来源类型**: 学术文献/国家卫健委统计

## 原文内容

**国家卫健委2024年统计数据**：
- 我国急性胰腺炎（AP）年发病率已达34.2/10万
- 慢性胰腺炎（CP）患病率为13.7/10万
- 重症急性胰腺炎（SAP）病死率仍高达15%-20%

**1990-2021年中国和全球胰腺炎疾病负担趋势分析（GBD研究）**：
- 2022-2031年预测：中国胰腺炎标化患病率（ASPR）将由2022年24.08/10万减少到2031年的21.21/10万
- 标化死亡率（ASMR）从0.64/10万减少至0.53/10万
- 标化伤残调整生命年（ASDR）从17.98/10万减少至14.63/10万

## 关键数据点
- 中国急性胰腺炎年发病率34.2/10万（2024年）
- 按中国14亿人口估算，年新发病例约47.9万例
- 上报人称"1000万患者"为累计患病数或夸大，实际年发病约48万例
- 重症急性胰腺炎病死率15%-20%
- 胰腺炎标化患病率呈缓慢下降趋势
