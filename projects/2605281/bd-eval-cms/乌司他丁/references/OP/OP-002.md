# OP-002: 上海医药收购天普生化股权及合作背景

**来源**: 智通财经《上海医药斥9亿控股天普》/ 东方财富PDF公告
**抓取日期**: 2026-05-31

## 关键信息

1. **股权沿革**:
   - 2010年: 奈科明（Nycomed）以2.1亿美元收购天普51.34%股份
   - 后奈科明被武田（Takeda）收购，天普成为武田间接控股子公司
   - 2018年: 上海医药以1.44亿美元（约9.15亿元）收购武田所持天普股权，持股比例升至约67.14%，实现绝对控股
2. **天普定位**: 成立于1993年，全球领先的人尿蛋白质生物制药企业，专注于危急重症领域
3. **上海医药背景**: 沪港两地上市（A股601607/H股02607），2024年上半年营收1394亿元

## 评估参考价值

- 上海医药通过绝对控股天普（67.14%），牢牢掌握乌司他丁核心资产
- 天普为上海医药核心子公司，非边缘资产，出让推广权动机极低
- 武田退出后上海医药主动加码控股，表明其对乌司他丁的战略重视
