# OP-003: 乌司他丁临床指南地位与医保状况

**来源**: 中国全科医学《乌司他丁用于临床常见急危重症的专家共识》/ 中国急性胰腺炎诊治指南(2021)
**抓取日期**: 2026-05-31

## 关键信息

1. **指南推荐**: 
   - 中国急性胰腺炎诊治指南(2021)将蛋白酶抑制剂纳入推荐
   - 乌司他丁专家共识(2023)明确推荐用于急性胰腺炎，尤其重症急性胰腺炎
   - 2025年IAP修订版指南提及蛋白酶抑制剂
2. **适应症扩展**: 专家共识覆盖急危重症多领域应用
3. **医保状态**: 已纳入国家医保目录
4. **急性胰腺炎发病率**: 发病率约2.48人/1,000人/年，重症急性胰腺炎病死率36%-50%

## 评估参考价值

- 指南推荐等级高，临床地位稳固
- 急危重症领域共识扩展，超出胰腺炎的增量空间
- 医保身份保障终端放量基础
