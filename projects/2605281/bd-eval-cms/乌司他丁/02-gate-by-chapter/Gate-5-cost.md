# Gate 5：团队与运营成本（成本门）

**品种**: 乌司他丁（天普洛安®）
**案件代号**: 2605-3101
**评估技能**: A-5（已上市产品推广权引进评估）
**业务主体**: 深康
**评估阶段**: 阶段A（公开信息）
**评估日期**: 2026-05-31

本 Gate 引用：[G2-001] [G5-001] [G6-001]

## 结论句

**接管乌司他丁需要从零搭建推广团队，因为天普团队为上海医药体系员工不可能随行迁移。预计需新建50-80人专业化团队覆盖核心三级医院，首年运营成本约6,000-8,000万元，加上一次性渠道重建和市场投入，首年总成本可能高达1.0-1.5亿元。在首年销售保留率仅40-55%（对应销售额5-7亿元）的前提下，毛利率虽高但首年SG&A（销售、一般及行政费用）占比可能超过30%，接近A-5门槛上限的35%。**

---

## 5.1 接管团队规模测算

### 5.1.1 现有团队（天普/上海医药）

- 天普拥有"垂直密集型的全国营销网络"和"专注危重症领域的专业化营销团队" [G2-001]
- 2018年数据：天普洛安营销队伍人均销售额262万元 [G2-001]
- 按2025年销售额13.1亿元估算，团队规模约**500人**（13.1亿÷262万）
- **关键问题**：天普团队为上海医药正式员工，不可能随行迁移至CMS

| 岗位类别 | 现有人数（估算） | 随行人数 | 新增招聘 | 备注 |
|---------|----------------|---------|---------|------|
| 销售代表 | ~400 | 0 | 40-60 | 覆盖核心三级医院消化科/ICU/急诊 |
| 大区/地区经理 | ~60 | 0 | 8-10 | 区域管理层 |
| MSL（医学科学联络官）/医学经理 | ~15 | 0 | 3-5 | 学术推广必需 |
| 市场准入/渠道管理 | ~15 | 0 | 3-5 | 与深康共用部分 |
| 后勤运营/订单支持 | ~10 | 0 | 2-3 | 共享中台 |
| **合计** | **~500** | **0** | **56-83** | |

### 5.1.2 团队承接模式

**以新建团队为主**——这是本案最大的运营成本挑战。

- 深康消化肝病领域已有团队基础，可部分复用
- 但ICU/急诊科覆盖为新增能力需求，需专项招聘
- 医学团队（MSL）需具备急重症领域学术背景
- 预计团队组建周期6-9个月

## 5.3 毛利率与成本结构分析

### 5.3.1 毛利率估算 [D级-基于假设]

| 参数 | 估算值 | 依据 |
|------|--------|------|
| 中标价 | 83.13元/瓶 | 公开中标数据 [G2-001] |
| 出厂价（CMS进货成本） | 50-60元/瓶 | 推断——上海医药给代理商的供货价 [D级-基于假设] |
| CMS销售价 | 83.13元/瓶 | 按中标价 |
| 毛利/瓶 | 23-33元 | |
| **毛利率（毛利润/营业收入）** | **28%-40%** | 低于A-5门槛的≥50% |

**关键发现**：作为推广代理商（非MAH），CMS的毛利率远低于A-5门槛的50%。这是A-5评估框架下的重要负面信号。

### 5.3.2 与A-5门槛对比

| 指标 | A-5门槛 | 本案估算 | 达标？ |
|------|---------|---------|--------|
| 毛利率（毛利润/营业收入） | ≥50% | 28%-40% | ❌ 未达标 |
| SG&A占收入比例 | ≤35% | 12-19%（首年） | ✅ 达标 |
| 基准峰值销售额 | ≥2亿元 | 10-12亿元 | ✅ 达标 |

**毛利率不达标的原因**：CMS作为推广代理商的毛利率天然低于MAH自营。天普洛安出厂价中已包含天普的生产利润和品牌溢价，留给代理商的毛利空间有限。

## 5.4 过渡期损耗评估

### 5.4.1 渠道空白期

- 合同生效到实际发货切换之间可能存在3-6个月空白期 [C级-待验证]
- 天普/上海医药经销商体系切换至CMS体系需要时间
- 关键医院供应不中断为第一优先级

### 5.4.2 医院覆盖中断风险

- 已进院医院理论上无需重新准入，但需变更推广方备案
- 部分医院可能因推广方变更暂停采购，等待内部审批
- **预计影响**：Top 30医院中5-10家可能出现2-4周供应暂停 [D级-基于假设]

### 5.4.3 学术影响力断档

- 天普20+年学术推广积累的KOL关系将断裂
- 科室会、学术会议参与度下降
- **这是接管后最严重的无形损耗**——学术影响力重建需要2-3年

### 5.4.4 运营体系重建

- CMS需建立独立的乌司他丁推广体系
- 冷链物流需确保冻干粉针剂储存运输条件
- 票流/财务结算需与上海医药商业体系对接
- 预计重建周期6-12个月

## 5.5 运营效率诊断

### 5.5.1 组织承接能力

- 深康在消化肝病领域有团队基础，可部分复用
- 但急重症（ICU/急诊）推广经验不足
- 需评估深康现有团队是否具备跨科室推广能力

### 5.5.2 边际费用摊薄

- 若与深康现有消化科品种（如肝病产品线）同团队推广，边际费用可摊薄20-30%
- 但ICU/急诊为全新科室，摊薄空间有限

### 5.5.3 共享中台

- 法务、财务、物流、合规可通过深康/CMS集团中台共享
- 预计可节省10-15%运营成本
