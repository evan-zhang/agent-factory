# [P1-004] Ogluo (dasiglucagon) EMA/MHRA 批准信息
- **URL**: https://www.ema.europa.eu/en/medicines/human/EPAR/ogluo
- **抓取时间**: 2026-05-31 14:03 CST
- **来源类型**: 官方

## 原文内容

Ogluo是Gvoke的海外版本（商品名不同，同一活性成分dasiglucagon/胰高血糖素）：
- EMA：2024年7月通过集中审评程序获批（全欧盟）
- MHRA：2021年4月29日英国获批
- 适应症：≥2岁糖尿病患者严重低血糖的治疗
- 在英国通过Tetris Pharma商业化
- 2021年在德国、丹麦、挪威上市

Xeris与Zealand Pharma关系：
- Zealand Pharma持有dasiglucagon原研权
- Xeris从Zealand获得美国商业化权（商品名Gvoke）
- Zealand保留欧洲/国际权（商品名Ogluo，后授权给Tetris Pharma）

Zealand Pharma情况：
- Zegalogue（dasiglucagon）2021年3月22日获FDA批准
- Zealand也在开发dasiglucagon的其他适应症（先天性高胰岛素血症）

## 关键数据点
- Ogluo EMA获批：2024年7月（全欧盟）
- Ogluo MHRA获批：2021年4月
- Gvoke FDA获批：2019年9月
- Zegalogue FDA获批：2021年3月
