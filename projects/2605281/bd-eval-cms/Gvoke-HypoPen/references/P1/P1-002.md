# [P1-002] Xeris Biopharma 2025年全年及Q4财务报告
- **URL**: https://xerispharma.com/news-releases/news-release-details/xeris-biopharma-reports-record-performance-fourth-quarter-and
- **抓取时间**: 2026-05-31 14:01 CST
- **来源类型**: 公司公告

## 原文内容

Xeris Biopharma Holdings (Nasdaq: XERS) 2025年全年业绩：
- 总收入：2.918亿美元（同比增长43.7%）
- Gvoke全年净收入：9,410万美元（同比增长13.6%）
- Gvoke Q4净收入：2,465万美元（同比增长5.9%）
- Gvoke 2024年底零售处方市场份额约35%
- Recorlev全年净收入：1.393亿美元（同比增长116.7%）
- Keveyis全年净收入：4,765万美元（同比下降3.8%）
- 2026年指引：总收入3.75-3.90亿美元（中位数增长>30%）
- 调整后EBITDA：5,940万美元
- 净利润：60万美元（2024年净亏损5,480万美元）
- 首次实现全年盈利
