# [P1-003] 全球胰高血糖素市场规模及趋势
- **URL**: https://www.fortunebusinessinsights.com/glucagon-market-108948
- **抓取时间**: 2026-05-31 14:02 CST
- **来源类型**: 行业媒体

## 原文内容

全球胰高血糖素市场：
- 2025年市场规模：3.397亿美元
- 2026年市场规模：3.628亿美元
- 2034年预测：6.141亿美元
- CAGR（2026-2034）：6.80%
- 北美占主导地位，2025年市场份额85.78%

中国关键数据：
- 中国预计CAGR：10.40%（全球最高增速之一）
- 中国糖尿病人口基数大且持续增长
- 对可负担的仿制胰高血糖素制剂需求增加

竞争格局：
- 注射剂和吸入剂两大类
- 吸入剂（Baqsimi）2026年占主导地位
- 注射剂增长动力来自新型液体稳定制剂（Gvoke HypoPen等）
- 关键产品：Baqsimi（鼻喷）、Gvoke（自动注射笔）、Zegalogue（预充注射笔）、GlucaGen（传统急救盒）

市场趋势：
- 从传统冻干粉急救盒转向即用型制剂
- 2023年UnitedHealthcare取消胰高血糖素自付费用
- 法国Baqsimi报销比例65%
- 欧洲：Ogluo已在德国、丹麦、挪威上市

## 关键数据点
- 全球市场2025：3.397亿美元
- 中国CAGR：10.40%
- 北美份额：85.78%
- 2034年预测：6.141亿美元
