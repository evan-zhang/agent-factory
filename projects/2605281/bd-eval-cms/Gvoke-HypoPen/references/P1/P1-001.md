# [P1-001] Gvoke HypoPen: An Auto-Injector Containing an Innovative, Liquid-Stable Glucagon Formulation for Use in Severe Acute Hypoglycemia
- **URL**: https://pmc.ncbi.nlm.nih.gov/articles/PMC6794225/
- **抓取时间**: 2026-05-31 14:00 CST
- **来源类型**: 学术

## 原文内容

Gvoke HypoPen是Xeris Pharmaceuticals开发的即用型液体稳定胰高血糖素自动注射笔，用于治疗糖尿病患者的严重急性低血糖。

**关键信息**：
- FDA于2019年9月批准，是全球第一款获批的即用型液体稳定胰高血糖素自动注射器
- 皮下注射，两步操作：拔帽→按压皮肤5秒
- 有两种预充剂量：儿童0.5mg/0.1mL，成人/青少年1mg/0.2mL
- 使用非水溶剂（triacetin）防止胰高血糖素聚集和纤维化
- 室温储存2年，无需冷藏
- 99%成功率完成全剂量给药
- 无外露针头，注射后针头自动回缩
- 成人Phase 3临床试验：100%患者成功恢复血糖
- 儿童Phase 3：100%成功率
- 不良反应：恶心和呕吐（与现有GEK相似）
- 适应症：严重急性低血糖，年龄≥2岁的糖尿病患者
- 同时申请孤儿药资格：先天性高胰岛素性低血糖症、减肥术后低血糖症

## 关键数据点
- FDA批准日期：2019年9月
- 成人有效率：100%（Phase 3）
- 儿童有效率：100%
- 自动注射成功率：99%（模拟紧急情况）
- 储存条件：室温2年
- 剂型：0.5mg/0.1mL（儿童），1mg/0.2mL（成人）
