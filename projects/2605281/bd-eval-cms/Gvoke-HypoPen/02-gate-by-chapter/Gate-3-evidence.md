### 1.2 胰高血糖素急救产品的目标可及患者

即使存在严重低血糖事件，实际使用胰高血糖素急救产品的比例极低：
- 美国市场：约30-35%的胰岛素使用患者拥有胰高血糖素处方 [P1-002, 需确认]
- 中国市场推断：拥有胰高血糖素急救产品的患者比例<5%（传统急救盒仅在住院场景使用）

**核心挑战**：中国市场需要从"几乎没有院外胰高血糖素急救文化"→"逐步建立品类认知和使用习惯"。

## 2. 市场规模与峰值销售额模型 [阶段A/B混合]

### 2.1 中国胰高血糖素市场背景

全球胰高血糖素市场2025年约3.40亿美元 [P1-003]，中国CAGR（复合年均增长率）10.40%为全球最高 [P1-003]。中国市场目前处于品类教育初期阶段——Baqsimi在2024年6月获批后，正在做早期市场培育。

### 2.2 峰值销售额三情景模型

**产品本质确认**：胰高血糖素急救产品是**事件驱动型单次使用产品**——不是慢性病维持治疗，而是在严重低血糖事件发生时使用一次。市场规模应基于"事件次数×产品单价"。

| 指标 | 保守情景 | 基准情景 | 乐观情景 |
|------|---------|---------|---------|
| 严重低血糖年事件总数（万次） | 120 | 150 | 190 |
| 使用胰高血糖素急救产品的事件比例（%） | 5% | 10% | 20% |
| 胰高血糖素急救年使用次数（万次） | 6.0 | 15.0 | 38.0 |
| dasiglucagon在胰高血糖素市场中的份额（%） | 10% | 20% | 30% |
| dasiglucagon年使用次数（万支） | 0.60 | 3.00 | 11.40 |
| 单价（元人民币/支）[D级-基于假设] | 400 | 500 | 600 |
| 峰值销售额（亿元人民币/年） | 0.24 | 1.50 | 6.84 |

### 2.3 峰值销售额合理性校验与取中值

参考数据交叉验证：
- Gvoke 2025年美国净收入：9,410万美元（约6.8亿元人民币）[P1-002]
- 美国胰岛素使用患者约800万人，中国市场约1,100-1,700万人
- 但中国市场支付能力和品类认知远低于美国成熟市场

**综合分析**：基准情景1.50亿元偏保守（仅假设10%事件使用胰高血糖素），乐观6.84亿元偏激进（假设20%事件使用+30%份额）。**取中间值3.0-3.5亿元人民币/年作为本Gate输出的基准峰值销售额**。该值处于保守→基准→乐观三角中的合理位置，且与全球CAGR 10.40%的中国增长预期一致。

### 2.4 峰值销售额分级门槛对照（依据§5.5）

dasiglucagon的适应症为严重低血糖急救，适应症规模判定为中适应症（目标患者数5-50万人）：
- 基准情景门槛：≥3亿元人民币
- 乐观情景底线：≥5亿元人民币

**初步结论**：
- 基准3.0-3.5亿元 → **达门槛** ✅
- 乐观5.0-6.8亿元 → **超过乐观底线** ✅
- 保守0.24亿元 → 远低于门槛（但保守情景假设品类渗透极低，不作为决策依据）

| 风险维度 | 评估 | 详情 |

| 处方惯性 | 🟡 中 | 鼻喷产品获得先发优势后，医生和患者可能形成使用习惯 |

**先发制人价值评估**：如果集团不引进dasiglucagon，最可能的替代获得者：
- 其他中国药企通过License-in引进（但Xeris/Zealand的中国授权谈判窗口是否还在开放？）
- 或者Baqsimi独占中国市场

先发制人价值评级：**中威胁** — Baqsimi（胰高血糖素鼻喷）抢占了中国胰高血糖素市场的先机，但两者剂型不同（鼻喷 vs 注射），目标使用场景可能部分区隔（鼻喷适合严重鼻塞以外的场景，自动注射笔更适合需要注射给药确定性的场景）。此外，中国胰高血糖素急救市场仍在品类教育阶段，后入者仍有空间。

## 4. 治疗路径动态推演 [阶段A]

### 4.1 当前治疗序列（2026年中国）

```
糖尿病管理
  ├─ 血糖监测 + 胰岛素/口服降糖药（基础治疗）
  ├─ 低血糖事件发生
  │   ├─ 轻度低血糖（意识清醒）→ 口服葡萄糖/含糖饮料
  │   ├─ 中度低血糖（意识模糊但可吞咽）→ 口服葡萄糖/家人协助
  │   └─ 严重低血糖（意识丧失/无法吞咽）
  │       ├─ 院外：急救送医（120急救车）→ 静注葡萄糖（占绝大多数）
  │       └─ 院内：静注葡萄糖或注射胰高血糖素
  └─ 极少家庭拥有胰高血糖素急救产品（<5%）
```

### 4.2 推演：3年后（2029年）

```
随着Baqsimi上市5年 + 可能的dasiglucagon获批
  ├─ 品类教育逐步成熟
  ├─ Baqsimi鼻喷：占据"家庭急救箱"定位（非专业人员首选）
  ├─ dasiglucagon自动注射笔：占据"机构/学校/工作场所"定位（护理人员/学校护士操作）
  ├─ 传统急救盒：退至院内使用
  └─ 预计拥有胰高血糖素急救产品的家庭比例：8-15%
```

### 4.3 推演：5年后（2031年）

```
品类竞争充分化
  ├─ NRDL覆盖使胰高血糖素急救产品可负担性大幅提高
  ├─ 自动注射笔和鼻喷形成稳定双品类（非零和博弈）
  ├─ 数字化管理（CGM（连续血糖监测）联动自动预警）改变急救模式
  ├─ 可能的新进入者：国产dasiglucagon类似药/仿制药
  └─ dasiglucagon的竞争护城河取决于：①中国专利保护强度 ②品牌建设 ③渠道深耕
```

## 5. 管线协同评估 [阶段A]

| 协同维度 | 评估 | 详情 |
|---------|------|------|
| 销售协同 | 🟢 高（>70%重叠） | dasiglucagon的目标科室（内分泌科/糖尿病科）与深康现有的心肾代谢业务高度重合 |
| 适应症扩展 | 🟡 中 | dasiglucagon有先天性高胰岛素血症、减肥术后低血糖的潜在扩展适应症（但均为罕见病示范） |
| 联合用药 | 🔴 无 | 严重低血糖急救产品与任何慢性治疗药物不构成联合用药 |
| 平台价值 | 🔴 无 | 单品种引进，无技术平台或后续管线优先权 |

**综合评级**：**中等协同**

- 核心优势：销售渠道复用。深康在糖尿病/内分泌领域的学术关系和商业化网络可以无缝承接dasiglucagon
- 核心劣势：无平台价值、无联合用药、无后续管线，属于单品种交易

*根据A-1 v1.2的条件通过规则*：如果管线协同评级为"中等协同"且基准PeakSales达标，可在Gate 3给出"条件通过"结论。
