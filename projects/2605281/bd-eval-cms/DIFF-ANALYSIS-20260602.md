#### 1.3.4 核心差异总结

**新版 SKILL.md 的本质**：
- 是一个**静态规范文件**，定义体系规则、Gate逻辑、财务门槛、一票否决清单
- **不包含可执行的操作流程**（无 Phase、无人spawn、无脚本调用）
- 更接近"技能定义"而非"执行流水线"

**旧版 SKILL.md 的本质**：
- 是一个**可执行流水线**，包含 Phase 1~5.5 的具体操作步骤、子Agent spawn策略、脚本调用
- 是 Agent 可直接执行的操作手册

**兼容性结论**：新版 SKILL.md 是旧版 SKILL.md 的**上游规范层**，两者是**互补关系**而非替代关系。旧版的 Phase 流水线执行能力不可丢弃。
