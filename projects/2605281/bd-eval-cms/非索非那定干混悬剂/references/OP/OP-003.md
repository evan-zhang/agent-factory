# OP-003 中国抗组胺药市场格局（2023-2024）

- **来源**: 健康界/摩熵医药 2025-03-28
- **URL**: https://www.cn-healthcare.com/articlewm/20250328/wap-content-1647644.html
- **抓取日期**: 2026-05-31

## 核心数据

1. 2023年全国医院端抗组胺药物销售额：**50.94亿元**
2. 2024年上半年全国医院端抗组胺药物销售额：**25.69亿元**（年化约51.4亿元）
3. 2023年全国药店零售端抗组胺药物销售额：**26.25亿元**
4. 2024年上半年全国药店零售端：**12.55亿元**（年化约25.1亿元）
5. 合计：院端+零售约**76-77亿元/年**（2024年估计）

## 竞争格局

- 比拉斯汀（新一代抗组胺药）与H1受体亲和力比非索非那定高6倍
- 比拉斯汀片已有美纳里尼（5.1类进口）和华阳制药（4类首仿）获批
- 比拉斯汀口服溶液多家在审，竞争激烈
- 非索非那定面临新一代抗组胺药的替代竞争
