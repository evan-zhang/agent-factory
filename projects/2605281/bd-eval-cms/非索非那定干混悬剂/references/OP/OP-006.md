# OP-006 敏菲乐临床研究进展（儿童）

- **来源**: 中国临床试验注册中心（ChiCTR）2025-07-21
- **URL**: https://www.chictr.org.cn/showproj.html?proj=278991
- **抓取日期**: 2026-05-31

## 核心信息

1. 注册临床试验：盐酸非索非那定干混悬剂用于儿童慢性变应性鼻炎伴哮喘的有效性及安全性
2. 多中心、前瞻性、随机、阳性对照研究
3. 注册号已分配（具体号略），注册时间2025-07-21
4. 说明济川药业正在积极推进敏菲乐的儿童适应症临床证据积累

## 对评估的影响

- 济川药业的临床投入持续加大，先发优势不断巩固
- 辰光品种即使获批，也需要面对济川已有的临床证据和品牌认知
