# OP-001 济川药业盐酸非索非那定干混悬剂获批（首家过评）

- **来源**: 新京报 2024-06-20
- **URL**: https://m.bjnews.com.cn/detail/1718861639169182.html
- **抓取日期**: 2026-05-31

## 核心信息

1. 2024年6月20日，济川药业全资子公司济川有限的盐酸非索非那定干混悬剂获批上市
2. 国内**首个**获批盐酸非索非那定干混悬剂型的药企，视同通过仿制药质量和疗效一致性评价
3. 适应症：缓解6个月以上儿童和成人的季节性过敏性鼻炎相关症状、慢性特发性荨麻疹的皮肤症状
4. 济川有限采用与参比制剂不同的掩味技术，提高适口性
5. 累计研发支出约1688.27万元（未经审计）
6. 原研干糖浆2014年在日本获批上市（赛诺菲），规格15mg/0.3g、30mg/0.6g

## 市场数据（米内网）

- 2023年中国城市实体药店非索非那定制剂销售额：338万元
- 合计约5004万元（2023年院内+零售）
