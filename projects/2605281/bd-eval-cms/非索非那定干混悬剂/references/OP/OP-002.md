# OP-002 敏菲乐进入国家医保目录及市场推广进展

- **来源**: 济川药业官网/健康界 2025-02
- **URL**: http://jichuantes.eastdays.net/cn/newsd.html?id=4336
- **抓取日期**: 2026-05-31

## 核心信息

1. 商品名：**敏菲乐**
2. 规格：15mg和30mg双规格
3. 原研干混悬剂剂型**尚未在国内上市**（济川为首仿/首家过评）
4. 已成功通过医保谈判，进入**2024年国家医保药品目录**
5. 荣获2024第五届论健·年度锐意创新之星榜单（中国医药新闻信息协会指导）
6. 专为6个月以上患儿设计
7. 济川药业在呼吸、儿科领域有成熟销售渠道（蒲地蓝消炎口服液、小儿豉翘清热颗粒等）

## 竞争格局关键发现

- 济川药业为国内**首家**获批干混悬剂，辰光制药NDA（2025-02-18提交）为后续申报者
- 辰光制药并非"独家剂型"——济川已抢占先发优势
