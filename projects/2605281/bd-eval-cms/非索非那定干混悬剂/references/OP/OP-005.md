# OP-005 济川药业战略布局与敏菲乐定位

- **来源**: CFN金融/中国金融网 2026-02-03
- **URL**: https://www.cfn.cn/index/tushuo/eyesCfnNewsDetail/id/63103.shtml
- **抓取日期**: 2026-05-31

## 核心信息

1. 济川药业已布局JAK1/2抑制剂普美昔替尼（PG-011）鼻喷雾剂（过敏性鼻炎），最高1亿元合作对价
2. 敏菲乐已进入国家医保目录，为普美昔替尼未来覆盖儿童患者奠定基础
3. 济川药业长期聚焦儿科、呼吸、消化、妇科，核心产品：小儿豉翘清热颗粒、蒲地蓝消炎口服液
4. 2024年研发投入4.53亿元（占营收5.65%），研发人员734人
5. 济川药业正在向"中药+仿制药+创新药"均衡发展转型

## 对辰光品种的启示

- 济川药业已有敏菲乐（干混悬剂首家过评）+ 医保目录 = 强渠道壁垒
- 济川正在构建完整的过敏性鼻炎产品矩阵（口服+鼻喷+JAK抑制剂）
- 辰光品种作为后发者，面临济川先发优势+医保覆盖+渠道壁垒的三重挑战
