# G2-003 国药集团广东环球制药干混悬剂制剂专利 CN106806346B

## 来源

## 关键信息

### 专利基本信息
- 专利号: CN106806346B
- 专利名称: 一种非索非那定盐酸盐干混悬剂制剂及其制备方法
- 申请号: CN201510873888.4A
- 申请人: 上海医药工业研究院 + 国药集团广东环球制药有限公司

### 时间节点
- 申请日/优先权日: **2015-12-01**
- 公开日: 2017-06-09（CN106806346A）
- 授权日: **2020-09-11**（CN106806346B）
- **预计到期日: 2035-12-01**
- 状态: **有效（Active）**

### 专利权利要求范围
- 保护非索非那定盐酸盐干混悬剂制剂的具体处方组成
- 处方组成: 非索非那定盐酸盐0.8%~4%、填充剂80%~93%、甜味剂0.2%~1.2%、助悬剂0.5%~1%、矫味剂0.2%~2%
- 保护制备方法

### 对辰光制药干混悬剂的影响评估
该专利为国药集团广东环球制药的制剂处方专利，保护特定处方配比。影响评估：
1. **处方绕过可行性**: 辰光制药可选择不同处方配比（不同辅料种类/比例）来规避该专利
2. **济川药业先例**: 济川药业已于2024年获批干混悬剂，说明存在可绕过该专利的处方方案
3. **方法专利**: 专利同时保护制备方法，辰光制药需确保生产工艺不侵权
4. **风险等级**: 中（可通过处方设计规避，但需详细FTO分析）

### 注意事项
该专利到期日为2035年，距现在约9年，是长期存在的制剂壁垒。但由于保护的是特定处方组成而非整个干混悬剂剂型概念，可通过差异化处方设计规避。
