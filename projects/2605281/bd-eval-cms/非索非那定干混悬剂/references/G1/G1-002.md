# G1-002: 盐酸非索非那定BCS分类与BE豁免可行性

- **来源**: Journal of Pharmaceutical Sciences (Elsevier) / PMC (NIH) [G1-002]
- **URL1**: https://tus.elsevierpure.com/en/publications/biowaiver-monograph-for-immediate-release-solid-oral-dosage-forms-3/
- **URL2**: https://pmc.ncbi.nlm.nih.gov/articles/PMC9379123/
- **查询日期**: 2026-05-31

## 关键信息

### BCS分类争议
1. **BCS Class III说**：部分文献将盐酸非索非那定归为BCS III类（高溶解性、低渗透性），口服生物利用度仅约33%，受P-gp外排影响显著 [PMC9379123]
2. **BCS Class IV说**：2024年Journal of Pharmaceutical Sciences发表的Biowaiver Monograph指出，非索非那定既非高溶解也非高渗透，应归为BCS IV类。在不同pH下溶解度差异大：pH 3.0高溶，pH 4.5和5.6低溶 [JPharmSci 2024]

### BE豁免可行性
3. 由于BCS分类存在争议（III vs IV），且无论III类还是IV类均**不符合**BCS I类药物的BE豁免条件
4. 干混悬剂作为口服液体制剂，虽溶出不是限速步骤，但仍需进行正式的BE研究
5. 恒瑞医药盐酸非索非那定片于2021年12月通过一致性评价（第二家），表明片剂BE可行

### CDE指南参考
6. 根据CDE《人体生物等效性试验豁免指导原则》，仅BCS I类（高溶高渗）的速释固体制剂可申请BE豁免
7. 非索非那定不符合BCS I类标准，BE豁免不可行，需开展正式BE试验
