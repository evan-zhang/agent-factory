# G1-004: 恒瑞医药盐酸非索非那定片过评——口服固体制剂一致性评价先例

- **来源**: 恒瑞医药官网公告 [G1-004]
- **URL**: https://www.hengrui.com/media/detail-127.html
- **查询日期**: 2026-05-31

## 关键信息

1. 2021年12月8日，恒瑞医药子公司成都盛迪医药的盐酸非索非那定片通过仿制药一致性评价（第二家过评企业）
2. 适应症：（1）季节性过敏性鼻炎——适用于缓解成人和6岁及6岁以上儿童的季节性过敏性鼻炎相关症状；（2）慢性特发性荨麻疹——适用于治疗成人和6岁及6岁以上儿童的慢性特发性荨麻疹的皮肤症状
3. 恒瑞医药于2004年6月24日获得该品种的药品注册批件，2014年12月许可成都盛迪医药生产
4. 过评有利于市场竞争力提升和临床优先选用

## 对本项目的参考意义
- 同API片剂已有多家过评，口服固体制剂BE路径成熟
- 干混悬剂作为液体制剂，与片剂BE不能互认，需单独开展
- 过评片剂可作为干混悬剂BE研究的参比制剂PK数据对照参考
