# G1-003: CDE参比制剂目录——盐酸非索非那定口服混悬液

- **来源**: NMPA《化学仿制药参比制剂目录（第七十批）》 [G1-003]
- **URL**: https://www.nmpa.gov.cn/directory/web/nmpa/images/1689586446582015811.docx
- **查询日期**: 2026-05-31

## 关键信息

1. CDE参比制剂目录第七十批中收录了"盐酸非索非那定口服混悬液"（Fexofenadine Hydrochloride Oral Suspension）
2. 干混悬剂参比制剂为日本获批的赛诺菲干糖浆（2014年），规格15mg/0.3g、30mg/0.6g
3. 参比制剂已列入CDE参比制剂推荐目录，申报路径明确
4. 济川药业2024年获批的干混悬剂即以日本参比制剂为参照，按3类仿制药路径申报
5. 辰光制药的NDA（2025-02-18提交）大概率也参照同一参比制剂
