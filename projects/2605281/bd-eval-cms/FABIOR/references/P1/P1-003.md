# [P1-003] Mayne Pharma Acquires Foam Assets from GSK (2016)
- **URL**: https://www.asx.com.au/asxpdf/20160818/pdf/439dpzck6q9784.pdf
- **抓取时间**: 2026-05-31 15:49 CST
- **来源类型**: 公司公告

## 原文内容
Mayne Pharma于2016年8月以约5010万美元从GSK收购皮肤科泡沫产品组合（Foam Assets），包括FABIOR (tazarotene) Foam和SORILUX (calcipotriene) Foam的美国权利。Mayne Pharma的Specialty Brands部门于2017年1月在美国重新推出这些产品。

## 关键数据点
- 收购价格：约5010万美元
- 收购时间：2016年8月
- 收购内容：FABIOR和SORILUX的美国市场权利
- FABIOR是有专利保护的外用泡沫产品
- 痤疮是皮肤科最大的适应症领域
