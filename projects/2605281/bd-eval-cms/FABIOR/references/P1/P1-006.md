# [P1-006] Mayne Pharma财务概况
- **URL**: https://www.maynepharma.com/investor-relations/results-reports/
- **抓取时间**: 2026-05-31 15:50 CST
- **来源类型**: 公司公告

## 原文内容
Mayne Pharma是一家澳大利亚上市公司（ASX: MYX），美国总部位于北卡罗来纳州。主要业务包括皮肤科、女性健康和仿制药。
- FY24：皮肤科收入1.749亿美元，同比增长207%（主要因收购Sol-Gel产品）
- FY25：皮肤科收入1.541亿美元，同比下降12%（受仿制药竞争加剧影响）
- HY25：皮肤科收入8140万美元，同比增长1%
- 品牌皮肤科产品：FABIOR, SORILUX, ABSORICA, TWYNEO, EPSOLAY
- 2025年全年收入约4.081亿澳元

## 关键数据点
- Mayne Pharma总部：澳大利亚（ASX上市），美国运营总部在北卡
- 皮肤科是核心业务之一
- FABIOR是其品牌皮肤科产品组合中的重点产品
- 2026年2月HY26报告显示业绩韧性
