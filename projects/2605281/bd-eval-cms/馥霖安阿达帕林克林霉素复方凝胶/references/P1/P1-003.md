# [P1-003] 【聚焦】北京福元医药900万喜提亿级新品，抢进口新药首仿
- **URL**: https://finance.sina.com.cn/wm/2026-05-26/doc-inhzffsk2911905.shtml
- **抓取时间**: 2026-05-31 08:58 CST
- **来源类型**: 行业媒体（新浪财经转载米内网）

## 原文内容
在中国三大终端六大市场，近几年抗痤疮制剂的市场规模稳步增长，2024-2025年均达到了16亿元以上，目前零售药店终端（城市实体药店+网上药店）是销售主阵地，2025年合计占比超过60%。

北京福元医药在公告中提到，公司针对氯吡格雷阿司匹林片累计研发投入为900.19万元。5月20日，福元药业提交了阿达帕林过氧苯甲酰凝胶的4类仿制上市申请获得CDE承办，公司将参与该产品国内首仿争夺。

阿达帕林过氧苯甲酰凝胶是国内首个获批的阿达帕林与过氧苯甲酰复方制剂，每日仅需使用一次，最快一周内见效，且无耐药风险，目前包括福元药业在内已有16家国内药企参与该产品国内首仿争夺，竞争激烈。

2025年在零售药店终端，北京福元医药是抗痤疮制剂TOP5集团（二级），畅销产品包括了红霉素软膏、阿达帕林凝胶、硫软膏。

## 关键数据点
- 中国抗痤疮制剂市场规模：2024-2025年均达到16亿元以上
- 零售药店终端占比：2025年合计超过60%
- 阿达帕林过氧苯甲酰凝胶：国内首个阿达帕林与过氧苯甲酰复方制剂
- 参与阿达帕林过氧苯甲酰凝胶首仿争夺的国内药企：16家
- 2026年1月12日：浙江华海药业按化药注册分类4类申报的阿达帕林凝胶上市申请获受理
