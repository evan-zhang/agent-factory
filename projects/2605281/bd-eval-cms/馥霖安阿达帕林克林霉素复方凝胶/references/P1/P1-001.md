# [P1-001] 针对痤疮！「阿达帕林克林霉素凝胶」在中国获批上市
- **URL**: https://bydrug.pharmcube.com/news/detail/ff6280ade982b72244d6c1e86dcd3153
- **抓取时间**: 2026-05-31 08:55 CST
- **来源类型**: 行业媒体

## 原文内容
2月23日，中国国家药监局（NMPA）官网公示，兆科眼科公司申报的阿达帕林克林霉素凝胶上市申请已获得批准。公开资料显示，这是兆科眼科研发的一款用于治疗中度寻常型痤疮的创新产品，李氏大药厂拥有该产品在大中华地区的商业化权利。阿达帕林为一种视黄酸受体（RAR）激动剂，可刺激皮肤生长；克林霉素为一种抗生素，可阻断细菌蛋白质合成。研究已经证明，在寻常型痤疮治疗中，将两种药物结合使用之疗效会较单独使用更佳。兆科眼科于2020年与李氏大药厂签订许可协议，授予后者在大中华地区商业化阿达帕林克林霉素凝胶的独家权利。

## 关键数据点
- NMPA批准日期：2024年2月23日
- 申报方：兆科眼科
- 商业化权利方：李氏大药厂（大中华地区独家）
- 适应症：中度寻常型痤疮
- 作用机制：阿达帕林（RAR激动剂）+克林霉素（抗生素）
- 合作协议签订时间：2020年
- 协议范围：大中华地区商业化独家权利
