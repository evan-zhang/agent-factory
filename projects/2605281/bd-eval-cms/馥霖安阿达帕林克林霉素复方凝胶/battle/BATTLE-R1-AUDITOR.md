## 1. 逐Gate审查

### Gate 1：前提门（产品力现状评估）

**结论合理性**：通过 ✅ — 产品力评分4.05/5，判断为"强"，合理。

**异议清单**：

🟡-G1-01：**阿达帕林+克林霉素复方制剂专利保护期未确认**
- Gate 1提及"兆科眼科持有一种凝胶制剂中阿达帕林的分散工艺专利"（中国专利号201711392438.9）[D0-001]，但未明确该专利到期时间
- 建议：补充专利到期年份，以更准确评估独占窗口期
- 影响：中等 — 直接影响竞品威胁时间线

🟢-G1-01：**Cabtreo（Galderma三联凝胶）中国上市时间线缺失**
- Gate 1提及Cabtreo在美国已上市（2023年10月），但未评估其进入中国的时间线
- 建议：补充Cabtreo在中国的注册/申报状态（如有）
- 影响：低 — 三联凝胶对中国市场的影响为中长期

### Gate 2：定调门（渠道资产与商业化可行性评估）

**结论合理性**：条件通过 ✅ — 首次商业化（非迁移），渠道需新建，判断合理。

**异议清单**：

🟢-G2-01：**德镁皮肤科现有推广团队信息缺失**
- Gate 2未提供德镁医药皮肤科推广团队的具体信息（规模、分布、产品线）
- 这是评估渠道建设难度的关键变量
- 建议：补充德镁皮肤科团队基本画像
- 影响：低 — 不影响整体结论方向，但影响渠道建设成本估算精度

### Gate 3：证据门（市场现状与增长空间评估）

**结论合理性**：条件通过 ✅ — 峰值销售额4.0亿元（基准），市占率~25%，数据支撑充分。

**异议清单**：

🔴-G3-01：**增量来源拆解中联用方案贡献偏高**
- Gate 3给出"与喜辽妥联用方案协同"增量2,000~4,000万元，但喜辽妥联用方案仅为概念性提及[P1-003]，无具体合作协议或临床数据支撑
- 该增量计入基准峰值可能过于乐观
- 建议：将联用方案增量移至乐观情景，或标注为"待确认联用合作后重估"
- 影响：高 — 联用增量占总增量5%~9%，影响峰值销售额测算

### Gate 4：支付门（价格·医保·准入评估）

**结论合理性**：条件通过 ✅ — 集采短期低/中期中/长期高的判断合理。

**异议清单**：

🟡-G4-01：**馥霖安是否已纳入NRDL未确认**
- Gate 4明确指出馥霖安医保身份"待确认"，但未说明获取该信息的路径
- 建议：明确信息获取计划（如联系CMS准入团队或查询国家医保局网站）
- 影响：中等 — 医保身份直接影响院内准入和定价策略

🟡-G4-02：**DRG/DIP对皮肤科门诊影响分析过简**
- Gate 4仅一句提及"DRG/DIP影响较小（门诊不计入住院DRG）"，但未考虑日间手术/门诊特殊病种等DRG扩展场景
- 建议：补充皮肤科门诊DRG/DIP政策扫描（至少覆盖Top 5省份）
- 影响：低-中 — 外用制剂以门诊为主，DRG影响有限

### Gate 5：成本门（团队与运营成本评估）

**结论合理性**：条件通过 ✅ — SG&A估算1,900~3,600万元/年合理。

**异议清单**：

🔴-G5-01：**进货成本为关键变量但未能量化**
- Gate 5明确指出"进货成本为最大不确定性"，但未给出合理的估算区间
- 上游交易结构已知（李氏大药厂向兆科眼科分成），但CMS/德镁侧的进货成本完全未知
- 建议：基于行业惯例给出进货成本的三情景估算（已标D级），并明确"需在商务谈判中确认"为最高优先级
- 影响：高 — 进货成本直接决定毛利率和IRR

🟡-G5-02：**过渡期双付成本估算可能偏低**
- Gate 5估算过渡期双付成本200~300万元（一次性），但如果团队交接不顺利，过渡期可能延长至18个月，双付成本可能达500~800万元
- 建议：将过渡期双付成本调整为200~500万元（基准），并标注最高风险情景800万元
- 影响：中等 — 不改变整体结论，但影响首年现金流

### Gate 6：可做门（交易结构与财务回报评估）

**结论合理性**：条件通过 ✅ — 基准情景下全部财务门槛达标，IRR 25%~35%，推荐附条件推进合理。

**异议清单**：

🟡-G6-01：**上游分成比例假设（20%）的依据不足**
- Gate 6基准情景假设上游分成比例为收入的20%[D级-基于假设]，但未给出行业对标或推理依据
- 已知李氏大药厂与兆科眼科为"利润分成"模式[G6-001]，CMS/德镁作为下游还需额外向李氏大药厂支付
- 建议：补充分成比例的行业对标（国内处方药商业化授权引进通常为15%~30%）
- 影响：中等 — 分成比例每变化5%，IRR变化约3%~5%

🟢-G6-01：**风险调整后IRR计算方法可优化**
- Gate 6使用概率加权法（50%/30%/20%）计算风险调整后IRR，但未说明概率赋值依据
- 建议：使用蒙特卡洛模拟或至少提供概率赋值的理由
- 影响：低 — 结论方向不变
