## 结论卡

**【Gate 4：定调门（准入与定价路径）】**

**结论：停止**

**置信度：中（准入路径分析确信度高，定价测算置信度中）**

**关键支撑证据：**
1. 2025年12月，两款长效生长激素（金赛增、益佩生）已纳入NRDL（国家基本医疗保险目录），降价53-75%，长效年费用降至3万元[G4-001] [G4-002] [G4-003]
2. Humatrope为短效日制剂，年治疗费用预估5-7万元（与国内短效水针6-7万元接近）[G4-003]，在长效已降至3万元/年的背景下，Humatrope**既无价格优势也无依从性优势**
3. 生长激素已进入多省集采（广东11省联盟2022年、福建/河北/浙江等）[G4-003]，进口短效制剂面临更大价格压力
4. NRDL已纳入长效产品，Humatrope短效制剂进入NRDL的可能性极低——NHSA（国家医疗保障局）已选择更优依从性的长效产品作为医保覆盖重点
5. 金赛药业87.52%市占率（医院端）+ NRDL长效覆盖 → 医院处方路径已被金赛+医保锁定，进口短效几无进院可能

**需补充证据 Top 5：**
1. Humatrope全球定价数据及礼来中国定价意向（预计获取难度：高）
2. 各省生长激素集采中选价格明细（预计获取难度：中）
3. DRG/DIP对儿童内分泌疾病住院费用的影响（预计获取难度：中）
4. 商业保险（百万医疗/惠民保）覆盖生长激素的情况（预计获取难度：中）
5. 进口rhGH在省际联盟集采中的参与先例和价格（预计获取难度：高）

**红旗事项：**
- ⚠️ **准入路径几乎完全封闭**：NRDL已选长效，集采压短效价格，自费市场被国产品牌占据——Humatrope在三条准入路径（NRDL/集采/自费）上均无优势
- ⚠️ **定价倒挂风险**：长效NRDL后年费3万 vs Humatrope短效年费5-7万 → 进口短效价格反而高于国产长效，完全无法竞争

**下一步行动：**
- 建议立即终止Gate 4评估并向投委会报告：Humatrope准入路径已封闭，无商业化可行性

## 1. NRDL（国家医保目录）谈判准入潜力评估 [阶段A]

### 1.1 当前NRDL格局

2025年12月，国家医保局将**两款长效生长激素**首次纳入NRDL：
- 金赛药业**金赛增**（金培长效生长激素注射液）：9mg规格从3500元降至约900元/支，降幅约75% [G4-002] [G4-003]
- 特宝生物**益佩生**（怡培生长激素注射液）：5mg规格从1798元降至853.2元/支，降幅约53% [G4-001] [G4-003]

2026年1月1日起正式执行。30kg体重患儿长效制剂年治疗费用从约12万降至**3万元** [G4-002]。

### 1.2 Humatrope进入NRDL的可能性判断

**结论：可能性极低。** 理由如下：

① **NHSA已做选择**：医保局已选择依从性更优的长效周制剂作为覆盖重点，同一领域不太可能再纳入短效日制剂

② **缺乏差异化价值**：Humatrope为短效日制剂，与已纳入NRDL的长效制剂相比无临床优势，NRDL谈判缺乏价值支撑

③ **分子层面无创新性**：somatropin为成熟分子，国内已有多个国产短效品种上市，Humatrope作为进口短效无"填补空白"的谈判理由

④ **价格劣势明显**：即使Humatrope愿意大幅降价，短效日制剂在依从性上天然劣于长效周制剂，NHSA不会为依从性更差的产品支付额外费用

**NRDL准入评估：❌ 不推荐进入。** [D级-基于假设]

### 2.3 定价困境分析

**核心矛盾**：长效已降至3万/年（NRDL后），Humatrope短效即使以基准价5-7万/年定价，也**比长效贵2-4万且依从性更差**。

患者和医生的选择逻辑：
- 预算3万/年 → 选长效（周注射，NRDL覆盖）✅
- 预算3万/年 → 选短效粉针（日注射，集采低价）✅
- 预算5-7万/年 → 选进口短效Humatrope（日注射，无医保）❌

**结论：Humatrope在现有价格体系下找不到有竞争力的定价区间。**

## 3. 自费/商保/院内自购准入策略 [阶段A]

### 3.1 自费市场

中国生长激素治疗以自费为主（NRDL覆盖前），但市场已被国产品牌深度覆盖。金赛87.52%市占率（医院端）[G3-001]意味着患者心智和处方习惯已被锁定。

Humatrope自费策略的可行性：
- **品牌认知劣势**：中国患者和医生对Humatrope品牌认知度极低，礼来未在中国推广过Humatrope
- **渠道劣势**：国产品牌已建立完善的院内+院外处方和配送体系
- **价格竞争力不足**：国产短效粉针2-3万/年，Humatrope作为进口产品成本更高，难以与之竞争

### 3.2 商业保险

生长激素治疗目前主要不在百万医疗险和惠民保的常规覆盖范围（因其主要适应症为儿童GHD，属儿童门诊长期用药，非住院治疗）。

**商业保险覆盖评估**：Humatrope难以通过商保路径获得有意义的市场份额。[D级-基于假设]

### 3.3 院内自购

DRG/DIP（按疾病诊断相关分组/病种分值付费）支付方式改革已基本完成全覆盖。在DRG/DIP框架下，生长激素作为长期门诊用药，对住院费用影响有限，但对医院药占比考核有影响。

**Humatrope进院评估**：
- 医院已有金赛/安科品种，且金赛增已入NRDL
- 医院引进进口短效的意愿极低（无差异化价值，增加药占比压力）
- **进院可行性：极低**

## 4. 省级集采风险评估 [阶段A]

### 4.1 集采现状

生长激素尚未纳入国家层面VBP（带量采购），但已进入多省/省际联盟集采：

| 集采批次 | 覆盖范围 | 时间 | 状态 |
|---------|---------|------|------|
| 广东11省联盟 | 广东+10省 | 2022年 | 首次纳入生长激素 [G4-003] |
| 福建 | 福建省 | 2023年 | 已纳入 |
| 河北 | 河北省 | 2023年 | 已纳入 |
| 浙江 | 浙江省 | 2024年 | 已纳入 |

### 4.2 对Humatrope的影响

集采风险评级：**高**

理由：
1. Humatrope获批上市（最快2030年[G1-001]）时，生长激素可能已进入国家VBP
2. 即使未进入国家VBP，省级集采范围将进一步扩大
3. 进口产品在集采中通常面临更大幅度的降价要求
4. 集采后短效粉针价格已降至年费2-3万，Humatrope的成本结构难以匹配

**预计集采年份**：Humatrope上市前（2028-2030年）生长激素大概率进入国家VBP

## 5. DRG/DIP影响评估 [阶段A]

### 5.1 对生长激素处方的影响

DRG/DIP主要影响住院费用，而生长激素治疗以门诊长期处方为主：
- 儿童GHD诊断期住院（激发试验）→ DRG覆盖，但金额较小
- 长期治疗期 → 门诊处方，DRG不直接影响

**间接影响**：
- 医院药占比考核压力 → 倾向选择已入NRDL的品种（金赛增/益佩生）
- 医院绩效考核 → 处方集目录内品种为主
- **结论：DRG/DIP对Humatrope为负面因素，但非决定性因素**

## 综合评估

### 准入策略结论

| 准入路径 | 可行性 | 说明 |
|---------|--------|------|
| NRDL谈判准入 | ❌ 极低 | NRDL已选长效，短效无价值 |
| 省级集采 | ⚠️ 被动参与 | 上市后可能被纳入，但面临大幅降价 |
| 自费市场 | ❌ 极低 | 国产品牌已深度覆盖，进口短效无竞争力 |
| 商业保险 | ❌ 极低 | 生长激素不在商保常规覆盖范围 |
| 院内进院 | ❌ 极低 | 医院无意愿引进进口短效 |

### 定价结论

Humatrope在中国市场**无法找到有竞争力的定价区间**：
- 比长效贵且依从性差 → 患者不会选
- 比国产短效贵 → 价格敏感患者不会选
- 降价到国产短效水平 → 进口成本无法支撑

**Gate 4综合结论：准入路径全面封闭，定价竞争力为零，建议停止推进。**
