# [P1-002] 2025年中国生长激素行业市场规模及重点企业
- **URL**: https://finance.sina.com.cn/roll/2025-09-14/doc-infqnmfc0473661.shtml
- **抓取时间**: 2026-05-31 15:49 CST
- **来源类型**: 行业媒体

## 原文内容
中国生长激素市场规模由2018年的6亿美元增至2022年的17亿美元，年复合增长率为29.3%。
短效水针是主流剂型，市场份额约60%。短效粉针因注射过程复杂且易引起不适，市场接受度较低。长效水针作为新剂型，有望持续抢占市场份额。
金赛药业子公司长春高新2022年总营收126.27亿元，同比增长17.5%，主要产品：注射用人生长激素（粉针）、重组人生长激素注射液（水针）、聚乙二醇重组人生长激素注射液（长效水针）。

## 关键数据点
- 中国市场2018年6亿美元→2022年17亿美元，CAGR 29.3%
- 短效水针占60%市场份额
- 金赛药业国内市占率领先，2022年营收126.27亿元
- 长效水针为新增长方向
