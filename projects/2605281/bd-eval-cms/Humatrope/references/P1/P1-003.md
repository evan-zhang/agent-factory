# [P1-003] 全球人类生长激素药物市场：2032年综合分析
- **URL**: https://www.wiseguyreports.com/cn/reports/human-growth-hormone-medicine-market
- **抓取时间**: 2026-05-31 15:48 CST
- **来源类型**: 行业报告

## 原文内容
2023年全球人类生长激素药物市场规模约56.3亿美元，预计从2024年的59.5亿美元增长到2032年的93亿美元，CAGR约5.74%。
主要参与者：辉瑞（Genotropin）、礼来（Humatrope）、诺和诺德（Norditropin）、默克（Saizen）、易普生等。
礼来公司主要产品Humatrope用于治疗儿童和成人生长激素缺乏症。
生长激素缺乏症（GHD）在儿童中的患病率约为1/4000~1/10000。

## 关键数据点
- 全球市场2023年56.3亿美元，2032年预计93亿美元
- CAGR 5.74%
- Humatrope为礼来核心生长激素产品
- 儿童GHD患病率1/4000~1/10000
- 竞品：Genotropin(辉瑞)、Norditropin(诺和诺德)、Saizen(默克)
