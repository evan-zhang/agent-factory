# [G2-003] somatrogon（周制剂）vs somatropin（日制剂）比较研究
- **URL**: https://experts.umn.edu/en/publications/efficacy-and-safety-of-weekly-somatrogon-vs-daily-somatropin-in-c/
- **抓取时间**: 2026-05-31 15:57 CST
- **来源类型**: 学术文献

## 原文内容
J Clin Endocrinol Metab, 2022.
Somatrogon是一种长效重组人生长激素（rhGH），开发用于儿童GHD的每周一次治疗。
Phase 3研究比较somatrogon（周制剂）vs somatropin（日制剂）的疗效和安全性。
结果显示somatrogon的年生长速率（height velocity）非劣效于日制剂somatropin。
安全性相当，但周制剂治疗负担更低，治疗体验更优。
Asian children可能因漏打更少而从周制剂获益更多。

## 关键数据点
- somatrogon周制剂非劣效于somatropin日制剂
- 安全性相当
- 周制剂治疗负担显著低于日制剂
- 亚洲儿童依从性改善更明显
