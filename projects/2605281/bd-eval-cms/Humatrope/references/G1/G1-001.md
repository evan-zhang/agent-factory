# [G1-001] 桥接试验百科 — ICH E5框架与NMPA接受境外数据指南
- **URL**: https://chinamedglobal.com/wiki/bridging-study
- **抓取时间**: 2026-05-31 15:55 CST
- **来源类型**: 学术/行业

## 原文内容
桥接试验基于ICH E5指南，用于评估种族因素对药物安全性和有效性的影响，帮助将境外临床数据外推至新区域。
NMPA 2018年发布《接受药品境外临床试验数据的技术指导原则》：
- 境外临床试验需符合ICH E6(GCP)要求
- 数据需来自规范的多中心临床试验
- 需评估种族差异对药物安全性和有效性的影响
PK桥接通常24-48人，周期较短。
如境外关键试验包含10%-20%亚裔/中国受试者，NMPA可能直接接受数据。

## 关键数据点
- NMPA接受境外数据的三个条件：ICH GCP合规+多中心+种族差异评估
- PK桥接通常24-48人
- MRCT中国亚组10%-20%可直接接受
- 重组人生长激素属多肽激素，可能存在种族敏感性（体重差异）
