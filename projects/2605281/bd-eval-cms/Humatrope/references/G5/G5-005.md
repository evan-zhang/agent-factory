# [G5-005] PMC — rhGH治疗成本效果分析
- **URL**: https://pmc.ncbi.nlm.nih.gov/articles/PMC6710537/
- **抓取时间**: 2026-05-31 16:07 CST
- **来源类型**: 学术文献

## 原文内容（搜索摘要）
2019年发表于PMC的成本效果分析：
- "Somatropin with easypod can be cost-saving versus all other r-hGH treatments except Omnitrope when listed drug prices are considered."
- 生长激素治疗费用在不同品牌之间存在显著差异
- Omnitrope（Sandoz生物类似药）在价格上最具竞争力

## 关键数据点
- 不同品牌rhGH治疗成本差异显著
- Omnitrope（生物类似药）价格最低
- Humatrope在成本效果上并非最优选择
- 这对中国市场定价策略有重要参考价值
