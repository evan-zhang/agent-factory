## 2.1 渠道资产盘点

### 医院覆盖

注射用心肌肽的目标终端高度集中：心外科手术仅在三级医院开展，2023年全国心脏外科手术28.9万人次中98.1%由三级医院完成[G2-001]。

目标科室：心外科、麻醉科、重症医学科（ICU）。

关键覆盖特征判断：

- **手术量天花板**：2023年全国CABG（冠脉旁路移植术）约6万例 + 瓣膜手术约5.6万例 + 其他心外科手术，合计约28.9万例[G2-001][G2-003]。产品覆盖的手术类型（体外循环CABG + 非体外循环CABG + 瓣膜置换）占心脏外科手术主体，潜在适用手术量约20~25万例/年 `[D级-基于假设]`。
- **医院集中度极高**：心脏外科手术量Top 10省份/直辖市（北京、山东、河南、广东、上海、江苏、四川、湖北、浙江、辽宁）贡献全国手术量约70%以上 `[D级-基于假设]`，北京阜外、安贞、解放军301等头部中心为核心目标医院。
- **现有覆盖规模未知**：珍奥未公开其医院覆盖数据。P1-004显示四期临床在12家大型心血管病医院开展（包括阜外、安贞、301、北大一院、沈阳军区总院、哈医大二院、复旦中山、广东省人民医院等），这些医院可能仍维持一定处方习惯，但无法确认当前活跃覆盖数量。

### 科室渗透与处方结构

- 心脏外科手术围术期用药场景高度标准化，处方决策链：心外科主刀医生 → 麻醉科执行 → ICU续用
- 用药周期：术中单次给药 + 术后连续3日给药，单次手术用药量约6~9mg/kg（60kg患者约360~540mg，折合18~27支/手术）
- 科室渗透相对单一：几乎完全依赖心外科手术量，无慢性病续方场景

### 核心KOL（关键意见领袖）

P1-004记录了产品上市时参与评审的专家团队，包括中国工程院朱晓东院士及12家顶级心外科中心的数十位专家。这些KOL关系属于产品临床证据建立的学术资产，具有可迁移性。

- **KOL合作深度**：四期临床试验在12家大型中心完成近千例试验[P1-004]，建立了较深的学术关系
- **学术活动频次**：未知，但珍奥作为保健品企业，其处方药学术推广投入预计有限
- **KOL关系归属**：更偏产品本身（基于临床证据），而非推广方品牌

### 经销商与配送

- 珍奥主营保健品（直销/会销模式），处方药流通渠道建设可能严重不足
- 药源网等招商平台显示产品以代理/招商模式推广[G2-004]，非自建专业配送网络
- 具体一级经销商名录、配送商名录、回款习惯均未知，需阶段B获取

### 活跃处方医生数量

**数据缺失**。无法确认近6个月有稳定处方记录的医生数量。

### 核心省份/城市贡献度

基于手术量分布推断[G2-001]：
- 北京（阜外+安贞为核心）预计贡献最大
- 山东、河南手术量大省
- 上海（中山医院+瑞金等）、广东（广东省人民医院）
- 辽宁（沈阳军区总院/北部战区总医院）为珍奥大本营所在

## 2.4 品牌认知迁移风险

### 品牌认知归属判断

```
品牌资产主要归属：产品本身
迁移后品牌断层风险：低
原推广方竞品干扰风险：低
需额外品牌重建投入：中等
```

**分析依据**：

1. **医生认知基于产品而非推广方**：注射用心肌肽为独家品种、1类新药，医生对"心肌肽"的认知来自其临床证据（四期临床12家中心近千例），而非"珍奥"品牌。珍奥在处方药领域的品牌认知度极低。

2. **无品牌绑定风险**：珍奥在心外科领域的品牌影响力有限，其在公众领域的品牌主要关联保健品。医生不太可能因"珍奥离场"而停止处方。

3. **学术遗产可迁移**：四期临床数据、KOL关系、发表的学术论文均为产品资产，不依赖珍奥持续投入。

4. **商品名"开地米丁"的认知度**：不确定医生是按通用名"心肌肽"还是商品名"开地米丁"处方。如以通用名为主，迁移更无障碍。

### 品牌重建投入评估

- **需中等投入重建学术推广体系**：珍奥可能长期缺乏系统性学术推广，产品临床价值未被充分传递
- **CMS的学术推广能力恰为所长**：深康在心血管领域的学术推广网络和KOL资源可直接复用
- **品牌重建方向**：从"辅助用药"定位升级为"围术期心肌保护标准治疗组成部分"

## 2.6 渠道迁移比例预估

| 情景 | 渠道迁移比例 | 逻辑说明 |
|------|------------|---------|
| 乐观（30%） | 现有覆盖医院中约30%可在接管后6个月内恢复稳定供货和处方 | 假设珍奥在核心心外科中心仍有一定覆盖基础，且医生对产品认可度高 |
| 基准（15%） | 仅在少数有临床历史的核心中心（阜外、安贞等参与四期临床的医院）可快速恢复 | 考虑到珍奥可能已多年缺乏有效推广，大部分覆盖已流失 |
| 保守（5%） | 几乎全部需从头开发，仅极少数医院维持采购 | 考虑到产品非医保、非基药，停供后可能已被替代 |

**关键前提**：渠道迁移比例低并不等于项目失败。A-5的核心逻辑是"存量基本盘虽小，但通过专业推广重建渠道后，增量空间远大于存量"。

### 销售团队随行比例预估

**0%~10%**（估计0~2人可随行）。珍奥几乎无专业处方药推广团队，接管后需完全自建。

### 流向数据真实性评级

**无法评估**。阶段A无法获取流向数据，需阶段B通过珍奥或经销商获取后进行核查。

### 基准渠道迁移比例判断

基准迁移比例**15%**，远低于A-5门槛要求的60%。但鉴于本项目的特殊性质（原推广方推广能力不足导致渠道未充分开发），建议不以此单一指标否决项目，而应在阶段B重点验证"现有存量基本盘到底有多大"。

## 2.7 渠道重建策略初判

鉴于渠道资产几乎不可迁移，接管后的核心工作不是"迁移"而是"重建"：

1. **团队建设**：需自建约30~50人的心外科专科推广团队（覆盖全国核心心外科中心约150~200家）
2. **学术重启**：以四期临床数据为基础，重新启动学术推广，重点在头部心外科中心
3. **准入攻坚**：推进重点省份挂网和医院进院工作（非医保品种需逐院推进）
4. **KOL激活**：重新建立与12家临床试验中心及朱晓东院士等核心KOL的学术联系
5. **配送网络**：依托CMS现有配送网络，无需依赖珍奥原有经销商
