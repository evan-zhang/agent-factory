## 1. 交易结构分析 [阶段A]

### 1.1 推荐交易结构

**推广权/代理权引进**（非资产收购、非股权投资）

**理由**：
- 注射用心肌肽为珍奥唯一处方药品种，珍奥以保健品为主业，不太可能出售整个公司或药品资产
- 推广权引进模式风险可控、投入有限、退出灵活
- 珍奥保留生产权和MAH（上市许可持有人）身份，双方权责清晰

### 1.2 交易结构要素

| 交易要素 | 推荐方案 | 备选方案 |
|---------|---------|---------|
| 合作模式 | 独家推广权引进 | 非独家推广权+优先续约权 |
| 地域范围 | 中国大陆（不含港澳台） | 同左 |
| 合作年限 | 首期5年+自动续约5年 | 3+3年 |
| 供货方式 | 珍奥生产→CMS指定配送商 | 珍奥直接配送至终端 |
| 定价权 | CMS自主定价（挂网限价内） | 双方协商定价 |
| 知识产权 | 珍奥保留，CMS获得推广使用权 | 同左 |

### 1.3 TS（条款清单）核心条款建议

**关键条款**：

1. **独家推广权**：CMS获得注射用心肌肽在中国大陆的独家推广权
2. **供货价**：基于珍奥COGS（销售商品成本）+合理利润确定基准供货价，建议目标供货价为终端价的30%~40%
3. **销售分成**：可替代方案——按供货价差结算，CMS利润空间≥50%
4. **最低采购量/保底**：分年度递增，首年800~1,200万元，逐年递增15%~20%
5. **里程碑**：可选，如年度销售额达到X万元触发额外条款
6. **终止条款**：
   - CMS单方终止权：产品发生重大质量安全问题、被纳入重点监控目录导致大面积停用、政策变化导致无法继续推广
   - 珍奥单方终止权：CMS连续2年未达最低采购量的80%
7. **知识产权**：珍奥保留全部IP，CMS获得推广使用授权
8. **竞业限制**：珍奥不得向第三方授予中国大陆推广权（独家保障）

## 4. 知识产权归属与排他条款 [阶段A]

### 4.1 知识产权现状

核心化合物专利"心肌肽素及其用途"（CN1552439A/CN1255183C）已于2023年6月到期[G1-001]。国际专利（EP1661907B1、US7427663B2）同样已到期。

**知识产权风险**：
- 专利已过期，无排他性保护
- 独家优势来自"独家品种"（无仿制药申报），而非专利壁垒
- 若仿制药获批，独家推广权的价值将大幅下降

### 4.2 推广权排他条款

**建议**：
- 独家推广权（珍奥不得自行推广或授权第三方推广）
- 排他地域：中国大陆
- 排他期限与合作期限一致
- 违约赔偿：珍奥违约需赔偿CMS已投入的全部推广费用+预期利润损失

## 8. 最终可行性结论 [阶段A]

### 8.1 综合评估

**交易可行性**：**可执行，但回报规模有限**

**核心理由**：
1. 交易结构简单（推广权引进），法律和执行风险可控
2. 珍奥谈判地位有利，CMS可获得较优合作条件
3. 深康心外科渠道高度复用，边际推广成本较低
4. 最大损失敞口可控（~3,000~5,000万元）

**核心矛盾**：
1. **PeakSales门槛未达标**：基准1.2~1.8亿元远低于3亿元中适应症门槛
2. **渠道资产几乎为零**：需全面自建，但深康渠道协同可部分弥补
3. **非医保品种天花板**：DRG/DIP改革持续压缩非医保辅助用药空间

### 8.2 投委会建议

**建议结论**：**条件推进（Conditional Go）**

**条件清单**：
1. 深康确认心外科渠道协同评级为"强协同"（目标科室/医院重叠度>70%）
2. 珍奥确认合作意愿，且推广权合作条件在CMS可接受范围内
3. 阶段B流向数据验证当前年销售额≥4,000万元
4. PeakSales乐观情景（2.0~2.5亿元）具备可达路径

**如条件不满足**：建议No-Go，归档备查。

### 8.3 推荐行动

1. **立即行动**：与珍奥建立联系，签署NDA（保密协议），确认合作意愿
2. **短期行动**：获取珍奥内部数据（销售、流向、GMP记录），启动阶段B尽调
3. **条件推进**：如阶段B数据验证上述条件，启动正式合作谈判
