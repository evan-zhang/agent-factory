# A-2 国内合作·商业化授权评估（Domestic Collaboration & Commercialization License-in）

---

## 模块一：技能元信息

| 字段 | 内容 |
|------|------|
| **技能编号** | A-2 |
| **技能名称** | 国内合作·商业化授权评估 |
| **版本** | v1.5（体系校准版） |
| **定位** | A-2是国内合作主模板，用于评估中国大陆合作方持有的国内资产，在产品上市前或商业化权利尚未由CMS承接前，是否值得通过商业化授权、共同开发、区域独家、联合推广等方式合作。 |
| **适用对象** | 国内Biotech/药企/科研转化平台持有的国内资产；阶段可为IIa/PoC后、IIb/EOP2前、EOP2后、III期、NDA已提交/即将提交、获批前上市准备期。若仅I期或PoC前，原则上先走D-1临床阶段分层评估，交易口径按早期期权/rNPV处理；如为CMS自研或股权投资，分别联动A-3或E-1。 |
| **触发条件** | 用户提及「国内合作」「国内品种引进」「Biotech合作」「商业化授权」「国内成熟品种」「国内管线引进」「共同开发」「区域独家」「商业化+股权」时触发。 |
| **与A-1边界** | A-1用于海外产品进入中国；A-2用于国内合作方/国内资产。 |
| **与A-5边界** | A-5用于中国已上市、有NMPA批文和销售历史的推广权/代理权/批文接管；A-2不以“现推广方接管”和“历史流向迁移”为底层逻辑。 |
| **与A-3边界** | A-3用于CMS作为Sponsor或全球IP持有人的自主研发/委托研发项目；A-2用于IP或核心资产仍归外部国内合作方、CMS作为被授权方或合作商业化方。 |
| **与E-1边界** | 交易结构含股权条款且合作方为国内未上市Biotech时，A-2作为产品/商业化主评估，E-1作为股权投资叠加评估；纯股权投资不单独触发A-2。 |
| **适用业务主体** | 深康（心肾代谢/消化肝病/CNS与精神）、德镁（皮肤健康药品线）、维盛（眼科）；如涉及已上市推广权接管，转A-5。 |
| **商机接入路径** | D-0路由 → A-2；A-0扫描A级分拨 → A-2；A-7横向筛选A级（国内合作类）→ A-2；国内Biotech主动接触 → A-2。 |

## 模块三：评估阶段映射（三阶段模型）

```
阶段A（公开信息 + 初步材料）：One-pager + Gate 1 + Gate 2快评
  核心问题：资产权属清楚吗？注册/临床前提成立吗？合作结构值得谈吗？
  输出物：A-2国内合作立项建议书 + D-3 Gate 0/立项一页纸输入

阶段B（NDA后双轨并行）：Gate 3 + Gate 4 + Gate 5深化
  技术线：临床证据、注册路径、IP/FTO、供应链与上市准备
  商务线：权益范围、独家性、首付款/里程碑/分成、是否含股权
  输出物：A-2阶段A+B综合评估报告 + 红旗台账

阶段C（投委会终审）：Gate 6 + D-3 Final
  核心问题：交易条款、财务回报、止损条款、承接主体是否可执行？
  输出物：D-3 Final投委会包；如含股权，同步引用E-1结论
```

## 模块四：评估章节结构（本地Gate + One-pager，映射Gate 0~5 + Final）

### One-pager：国内合作终局先立

**A-2专项终局六问**：
1. CMS最终想获得什么权益：商业化权、共同开发权、区域独家、适应症权益，还是股权+商业化双引擎？
2. 标的资产的注册和临床关键前提是否足以支持立项，而不是只支持观察？
3. CMS的现有业务主体能否承接该TA、科室、渠道和上市节奏？
4. 若产品获批延迟12个月，财务回报和交易条款是否仍可接受？
5. 若交易结构含股权，产品获批能否支撑合作方估值和IPO叙事？
6. 哪些前提不成立时必须停止谈判？

### Gate 1：前提门（资产权属·注册路径·合作可能性）

**核心问题**：
- 合作方是否拥有可授权的资产权利，且授权链完整、无实质争议？
- 产品在中国大陆是否存在清晰注册路径和可验证开发计划？
- 若产品正在国内临床阶段，Sponsor/IND申请人、CDE受理或60日默示许可、临床试验登记、伦理批件、GCP机构/PI、HGR、SAE/SUSAR/DSUR和原始数据访问权是否可核验？
- 是否存在突破性治疗、附条件批准、优先审评、真实世界研究/RWE、临床急需进口、海南乐城、大湾区港澳药械通或北京天竺等可用于注册提速、早期可及或证据补强的路径？
- 临床数据、申报资料、合规记录是否存在不可接受缺陷？
- 是否命中CP-1/CP-2/CP-3合作可能性不足且无例外条件？

**国内临床阶段专项核查（强制）**：

| 国内阶段 | A-2评估重心 | 必须补证 | 交易口径 |
|----------|-------------|----------|----------|
| Pre-IND/IND准备 | 非临床、CMC和IND-enabling是否支撑申报 | GLP毒理、CMC、拟IND资料、CDE咨询计划、HGR判断 | 仅可作为早期期权或共同开发，不按商业化授权成熟资产定价 |
| IND受理/60日默示许可/I期 | 是否真实获准开展人体试验，安全窗和剂量探索是否成立 | CDE受理/默示许可、登记公示、伦理、I期方案、BA/PK/PD、SAE/SUSAR | 付款绑定I期安全性、PK和剂量结果；必须取得数据审计权 |
| IIa/PoC | PoC是否与目标适应症、人群、终点和商业主张一致 | 方案/SAP、效应量、剂量-反应、BA/PK/PD、原始数据、中心质量 | 仅PoC可解释时进入常规授权谈判；否则Watch或条件推进 |
| IIb/EOP2前 | 是否足以进入EOP2或关键III期设计 | IIb数据、拟III期方案、CDE沟通问题清单、CMC Phase III准备 | EOP2为硬止损节点，首付款和开发费用必须后置 |
| EOP2后/III期准备 | CDE是否支持关键III期设计，执行资源是否可承接 | EOP2/CDE纪要、III期方案、统计假设、中心计划、商业批准备 | 可提高PoS但不得写成疗效已确证；绑定III期启动/读出里程碑 |
| III期进行中/NDA前 | 执行质量、数据完整性、标签和上市准备是否可关闭 | 入组/事件、DMC/IDMC、偏离、稽查、数据库锁定、CSR/ISS/ISE、发补问题 | 估值转向读出、审评、标签和CMC/检查风险；保留失败/延迟压力情景 |

> 国内临床阶段项目必须同步触发D-1《国内临床项目注册与执行核查卡》。若合作方不能提供CDE、登记、伦理、GCP执行、HGR和原始数据可审计证据，不得仅凭"国内II/III期"进入基准财务模型。

### Gate 2：定调门（合作结构·权益边界·承接主体）

**核心问题**：
- 合作结构应定为商业化授权、共同开发、区域独家、联合推广，还是股权+商业化双引擎？
- 权益范围是否足够支撑CMS投入：区域、适应症、渠道、期限、独家性、再授权？
- 承接主体是否合法且唯一：深康/德镁/维盛按TA归口，院外业务只能作为叠加层。
- 是否触发E-1：国内未上市Biotech + 股权条款 + 产品获批带来估值/IPO溢价逻辑。

**合法路径判定**：

| 情形 | 主评估 | 叠加评估 | 备注 |
|------|--------|----------|------|
| 国内未上市/上市前成熟品种商业化授权 | A-2 | 可叠加D-2/D-3 | A-2主场 |
| 国内未上市Biotech + 商业化授权 + 股权条款 | A-2 | E-1 | 双引擎主路径 |
| 中国已上市产品推广权/代理权接管 | A-5 | 可叠加A-6 | 不走A-2 |
| CMS自主持有全球IP/委托研发 | A-3 | 不触发E-1 | CMS为Sponsor/IP Holder |

```
权益范围：[区域/适应症/渠道/期限/排他性]
承接主体：[深康/德镁/维盛]；院外叠加：[是/否]
是否触发E-1：[是/否，理由]
主要谈判红线：[Top3]

  5. 竞品或同类交易结构参考
```

### Gate 3：证据门（临床价值·差异化·医学可解释性）

**核心问题**：
- 关键临床证据是否足以支持目标适应症、目标患者和商业化主张？
- 与SOC（标准治疗方案）或主要竞品相比，疗效、安全性、便利性、依从性是否有差异化？
- 国内患者、指南、真实世界诊疗路径是否支持该定位？
- 证据缺口是否可通过阶段B补证，而不是根本性失败？

**临床阶段差异化评估（强制）**：

| 当前阶段 | A-2处理口径 | 必须补齐的专业判断 | 结论限制 |
|----------|-------------|--------------------|----------|
| I期 / PoC前 | 原则上不按常规商业化授权成熟资产推进；仅可作为早期期权、共同开发或股权联动线索 | 安全窗、BA/PK/PD、转化医学、同机制临床验证、下一PoC设计、首付款和止损结构 | 不得给出确定性峰值销售和承接主体资源承诺 |
| IIa / PoC后 | 可进入A-2 Gate 3，但必须证明PoC与目标适应症、人群、终点和商业主张一致 | PoC效应量、剂量-反应、BA/暴露-效应、亚组一致性、SOC对照、外部可重复性、中国患者适配 | 仅可条件推进；交易付款须绑定IIb/EOP2或下一关键读出 |
| IIb / EOP2前 | 可作为重点授权评估，但EOP2是关键止损节点 | 拟III期方案、主要终点、统计假设、样本量、对照、BA/PK/PD、CDE沟通问题清单、CMC Phase III准备 | 财务模型必须分列EOP2前/后PoS和rNPV |
| EOP2后 / 关键III期准备 | 可提高评估置信度，但需核实监管纪要和III期设计是否真正关闭 | EOP2/CDE沟通纪要、III期方案、统计计划、CMC准备、补充研究要求 | 不得写成"已确证"；仍需保留III期失败和执行延迟压力情景 |
| III期进行中 | 重点从科学假设转向执行质量、读出概率、数据完整性和上市准备 | 入组进度、事件数、方案偏离、DMC/IDMC、GCP稽查、商业批/放大准备 | 未读出前不得取消临床风险折现 |
| NDA已提交/获批前 | 重点核查审评问题、标签范围、CMC/检查和上市准备 | CSR/ISS/ISE、标签草案、CDE发补、检查状态、药物警戒、上市供应 | 标签收窄、发补、检查失败须进入压力情景 |

**证据分级**：

| 等级 | 定义 | 处理 |
|------|------|------|
| A | 至少1项高质量关键性临床研究或NDA级数据，结论与目标定位一致 | 可进入Gate 4 |
| B | 有IIa/IIb/III期积极信号，但PoC、EOP2、关键终点、样本量、统计计划或中国适配仍需补证 | 条件通过 |
| C | 仅探索性数据、单臂数据、I期早期信号或合作方内部口径，缺乏外部验证 | Watch或补证 |
| D | 主要终点失败、严重安全性缺陷或数据完整性不可接受 | 停止 |

## 模块六：一票否决清单

以下任一条触发，A-2应停止或降级Watch：

1. **资产权属不清**：合作方无法证明可授权权利，或授权链存在实质争议。
2. **注册路径不可行**：NMPA/CDE路径不成立，或关键补充研究无法完成。
3. **临床证据根本失败**：主要终点失败且无可接受替代证据；或安全性风险不可控。
4. **数据完整性缺陷**：临床、CMC或申报资料存在造假、重大稽查缺陷或不可补救缺口。
5. **IP/FTO障碍不可绕过**：核心专利、数据保护、第三方权益阻断上市或商业化。
6. **合作可能性明显不足**：命中CP-1/CP-2/CP-3且无任何例外条件；处理为Watch或关闭。
7. **财务压力情景不可接受**：按对应资产类型测算，压力情景下回报显著低于体系底线且无条款修正空间。
8. **独家性/权益范围不足**：权益范围不足以支撑CMS投入，且无法通过谈判修正。
9. **承接主体不匹配**：深康/德镁/维盛均无清晰TA承接，且院外/国际叠加不能解决根本冲突。
10. **股权叠加逻辑失败**：含股权交易中，E-1显示估值、退出或治理风险不可接受，且交易无法拆分为纯商业化授权。

## 模块八：串接接口

### 上游接口
- D-0路由（国内合作/商业化授权）→ A-2
- A-0 A级分拨（国内合作类）→ A-2
- A-7 A级（国内合作分类）→ A-2
- 合作方主动接触或BD团队轻触探意愿成功 → A-2

### 下游接口

```
A-2评估完成
  ├─ 商业化授权可行 → D-3 Final投委会包
  ├─ 交易结构含股权 + 国内未上市Biotech → E-1独立叠加
  ├─ 产品已在中国获批并转为推广权/代理权接管 → A-5
  ├─ 院内基础形成后有院外延伸价值 → A-6（满足上市且院内稳定条件后）
  ├─ 涉及国际市场权益 → C-1/C-2/C-3独立叠加
  └─ 任一Gate否决 → Watch或关闭档案
```

### 互斥声明
- A-2与A-5不得对同一“已上市推广权接管”场景并行作为主模板。
- A-2与A-3不得对同一“CMS自主持有全球IP/Sponsor”场景并行作为主模板。
- E-1不得替代A-2的产品和商业化评估；A-2也不得替代E-1的股权估值和治理尽调。

## 模块九：输出规范

- **主报告格式**：HTML（深蓝 `#1a3a5c`，白底，无灰色字，无元描述）。
