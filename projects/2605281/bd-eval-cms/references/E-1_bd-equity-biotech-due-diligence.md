## 模块一：技能元信息

| 字段 | 内容 |
|------|------|
| **技能编号** | E-1 |
| **技能名称** | 国内Biotech股权投资+商业化授权尽调 |
| **版本** | v1.1 |
| **继承来源** | 新建（融合 `pharma-bd-due-diligence` + PE/VC股权评估框架） |
| **适用业务主体** | 集团战略层/投前高层 |
| **适用地域** | 仅限中国大陆；**不适用康联达及任何海外Biotech** |
| **触发条件** | 用户提及"Biotech股权投资"、"双引擎合作"、"商业化授权+股权"、"战略投资Biotech"、"CMS商业化授权"、"Biotech 投前合作"、"内资Biotech引进"、"股权+licensing结合"；即使用户说"我们想投这家Biotech顺便拿商业化权"也应触发。**典型触发项目**：宁丹新药（Y-3）、麦济（MG-k10）、新元素（ABP-671）、景泽（JZB0-05）——均为国内未上市Biotech+A-2+E-1双引擎结构 |
| **适用对象核心条件** | **目标Biotech的核心产品在中国尚未获批上市**。商业逻辑：产品获批→公司估值跳升→IPO→CMS持股最大溢价退出，同步获得商业化权益（双引擎）。若产品已在中国上市，对手方为成熟大型药企，不适用本技能 |
| **排除条件** | 纯股权投资（无商业化授权）→ 转集团财务投资流程；纯商业化授权（无股权）→ 转A群单独评估；海外Biotech → 转康联达C群；**A-5已上市产品推广权接管场景（对手方为已有稳定收入的大型药企，不是寻求IPO溢价的pre-IPO Biotech）→ 不触发E-1；A-3自主研发（CMS为Sponsor/IP Holder，委托研发属服务采购，无Biotech股权结构）→ 不触发E-1** |
| **商机接入路径** | 集团战略部/投前高层 → E-1三层评估 → 投决委员会终审；最常见路径：A-2（国内Biotech商业化授权谈判中发现股权跟投机会）→ 触发E-1并行 |
| **核心模式** | **目标Biotech产品尚未在中国获批**，Biotech负责研发推进（以产品获批驱动公司IPO估值），CMS（康哲医药集团商业化平台）负责商业化准备+股权投资（等待IPO溢价退出） |
| **输出格式** | 麦肯锡风格HTML双引擎评估报告，含股权估值模型、商业化价值测算、协同价值矩阵 |

## 模块四：评估章节结构（Gate结构）

### ▶ One-pager：双引擎终局先立（锚定门）

**目标**：在进入任何数据核查前，先明确"股权投资+商业化授权"双引擎的终局图景。

必填要素：
1. **双引擎价值分解**：
   - 股权价值（Equity Value）：持股X%，目标估值Y亿，预期IPO时间Z年
   - 商业化价值（Commercial Value）：授权产品A的峰值收入贡献W亿
2. **CMS核心价值主张**：CMS为Biotech提供什么？（资金/渠道/KOL/注册经验）
3. **Biotech核心价值主张**：Biotech为CMS提供什么？（研发管线/专利/团队）
4. **协同假设**：双引擎是否相互强化，还是存在潜在利益冲突
5. **锚定决策**：通过 → Gate 1 ｜ 暂缓 → 补充信息 ｜ 否决 → 归档

---

### ▶ Gate 1：Biotech企业主体资质（前提门）

**核心问题**：这家Biotech是一个值得信任的合作伙伴吗？存在哪些致命性风险？

**评估维度**：

| 维度 | 评估内容 | 数据来源 | 权重 |
|------|----------|----------|------|
| 1-1 团队完整性 | 创始人/CSO（首席科学官）/CMO（首席医学官）/CFO（首席财务官）背景 | 访谈+公开资料 | 25% |
| 1-2 公司治理 | 股权结构清晰度；关联交易披露；VIE（可变利益实体）结构风险 | 工商资料+法律尽调 | 25% |
| 1-3 知识产权（IP）状态 | 核心专利归属；发明人争议；职务发明确权 | 专利尽调 | 25% |
| 1-4 合规历史 | 是否存在FDA/NMPA警告、GCP（药物临床试验质量管理规范）违规 | 公开数据库 | 25% |

**Gate 1 一票否决**：
- [ ] 创始人存在刑事案件记录或重大商业诚信问题
- [ ] 核心专利存在实质性权属争议
- [ ] VIE结构设计存在穿透风险，导致境内资产不可控
- [ ] 公司已进入清算或重整程序

---

### ▶ Gate 2：管线科学与临床价值（证据门）

**核心问题**：授权管线的科学基础是否扎实？临床成功概率是否合理？

> **注意**：Gate 2自动触发A群相关技能（`pharma-bd-due-diligence`）对授权标的进行专项管线评估，结果同步纳入E-1报告。

**评估维度**：

| 维度 | 评估内容 | 权重 |
|------|----------|------|
| 2-1 靶点验证度 | 同类靶点全球临床成功率；Biotech数据是否重复验证了作用机制 | 20% |
| 2-2 临床数据质量 | IND（新药临床试验申请）状态；Phase I/II数据完整性；样本量 | 30% |
| 2-3 差异化程度 | 相较已上市竞品/在研竞品的优效或非劣效假设 | 25% |
| 2-4 注册路径清晰度 | NMPA优先审评资格；快速通道；预计获批时间 | 25% |

**PoS（成功概率）矩阵**：

| 阶段 | 行业平均PoS | 本标的PoS估计 | 差异原因 |
|------|-----------|------------|---------|
| IND申报→Phase I | 85% | [填写] | |
| Phase I→Phase II | 65% | [填写] | |
| Phase II→Phase III | 40% | [填写] | |
| Phase III→NDA（新药上市申请） | 60% | [填写] | |
| NDA→批准 | 85% | [填写] | |

### ▶ Gate 3：股权结构与IPO路径（定调门）

**核心问题**：CMS股权投资的退出路径是否清晰？估值是否合理？

**股权估值方法**：

| 方法 | 适用场景 | 计算要点 |
|------|----------|---------|
| DCF（折现现金流）法 | 有商业化产品 | 折现率15-20%；管线PoS调整 |
| 可比交易法 | 近期同类Biotech融资参考 | EV（企业价值）/管线数量；EV/峰值销售 |
| Pipeline NPV（净现值）法 | 纯研发型Biotech | 各管线rNPV（风险调整后净现值）之和 |

**IPO路径评估**：

| 上市地 | 挂牌条件 | 预计时间 | CMS持股锁定期 |
|--------|---------|---------|-------------|
| 科创板 | 满足创新属性+财务门槛 | [填写] | 1-3年 |
| 港交所18A | 无收入要求，满足研发门槛 | [填写] | 6-12个月 |
| 纳斯达克 | 需VIE/红筹架构 | [填写] | 180天 |

### ▶ Gate 4：商业化协同价值（支付门）

**核心问题**：CMS商业化平台能为Biotech创造哪些独特价值？协同效益能量化吗？

**CMS协同价值清单**：

| 协同项 | 具体内容 | 可量化价值（万元） |
|--------|----------|-----------------|
| 销售渠道赋能 | 覆盖医院数量、KOL关系、地面队伍 | [测算] |
| 注册加速 | NMPA申报经验；与审评机构沟通能力 | [测算] |
| 市场准入 | 医保谈判资源；DRG（按诊断相关分组）/DIP（按病种分值付费）应对 | [测算] |
| 学术推广 | KOL培育网络；CME（继续医学教育）平台 | [测算] |
| 品牌背书 | CMS上市公司品牌对Biotech融资/招聘加成 | [定性] |

**利益对齐评估**：
- CMS的商业化成功是否与Biotech的里程碑/IPO估值正相关？
- 存在哪些潜在利益冲突（如CMS同时代理竞品）？

### ▶ Gate 5：投资条款与止损安排（可做门）

**核心问题**：投资条款是否保护CMS的核心权益？止损机制是否完善？

**核心条款清单**：

| 条款类型 | 评估要点 | 风险等级 |
|----------|----------|---------|
| 估值及稀释保护 | 反稀释条款（Anti-dilution）类型：完全棘轮/加权平均 | 高 |
| 优先清算权 | 1x/2x非参与优先/参与优先清算 | 高 |
| 商业化授权排他性 | 地域/适应症/期限排他范围 | 高 |
| 里程碑付款触发条件 | IND/Phase II/NDA/上市的对赌里程碑 | 中 |
| 对赌协议 | 销售额对赌；回购条款；触发条件合理性 | 高 |
| 董事会席位 | 是否获得观察员或投票权席位 | 中 |
| 信息权 | 财务报告频率；现场尽调权利 | 低 |
| 商业化协议关联条款 | 授权协议与股权协议的相互触发条款 | 高 |

## 模块五：财务内控对照表

| 财务指标 | 中文全称 | E-1双引擎门槛 | 计算说明 |
|----------|----------|-------------|---------|
| IRR（股权侧） | 内部收益率 | ≥25%（股权退出） | 持股期间IRR，含IPO退出情景 |

| CAPEX | 资本性支出 | 商业化CAPEX≤首年收入20% | CMS商业化投入约束 |

## 模块六：一票否决清单

### 企业层否决项（E级）
- [ ] **E-01** 创始人/核心管理层存在刑事记录或经证实的重大欺诈行为
- [ ] **E-02** 核心IP（知识产权）存在实质性所有权争议，无法在6个月内解决
- [ ] **E-03** 公司已进入破产程序或存在被强制执行的重大判决

### 管线层否决项（P级）
- [ ] **P-01** 授权管线靶点/机制已被临床证伪（同靶点Phase III全部失败）
- [ ] **P-02** 管线存在已知的严重安全信号（SAE），未向监管机构披露

### 结构层否决项（S级）
- [ ] **S-01** 双引擎结构中存在不可解决的核心利益冲突（CMS商业化与Biotech另有排他安排）
- [ ] **S-02** 投资条款不含任何反稀释保护，且无合理补偿
- [ ] **S-03** 商业化授权地域或适应症被竞争对手抢先锁定

### 财务层否决项（F级）
- [ ] **F-01** 股权IRR（内部收益率）基准情景<15%
- [ ] **F-02** 商业化侧rNPV（风险调整后净现值）<0（即使乐观情景也为负）

## 模块九：输出规范

### 9.1 报告结构

```
封面页
├── 执行摘要（双引擎价值分解，500字）
├── 前置信息（M-01~M-10汇总）
├── 双引擎One-pager终局画像
├── 三层评估结果
│   ├── 企业层：PE/VC视角评分卡
│   ├── 管线层：A群评估结果（嵌入）
│   └── 协同层：双引擎价值矩阵
├── Gate 1-6逐门评估结果
├── 财务模型
│   ├── 股权侧：估值区间/IRR/MOIC/IPO退出情景
│   ├── 商业化侧：5年P&L/IRR/rNPV
│   └── 双引擎综合回报（加权合并）
├── 核心条款评估（Term Sheet要点）
├── 一票否决核查表
├── 三级验收标准
├── 双引擎整合100天计划
└── 附录（管线详情/竞品对比/专利地图）
```

### 9.2 格式要求
- **语言**：简体中文，财务英文缩写须标注中文全称
- **格式**：麦肯锡风格HTML，含双引擎价值分解图、PoS漏斗图、IRR敏感性分析
- **字数**：≥7000字（因评估复杂度最高）
- **双重视角**：报告须同时呈现"CMS作为投资人"和"CMS作为商业化伙伴"两个视角

### 9.3 三级验收标准

| 级别 | 必达项 | 参考项 | 否决项 |
