# 检索词模板（v0.10.0）

> 本文档是 `scripts/search/lib/keyword_templates.json` 的说明。

## 用法

```bash
bash scripts/search/keyword_mapper.sh --skill D-1 '{"target":"Acme Pharma"}'
```

返回 3–4 个检索词，支持占位符替换：
- `{品种}` / `{target}`：具体品种名
- `{公司}`：合作方
- `{靶点}`：药物靶点
- `{适应症}`：主要适应症
- `{参照药}`：参比制剂

## 覆盖范围

| 群 | 技能 | 词数 |
|----|------|------|
| A群（中国市场药品） | A-1, A-2, A-3, A-4, A-5, A-6, A-7, A-8 | 8 |
| B群（消费属性） | B-1, B-2, B-3 | 3 |
| C群（国际市场） | C-1, C-2, C-3 | 3 |
| D群（基座/工具） | D-1, D-2, D-3 | 3 |
| **合计** | | **17 / 22** |

**未覆盖 5 个**：D-0（路由）、A-0（机会情报）、E-1（股权投资）—— 这三个是上游路由/工具，不需要专项检索词。后续按需补。

## 维护规则

- 检索词用英文为主、英文+中文并列
- 优先匹配权威源关键词（如"NMPA"、"FDA"、"ClinicalTrials"、"IQVIA"）
- 避免商业敏感词（"独家"、"首仿"等）
- 每技能 3–4 个词为佳，不超过 5 个

## 不在表内的处理

未列在 templates 里的技能（如 A-0、D-0）走 `default` 分支：sub-agent 自由选择 3–4 个检索词（建议用 `{品种} {公司} 投前评估` 等通用词）
