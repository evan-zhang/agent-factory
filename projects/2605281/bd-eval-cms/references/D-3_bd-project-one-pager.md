## 模块3：评估阶段映射

### D-3 在三阶段模型中的全程贯穿

```
阶段A：
  ├─ Gate 0 之前：D-3 v0（商机快扫一页纸）—— 1页
  └─ Gate 0 完成：D-3 v1（立项申请版一页纸）—— 2页

NDA 签署节点：
  └─ D-3 v1 加盖 NDA 保密标记，更新保密级别字段

阶段B：
  ├─ Gate 1/2 完成：D-3 v2（技术评估更新版）—— 4~6页
  ├─ Gate 3/4 完成：D-3 v3（商业+财务更新版）—— 6~8页
  └─ Gate 5/TS 阶段：D-3 v4（合同前版本）—— 8~10页

阶段C：
  └─ D-3 Final投委会：完整投委会摘要包—— 8~12页
```

| 版本 | 触发节点 | 主要新增内容 | 审阅层级 |
|------|---------|------------|---------|
| **v0** | 商机发现，D-0 路由完成 | 基本标识字段 + 路由结论 | 投前 负责人（内部） |
| **v1** | Gate 0 快扫完成 | 核心机会×3 + 核心风险×3 + 关键假设×3~5 | 业务主体 投前 总监 |
| **v2** | Gate 2 技术门完成 | 科学/技术/注册结论更新 + 技术风险评级 | 业务主体 CMO/CSO |
| **v3** | Gate 3+4 完成 | 市场/财务结论更新 + 财务模型关键指标 | 业务主体 CEO + CFO |
| **v4** | TS 谈判完成 | TS 关键条款摘要 + 谈判结果 + 供应链确认 | 集团 投前 委员会 |
| **Final** | D-3 Final投委会准备 | 各部门签署意见 + 三个决策选项 + 风险登记表 | 集团投委会 |

## 模块9：输出规范

### HTML 报告视觉规范

**整体风格：** 麦肯锡顾问幻灯片风格，清晰、数字驱动、无废话。

**色彩体系：**
- 主色：深蓝 `#1a3a5c`（页眉背景、章节标题栏、重要区块边框）
- 强调色：橙黄 `#e67e22`（关键数字、警示框、行动建议框）
- 正向色：深绿 `#1e8449`（达标✅标记、批准建议框）
- 负向色：深红 `#c0392b`（否决❌标记、一票否决项、高风险标记）
- 背景：白色 `#ffffff`；交替行：极浅蓝 `#f0f5fb`
- 禁用：任何灰色字体（`#777777` 及以下）
- 禁用：元描述段落（禁止"本报告旨在……"等套话）

**Gate 0 一页纸（v0/v1）排版：**
- A4 横版或竖版，1~2页
- 七个区块须在1~2页内完整呈现（不得分页溢出）
- 核心机会/风险用数字图标（①②③）标注，便于高管快速扫描
- 数字须加粗放大（字号+2px），与正文有视觉层级区分

**D-3 Final投委会包排版：**
- 每页独立打印可读
- 封面页须含项目照片/产品图（可为占位符）
- 各部门签署意见须在独立页面呈现，便于实体签署
- 财务模型须同时提供图表版（视觉）和数字表版（精确）
- 三个决策选项须在同一页面呈现，字号相同，不得有视觉引导倾向

**内容规范：**
- 所有财务缩写首次出现须标注中文全称：IRR（内部收益率）、NPV（净现值）、rNPV（风险调整净现值）、WACC（加权平均资本成本）、MOIC（投资回报倍数）、TS（条款清单）、MSC（最低销售承诺）、SAM（可服务市场）、TAM（总可及市场）、CAGR（复合年均增长率）、VBP（国家带量采购）、NRDL（国家医保目录）、KOL（关键意见领袖）、DTP（直达患者药房）、HTA（健康技术评估）、CMC（化学生产控制）、PAP（患者援助计划）
- 非人民币市场须双币标注（如：USD 50M / RMB 约3.6亿元，按报告日汇率折算，需注明汇率基准日）
- 所有数字须注明来源（内部估算/D-2报告/财务部门模型/第三方数据）
- 日期格式统一：YYYY-MM-DD
- 项目编号格式：[业务主体代码]-[年份]-[流水号]，如 SK-2025-042、DM-2025-008

**章节顺序规范（D-3 Final 版）：**
```
封面页
第1页  执行摘要（One-pager Final 版）
第2页  交易摘要（TS 关键条款）
第3页  财务模型关键指标
第4页  风险登记表（红旗风险台账 V2 摘要）
第5页  技术评估结论摘要
第6页  商业化评估结论摘要
第7页  各部门签署意见
第8页  三个决策选项
附录A  完整 D-1 深化报告链接/摘要
附录B  完整 D-2 市场全景报告链接/摘要
附录C  财务模型假设明细（Excel 附件参引）
附录D  一票否决项完整核查记录
```

> 附录须排列于全部正文章节（封面+第1~8页）之后，不得将附录插入正文内容之间。
