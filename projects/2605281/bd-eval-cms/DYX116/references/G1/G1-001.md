# [G1-001] CMS集团GLP-1相关管线资产梳理
- **搜索关键词**: 康哲药业 CMS GLP-1 管线 产品 糖尿病 肥胖
- **抓取时间**: 2026-05-31 14:07 CST
- **来源类型**: 多源交叉验证（公司公告+行业媒体）

## 关键发现

### CMS-D005（GLP-1R/GCGR双重激动剂）
- 自主研发创新药，2024年11月27日获NMPA临床试验批准 [B级-新浪财经]
- 靶点：GLP-1R和GCGR双重激动剂
- 适应症：肥胖/超重、代谢相关脂肪性肝炎、2型糖尿病
- 临床前研究显示良好减重效果和安全性
- 当前阶段：已获批临床（I期），尚未见II期进展公告

### CMS-D008（INHBE小核酸药物）
- 自主研发siRNA药物，2026年3月4日获NMPA临床试验批准 [B级-商业观察网]
- 靶点：靶向抑制INHBE基因
- 适应症：超重或肥胖
- 特点：减脂不减肌，机制不同于GLP-1类药物
- 官方表述：CMS-D008将与CMS-D005形成协同

### CMS其他代谢管线
- CMS-D002：GnRH受体拮抗剂
- CMS-D003：心肌肌球蛋白抑制剂
- CMS-D001：TYK2抑制剂
- Gelesis PLENITY：已获FDA批准的减重器械（BMI 25-40成人）

### 关键结论：利益冲突分析
1. CMS-D005为GLP-1R/GCGR双靶点，DYX116为GIPR/GLP-1R/GCGR三靶点——靶点有重叠但非完全相同
2. CMS-D005适应症（肥胖、脂肪肝、T2DM）与DYX116高度重叠
3. CMS已有自主GLP-1/GCGR管线在开发，引进DYX116存在明显的管线内耗风险
4. 但CMS-D005为双靶点（缺少GIPR），DYX116为三靶点，理论上疗效有差异——可作为差异化论证依据
5. CMS官方表述"CMS-D005+CMS-D008协同"暗示集团倾向于自研管线，对外部引进的意愿可能较低
