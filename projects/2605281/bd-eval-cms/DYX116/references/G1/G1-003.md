# [G1-003] DYX116专利状况与IP分析

- **搜索时间**: 2026-05-31 14:09 CST
- **来源类型**: 公司公告+行业分析

## 关键发现

### DYX116研发合作模式
- DYX116由德源药业与上海药明康德新药开发有限公司合作开发 [P1-004] [B级]
- 药明康德在德源年报中被多次提及为创新药项目合作方
- 德源2025年年报描述：南京德源推动与药明康德、中国药科大学、上海药物所等机构的创新药项目合作开发 [B级-年报PDF]

### 专利状况评估
- **公开信息中未检索到DYX116的化合物专利号**——这说明专利信息可能尚未公开或尚未授权 [C级-待验证]
- 作为1类创新药（化学合成多肽），DYX116的化合物专利通常在IND申报时已提交
- 德源药业拥有DYX116的注册申请权（NMPA批准通知书申请人列为"江苏德源药业股份有限公司"），说明IP归属明确为德源
- 药明康德作为CRO（合同研究组织）参与研发，IP归属通常按合同约定

### 潜在IP风险
1. **药明康德IP条款**：需确认药明康德在合作中的IP权益范围（是否有技术使用费/里程碑分成）
2. **化合物专利保护期**：作为2024年底获批IND的新药，化合物专利预计可保护至2044年前后（20年保护期）
3. **三靶点多肽序列专利**：全球三靶点领域存在礼来Retatrutide等先发者的专利壁垒，需评估DYX116是否涉及专利侵权
4. **药明康德制裁风险**：药明康德曾受美国生物安全法案影响，若CMS涉及国际市场需评估此风险

### 结论
- IP归属明确（德源药业），无已知专利纠纷
- 化合物专利号未在公开信息中查到，需阶段B获取
- 与药明康德的IP约定细节为关键待补信息
