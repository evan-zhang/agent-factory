# G6-003 创新药财务内控标准对照（§5.1）

## 来源
- CMS投前评估技能体系总规则（00号文件）第5节

## DYX116适用的财务门槛

### 项目分类
- 风险投资额：>3亿元（预计5-12亿元，含首付款+临床分摊+注册成本）
- 上市剩余年限：上市5年以上（预计2029-2031年上市，距今3-5年）
- 适用档次：>3亿元 + 上市3-5年 或 5年以上

### 财务门槛要求

| 指标 | 3-5年档 | 5年以上档 |
|------|---------|----------|
| IRR（内部收益率）| ≥22% | ≥25% |
| 回收期 | ≤5年 | ≤5年 |
| 5年倍数 | ≥2.7倍 | ≥3.1倍 |
| 10年倍数 | ≥7倍 | ≥9倍 |
| 毛利率 | ≥60% | ≥60% |
| 净利润率 | 10年均≥20% | 10年均≥20% |
| 盈亏平衡 | 上市后3年累计 | 上市后3年累计 |

### rNPV模型强制要求
- 临床阶段项目必须采用rNPV（风险调整后净现值），不允许直接使用NPV
- II期→III期成功率：约50-60%
- III期→NDA成功率：约60-70%
- NDA→获批成功率：约85%
- 综合II期→获批成功率：约25-35%
- 折现率：创新药项目12-15%
