# [G3-006] 界面新闻——Retatrutide减重天花板及GLP-1竞争格局
- **URL**: https://m.jiemian.com/article/14471488.html
- **抓取时间**: 2026-05-31 14:10 CST
- **来源类型**: 财经媒体

## 原文内容
礼来2026年5月21日公布Retatrutide III期TRIUMPH-1结果。

竞争格局要点：
- Retatrutide减重效果首次以非手术手段达到减重手术水平（20-35%体重下降）
- 替尔泊肽15mg减重20%出头，Retatrutide 12mg达28.3%
- Retatrutide将成为礼来继替尔泊肽、Orforglipron之后第三款减重产品
- 诺和诺德CagriSema（司美格鲁肽/长效Amylin）已递交上市申请
- 司美格鲁肽2026Q1合计销售额约82.77亿美元，同比下滑5%
- 减重司美格鲁肽片上市以来处方量超200万张
- GLP-1口服时代已开启

安全性数据：
- 不良事件与其他肠促胰素药物一致
- 因AE停药率随剂量升高：4mg 4.1%→9mg 6.9%→12mg 11.3%

## 关键数据点
- Retatrutide已确定2026年递交上市申请
- 司美格鲁肽增速放缓但仍是市场主导
- GLP-1竞争已进入口服+注射多维度竞争
