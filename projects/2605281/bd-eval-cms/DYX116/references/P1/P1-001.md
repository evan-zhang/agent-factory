# [P1-001] 德源药业1类新药注射用DYX116获得临床试验批准通知书公告
- **URL**: http://money.finance.sina.com.cn/corp/view/vCB_AllBulletinDetail.php?stockid=832735&id=10661423
- **抓取时间**: 2026-05-31 13:58 CST
- **来源类型**: 公司公告

## 原文内容
德源药业公告，公司收到国家药品监督管理局签发的《药物临床试验批准通知书》，注射用DYX116获准在2型糖尿病适应症中开展临床试验。
- 药物名称：注射用DYX116
- 剂型：粉针剂
- 申请事项：临床试验
- 申请人：江苏德源药业股份有限公司
- 注射用DYX116是一种化学合成的单分子GIPR/GLP-1R/GCGR三重激动剂多肽，对糖/脂代谢的调节功能具有协同作用，能降血糖、减体重、缓解脂肪肝。
- 与GLP-1/GIP双靶点相比，三重受体激动剂通过在体内调节糖脂代谢、抑制食欲的基础上，通过GCGR进一步增加能量消耗，有望在减重及脂肪肝领域展现出更优潜力。
- 目前国内外尚无同类产品获批上市。
- 公告日期：2024年12月20日

## 关键数据点
- 2024年12月获得2型糖尿病适应症临床试验批准
