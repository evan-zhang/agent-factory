# [P1-004] 德源药业2025年年度报告摘要
- **URL**: https://www.wabei.cn/Home/News/351931
- **抓取时间**: 2026-05-31 13:59 CST
- **来源类型**: 行业媒体

## 原文内容
德源药业2025年实现营业收入10.58亿元，同比增长21.80%；实现净利润2.37亿元，同比增长33.87%。
- 拉长时间周期，德源药业2021年2月上市，五年来营收、净利均保持增长。
- 公司专注于代谢紊乱慢性病治疗药物的研发、生产和销售。
- 2024年营收8.68亿元，归母净利润1.76亿元（业绩快报数据）。
- 2025年上半年研发投入6,079.31万元。
- 股票代码：920735（北交所），原代码832735。
- 公司DYX116与上海药明康德新药开发有限公司合作开发。
- 2026年Q1营收2.94亿元同比增15.32%，归母净利润5551.35万元同比增19.53%。

## 关键数据点
- 2025年营收10.58亿元（同比+21.80%），净利润2.37亿元（同比+33.87%）
