# G4-001 司美格鲁肽与替尔泊肽2025年末集体降价

**来源**: 财联社/新浪财经 2025-12-27
**URL**: https://finance.sina.com.cn/jjxw/2025-12-27/doc-inhefcys0206757.shtml

## 关键数据

- 诺和泰（降糖版司美格鲁肽）1.34mg/ml×1.5ml最低成交价：329元/支
- 诺和盈（减重版司美格鲁肽）1ml:1.34mg×1.5ml最低售价：388元/支
- 穆峰达（替尔泊肽）2.4ml:10mg最低成交价：约480元/支，首次跌破500元
- 四川省药械招采：诺和盈2.27mg/ml×3ml从1893.67元降至987.48元（降幅47.8%）
- 四川省药械招采：诺和盈3.2mg/ml×3ml从2463元降至1284.36元（降幅47.9%）
- 2026年1月1日起替尔泊肽纳入医保执行新价
- 预售价格：替尔泊肽2.4ml:10mg为450元、2.4ml:20mg为750元

## 背景分析

- 司美格鲁肽核心专利2026年3月到期，10家以上企业生物类似药上市申请获CDE受理
- 降价为"价格换市场"战略抢跑，非短期促销
- 预计未来1-2年减重药物主流价格维持200元以上
- 生物类似药定价预计为原研产品的70-80%

## 竞争格局

- 信达生物玛仕度肽2025年6月/9月分别获批减重和降糖适应症
- 恒瑞医药HRS9531即将获批
- 国内进入临床阶段司美格鲁肽生物类似药项目超20个
- 2026年为司美格鲁肽生物类似药集中获批高峰期
