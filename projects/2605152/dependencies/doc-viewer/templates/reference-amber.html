3.0 产品类型定义 — CG-0255 (Evategrel/依福格雷)

产品定义

CG-0255（Evategrel/依福格雷）是由上海柯君医药（CureGene Pharmaceutical）自主研发的全球首个硫醚水解型前药（thioether hydrolysis-based prodrug），靶向P2Y12受体，属于新一代抗血小板药物。

该品种的活性代谢产物为氯吡格雷活性代谢物H4，但通过完全绕过CYP450代谢通路（尤其是CYP2C19），经羧酸酯酶（CES）水解一步转化为活性形式，从根本上解决了氯吡格雷耐药性问题。

  • 美国：正在推进III期临床试验（据2025年8月报道）
  • 中国NMPA：2025年获口服给药IND；2026年4月获缺血性脑卒中临床试验批件

关键差异化特征

  1. 机制创新：全球首个基于硫醚水解的前药设计，绕过CYP2C19，克服氯吡格雷耐药
  2. 双剂型：口服+注射双线开发，IV剂型填补全球卒中领域空白
  3. 剂量效率：2mg剂量可达300mg氯吡格雷同等药效，效价提高约150倍
  4. 起效速度：口服和静脉给药后均能快速产生抗血小板效果
  5. 安全性：个体差异小、药物-药物相互作用风险低

3.1 S-1 靶点科学评估 — P2Y12受体

1. 靶点基础证据强度

1.1 致病因果关系的遗传学证据

P2Y12受体（P2RY12基因编码）与动脉血栓性疾病的因果关系具有极强的遗传学证据：

  • 功能丧失性突变（LoF）：P2RY12基因LoF突变导致出血性疾病（如Hermansky-Pudlak综合征样表型），反向验证了该靶点在血小板聚集中的核心角色
  • GWAS证据：全基因组关联研究反复证实P2RY12基因多态性与血小板反应性、心血管事件风险显著相关
  • CYP2C19-P2Y12通路：氯吡格雷药效基因组学已建立完整的CYP2C19 LoF → 活性代谢物减少 → P2Y12抑制不足 → 心血管事件增加的因果链（FDA黑框警告）
  • 人类孟德尔遗传病：P2RY12基因缺陷导致先天性出血障碍，直接证明靶点-疾病因果关系

评级：极强（5/5） — 靶点-疾病因果链完整，有人体遗传学正/反向验证

1.2 动物模型验证

  • P2Y12基因敲除小鼠：出血时间显著延长，ADP诱导的血小板聚集完全消失
  • 脑卒中模型：P2Y12抑制可减少动脉血栓形成，但不增加颅内出血风险（关键安全信号）
  • PCI/支架血栓模型：P2Y12抑制显著降低支架内血栓形成率

评级：充分（5/5） — 动物模型表型与人类疾病高度一致

1.3 靶点表达谱

  • P2Y12主要表达于血小板表面，在血管平滑肌细胞和中枢神经系统（小胶质细胞）也有低水平表达
  • 血小板高选择性表达确保抗血小板治疗的靶向性
  • 小胶质细胞表达的潜在影响：需关注长期用药的神经安全性信号

2. 靶点可成药性（Druggability）

2.1 蛋白结构与药物结合位点

  • P2Y12属于GPCR（G蛋白偶联受体）家族，Class A（视紫红质样）
  • 晶体结构已解析：PDB 4PXZ（2014年Nature发表），与人源P2Y12-拮抗剂复合物结构已公开
  • 药物结合位点明确：正构结合口袋位于7次跨膜螺旋束内，已有多类拮抗剂共晶结构

2.2 靶点类别与历史成药成功率

  • GPCR靶点：历史上最成功的药物靶点类别，已上市药物超过500个
  • P2Y12子类：已有4个药物获批上市，成药性已被反复验证
    • 氯吡格雷（Clopidogrel/Plavix，1997年获批）
    • 普拉格雷（Prasugrel/Effient，2009年获批）
    • 替格瑞洛（Ticagrelor/Brilinta，2011年获批）
    • 坎格瑞洛（Cangrelor/Kengrel，2015年获批）

评级：极高（5/5） — GPCR类别，4个上市药物验证成药性

3. 靶点专属性与脱靶风险

  • 关键脱靶风险：P2Y1（另一个血小板ADP受体），需确认CG-0255/H4对P2Y1的选择性比值
  • 主要安全性关注：出血风险（与P2Y12抑制程度正相关）
  • 呼吸困难：替格瑞洛特有不良反应（归因于腺苷摄取抑制），CG-0255无此风险
