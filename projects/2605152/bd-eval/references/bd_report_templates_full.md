# 模板一：海外已上市 · 中国未上市产品（5.1类 / 6类）

## 1.0 产品类型定义

**定义**：该产品已在美国、欧盟、日本等主要监管市场获批上市并积累商业化数据，但尚未在中国内地提交或获得NMPA注册批准，拟通过5.1类（境外已上市境内未上市化学药）或6类（改良型/生物制品等参考路径）申请进入中国市场。

**典型特征**：

- 拥有完整境外注册资料（FDA NDA / EMA MAA），可供桥接研究设计参考
- 核心化合物专利在境外已进入或即将进入专利悬崖期，需评估国内专利保护窗口
- NMPA注册周期通常24–36个月，先发优势窗口有限
- 临床数据种族差异桥接是注册科学的核心挑战（PK/PD/有效性/安全性）
- 无中国上市历史，渠道从零建立，需结合现有销售团队资源评估可行性
- 代表案例：RHOFADE（盐酸羟甲唑啉乳膏）、Venofer（蔗糖铁）引进评估

**框架组合**：主框架——终审模板v6.2（全量）｜辅助——BD投前尽调（内容补充）+ 麦肯锡（假设验证）

## 1.1 PC-1 科学机制与差异化

**负责部门**：产品中心 ｜ **终审模板对应**：第二章 科学与商业·产品科学维度

### 评估要点

1. **作用机制明确性**
   - 药物作用靶点/通路是否经基础研究充分验证？
   - 从靶点路径判断是否具有先进性和临床需求匹配度
   - 是否存在未知机制的不确定性？如有，风险等级如何？

2. **靶点与疾病相关性**
   - 靶点与适应症病理生理机制是否直接关联？
   - 是否有临床前/临床研究支持靶点-疾病因果关系？
   - 能否拓展适应症或联合用药？潜在适应症清单

3. **同靶点/同类药物进展**
   - 是否为FIC（First-in-Class）或BIC（Best-in-Class）？
   - 同类药临床进展是否提示优效性或安全性风险？
   - 竞品差异化对比矩阵（疗效/安全性/给药便利性/价格）

4. **临床证据强度**
   - 已完成临床阶段（I/II/III期）及各阶段核心数据
   - 有效性指标：治愈率、缓解率、主要终点达标情况
   - 安全性指标：不良反应率、SAE（严重不良事件）发生率
   - 与同类药头对头比较数据（如有）
   - 真实世界数据（RWD/RWE）支持情况

5. **临床前研究数据**
   - 药理药效学、毒理学、药代动力学核心结果
   - 剂型合理性、生物利用度、半衰期、代谢途径
   - 毒性反应类型及可逆性

### 评估结论模板

| 维度 | 评估结果 | 风险等级 | 备注 |
|---|---|---|---|
| 靶点机制 | FIC / BIC / Me-better / Me-too | 高/中/低 | |

## 1.2 PC-2 TPP与临床定位

**负责部门**：产品中心 ｜ **终审模板对应**：第六章 医学研究策略

### 评估要点

1. **目标产品档案（TPP）制定**
   - 目标适应症（主适应症 + 潜在扩展适应症）
   - 目标患者人群（年龄/性别/疾病分型/严重程度/合并症）
   - 给药方案（剂量/频次/疗程/给药途径）
   - 疗效目标（主要终点/次要终点/临床获益阈值）
   - 安全性目标（可接受的不良反应谱/禁忌症）

2. **临床痛点匹配度**
   - 当前标准治疗方案（SOC）的局限性是什么？
   - 本品解决了哪些具体的未满足需求？
   - 临床医生对现有治疗的不满意点（来源：KOL访谈/文献/指南）
   - 患者对现有治疗的不满意点（来源：患者调研/患者组织反馈）

3. **适应症边界**
   - 适应症范围是否过窄（影响市场规模）或过宽（影响注册可行性）？
   - 是否存在超适应症使用的潜力和风险？
   - 与竞品适应症重叠度分析

4. **指南/共识定位**
   - 当前国内外指南中的推荐等级
   - 本品上市后预期的指南定位（一线/二线/三线/辅助）
   - 需要哪些额外证据才能获得更高推荐等级？

## 1.3 PC-3 桥接科学策略

**负责部门**：产品中心 + 注册 ｜ **终审模板对应**：第五章 注册策略

### 评估要点

1. **种族敏感性评估**
   - 该药物是否属于种族敏感性药物？（参考ICH E5指导原则）
   - 已有数据中是否包含亚洲/中国人群亚组分析？
   - 代谢酶多态性（CYP2C19/CYP2D6等）对中国人群的影响评估
   - 体重/体表面积差异对剂量的影响

2. **PK/PD桥接设计**
   - 是否可通过PK桥接研究替代III期临床？
   - PK桥接研究设计方案（样本量/对照组/终点指标）
   - PD生物标志物是否可用于桥接验证？
   - 暴露-效应关系是否已建立？

3. **III期本土临床必要性判断**
   - NMPA是否明确要求中国III期？（参考2022年境外已上市化学药品注册指导原则）
   - 如需III期：样本量估算、研究周期、预算、PI遴选方案
   - 如可豁免：豁免依据文件准备清单

4. **NMPA沟通策略**
   - Pre-IND沟通会议计划
   - 关键科学问题清单（拟与NMPA讨论）
   - 沟通时间轴与里程碑

## 1.4 PC-4 安全性风险矩阵

**负责部门**：产品中心 ｜ **终审模板对应**：第一章 科学与风险

### 评估要点

1. **SAE谱分析**
   - 境外上市后安全性数据库（FAERS/EudraVigilance）中的SAE信号
   - 按器官系统分类（SOC）的不良反应发生率
   - 与同类药安全性对比

2. **黑框警告风险**
   - 境外是否存在黑框警告（Black Box Warning）？
   - 如有：具体内容、触发原因、对中国商业化的影响评估
   - 是否需要REMS（风险评估与缓解策略）或类似措施？

3. **慢性毒性数据**
   - 长期用药安全性数据（≥1年）是否充分？
   - 致癌性/致畸性/生殖毒性研究结果
   - 药物相互作用（DDI）风险评估

4. **特殊人群适用性**
   - 老年人群（≥65岁）安全性数据
   - 肝/肾功能不全患者剂量调整需求
   - 儿童/青少年适用性（如适用）
   - 妊娠/哺乳期禁忌评估

## 1.5 M-1 流行病学与市场规模

**负责部门**：市场部 ｜ **终审模板对应**：第三章 市场调查

### 评估要点

1. **流行病学数据**
   - 中国发病率/患病率（按年龄/性别/地区分层）
   - 疾病负担（DALYs/死亡率/致残率）
   - 诊断率与治疗率（诊断漏斗：总患者→确诊→治疗→目标人群）
   - 患病率趋势（增长/稳定/下降）及驱动因素

2. **市场规模测算**
   - 理论用药人群 = 患病率 × 人口基数 × 诊断率 × 治疗率
   - 真实市场规模反推 = 同类药IMS销售额 / 人均用药量
   - 峰值市场容量预测（悲观/基准/乐观三情景）
   - 市场增长率预测（5年/10年）

3. **患者分层**
   - 按疾病严重程度分层（轻/中/重）
   - 按治疗线分层（一线/二线/三线）
   - 按支付能力分层（医保/自费/商保）
   - 本品目标患者亚群定义及规模

4. **政策法规影响**
   - 医保目录调整对市场的影响
   - 集采政策对竞品价格的影响
   - DRG/DIP对处方行为的影响
   - 互联网医院/院外处方流转政策

## 1.7 M-3 定价与医保路径

**负责部门**：市场部 + 准入 ｜ **终审模板对应**：第九章 准入策略

### 评估要点

1. **参照药确定**
   - 药物经济学参照药选择（同适应症/同机制/同给药途径）
   - 参照药当前医保支付价格
   - 临床定位对比（优效/非劣效/等效）

2. **定价情景分析**
   - 悲观情景：参照药价格 × 折扣系数（通常0.6–0.8）
   - 基准情景：基于临床获益的溢价定价
   - 乐观情景：创新药溢价 + 差异化定价
   - 国际参考价格（美国/欧洲/日本/亚太）

3. **医保谈判可行性**
   - 是否符合医保目录准入条件？
   - 预期谈判降幅（参考历史谈判数据）
   - 医保支付限制条件（适应症限制/用量限制/先行自付比例）
   - 谈判时间窗口（年度医保目录调整周期）

4. **挂网策略**
   - 各省挂网价格策略（高价省份 vs 低价省份）
   - 全国联动风险评估
   - 阳光采购平台挂网流程

5. **非医保定价策略**（如医保准入不可行）
   - 自费市场定价空间
   - 商业保险覆盖可能性
   - 患者援助计划（PAP）设计

## 1.10 F-1 财务模型

**负责部门**：财务 ｜ **终审模板对应**：第四章 财务

### 评估要点

1. **销售收入预测**
   - 10年销售预测（按年/按情景）
   - 销售收入 = 销量 × 定价 ×（1 - 配送费率 - 返利率）/（1 + 增值税率）
   - 峰值销售额及达峰时间
   - 患者漏斗模型：目标患者数 → 覆盖患者数 → 处方患者数 → 实际用药患者数

2. **成本结构**
   - 引进成本：预付款 + 里程碑付款 + 销售分成
   - 研发成本：注册费用 + 临床研究费用 + CMC补充研究费用
   - 生产成本：原料药采购 + 制剂加工 + 质检 + 包装
   - 市场推广成本：学术推广 + 渠道建设 + 人员成本

3. **财务结果指标**
   - 累计净销售额
   - 峰值净销售额
   - 毛利率 / 净利率
   - 累计NPV（净现值，折现率5%）
   - IRR（内部收益率）
   - 投资回收期
   - 风险投资额及回报倍数

4. **敏感性分析**
   - 销量变动±20%对NPV的影响
   - 定价变动±15%对IRR的影响
   - 注册时间延迟1年/2年对回收期的影响
   - 竞品提前上市对市场份额的影响

5. **投资标准达标判断**
   - IRR（内部收益率）≥ 25%？
   - 5年净利率 ≥ 20%？
   - 10年累计NPV（净现值）≥ 3亿元？
   - 如未达标：是否具备战略价值可申请委员会豁免？

## 1.11 L-1 权利完整性尽调

**负责部门**：法务 ｜ **终审模板对应**：第十一章 法律

### 评估要点

1. **权属清晰度**
   - 产品权利持有人是否为签约主体？
   - 是否存在上游许可链条（原始发明人→中间持有人→当前持有人）？
   - 各环节权利转让/许可文件是否完整？

2. **在先授权义务**
   - 是否存在已授予第三方的中国市场权利？
   - 不竞争义务/排他条款范围
   - 内部排查：我方是否存在与本次引进冲突的在先义务？

3. **独占条款验证**
   - 独占范围（地域/适应症/渠道）是否满足商业化需求？
   - 独占期限是否与专利保护期匹配？
   - 独占条款的违约救济机制

4. **合规风险**
   - 反商业贿赂合规审查
   - 数据跨境传输合规（个人信息保护法/数据安全法）
   - 出口管制/制裁合规（如涉及美国技术）

## 1.12 一票否决检查表

> ⚠️ 以下任意一项成立，项目自动暂停或否决。

- ☐ **V1 — 专利有效期不足**：核心化合物专利（中国境内）在预计上市之日起有效保护期不足5年，且方法/用途专利无法提供实质保护
- ☐ **V2 — 仿制药竞争已成定局**：已有ANDA/仿制药申请在NMPA受理且预计在引进独占窗口内获批，或境外原研方已与国内仿制药企达成专利许可协议
- ☐ **V3 — 桥接不可豁免且无力承担III期**：NMPA明确要求中国III期临床，但引进方无力承担时间成本（通常>4年）或资金成本（通常>1亿元）
- ☐ **V4 — CMC资料存在不可解决性缺口**：境外持证商拒绝提供中国注册所需CMC全套资料，或原料药/制剂生产工厂无GMP互认/认证路径
- ☐ **V5 — 安全性黑框风险无法商业化**：境外存在黑框警告（Black Box Warning）且中国患者人群无替代适应症支撑，商业化预期不可行
- ☐ **V6 — 财务未达标（需委员会豁免）**：IRR（内部收益率）< 25%，且5年净利率预测 < 20%，且10年累计NPV（净现值）< 3亿元人民币（三项均未达标则自动否决，任意两项未达标需委员会审批豁免）

# 模板二：中国已上市产品 · 代理权引进

## 2.0 产品类型定义

**定义**：该产品已在中国内地取得NMPA批准文号并已商业化上市，拟通过协议授权、分销协议或商业权利转让等方式获得中国市场（全境或部分区域）的独家或非独家推广权。无需重新注册，核心任务是接管现有渠道资源并实现业务增长。

**典型特征**：

- 注册批文已存在，无注册风险，上市时间确定性最高

- 价格体系已固化，提价空间有限，竞争力主要来源于推广效率提升
- 代表案例：代理权到期的原研品种接盘、从竞争对手处接管推广权

**框架组合**：主框架——BD投前尽调（商业化章节为主）｜辅助——终审模板（裁剪版，省略注册/医学研究章节）

**可省略章节**：注册策略、医学研究策略、生产策略、投资者关系公告（如为非上市标的则同时省略）

# 模板三：早期创新药引进（临床前 / Ⅰ期阶段）

## 3.0 产品类型定义

**定义**：该产品处于药物发现/临床前研究阶段或Ⅰ期临床阶段，尚未产生Ⅱ期及以上有效性数据，拟通过License-in协议、联合开发协议或技术转让等方式引进中国开发权。核心任务是对靶点科学价值、候选化合物成药性和知识产权保护进行深度评估，以判断风险条件下的长期投资价值。

**典型特征**：

- 科学不确定性最高，Ⅰ期→Ⅱ期→Ⅲ期→上市的历史成功率约为5–10%（肿瘤领域更低）
- 引进决策主要依赖靶点验证程度、化合物质量指标（ADMET/成药性）和IP护城河
- 财务估算以rNPV（风险调整后净现值）为核心，需明确各阶段失败概率假设
- 里程碑付款结构设计是谈判重心，预付款通常控制在总对价的15–30%
- 合作方（授出方）的研发执行能力与资金稳健性是关键的可持续性风险
- 代表场景：从Biotech公司引进处于IND前/Ⅰ期的靶向药、抗体偶联药物（ADC）、双特异性抗体

**框架组合**：主框架——BD投前尽调（科学维度深度展开）+ 麦肯锡假设驱动 ｜ 终审模板仅用于决策纪律校验

---

## 3.1 S-1 靶点科学评估

**负责部门**：产品中心（医学/科学） ｜ **优先级**：★★★

### 评估要点

1. **靶点基础证据强度**
   - 靶点与目标疾病的致病因果关系是否有遗传学证据支持？（GWAS、人类孟德尔遗传病数据）
   - 靶点敲除/激活的动物模型表型是否与人类疾病高度一致？
   - 是否有人体遗传验证（如罕见病/功能获得性/丧失性突变与疾病表型关联）？
   - 靶点表达谱：在目标适应症组织中是否高选择性表达？表达丰度与疾病严重程度是否相关？

2. **靶点可成药性（Druggability）**
   - 靶点蛋白结构是否已解析（晶体/冷冻电镜）？是否有药物结合位点？
   - 属于哪类靶点（GPCR/激酶/核受体/蛋白-蛋白相互作用）？历史成药成功率参考值
   - 对于传统"不可成药"靶点：是否有创新模式（PROTAC/分子胶/共价抑制剂/RNA干预）可绕过障碍？
   - 功能化筛选（Hit discovery）结果：已知最优化合物活性指标（IC50/EC50/Ki）

3. **靶点专属性与脱靶风险**
   - 是否存在同家族靶点的交叉活性？脱靶是否导致毒性或有效性稀释？
   - 靶点在正常组织的表达情况（安全窗口评估）
   - 已报道的靶点相关毒性信号（来源：已上市同靶点药物/文献/不良反应数据库）
   - 与已上市最优同靶点药物的差异化机会评估

4. **同靶点竞品进展对比**
   - 全球同靶点在研药物管线扫描（ClinicalTrials.gov/Citeline）
   - 各竞品当前临床阶段及关键进展节点
   - 本品相较竞品的潜在优势/差异化假设（优效性/安全性/给药便利性）
   - 靶点验证程度判断：是否有竞品已证明该靶点临床有效（靶点风险降低）？

5. **疾病生物学完整性**
   - 目标疾病的驱动机制是否为单靶点可完全覆盖，还是需要联合用药？
   - 耐药机制预测：该靶点是否已有已知耐药突变/耐药路径激活机制？
   - 生物标志物策略：是否有可识别获益患者亚群的伴随诊断生物标志物？

## 3.2 S-2 候选化合物评估

**负责部门**：产品中心（化学/DMPK） ｜ **优先级**：★★★

### 评估要点

1. **效力与选择性数据**
   - 活性指标：靶点结合力（Kd/Ki）、细胞水平活性（IC50/EC50）、体内模型有效剂量
   - 选择性指数：靶点活性 vs 同家族成员脱靶活性比值（理想值 > 100倍）
   - 功能性评估：体外细胞功能实验（杀伤率/抑制增殖/通路抑制）结果

2. **ADMET（吸收/分布/代谢/排泄/毒性）性质**
   - 吸收（A）：口服生物利用度（如为口服制剂），渗透性（Caco-2/PAMPA数据）
   - 分布（D）：血脑屏障穿透性、血浆蛋白结合率、组织分布特征
   - 代谢（M）：CYP酶代谢谱、半衰期（t₁/₂），代谢产物活性/毒性评估
   - 排泄（E）：主要消除途径（肾/肝胆），尿/粪排泄比例
   - 毒性（T）：hERG心脏毒性（IC50 > 30× 有效浓度为合格线），遗传毒性（Ames/微核），细胞毒性选择窗口

3. **药代动力学（PK）参数质量**
   - 动物PK数据完整性（小鼠/大鼠/犬/猴）
   - PK/PD关系是否已建立？预测人体有效暴露量是否可实现？
   - 清除率（Cl）与稳态分布容积（Vss）是否支持目标给药频次（QD/BID）？
   - 人体首次给药剂量预测方法（MAD/异速生长法）

4. **化学可合成性与CMC可行性**
   - 合成路线步骤数及关键中间体的商业可及性
   - 起始物料（SM）供应稳定性；有无已知的供应商集中风险
   - 是否存在高风险合成步骤（剧毒试剂/高压/高温/光敏反应）
   - 已开发的分析方法：纯度（HPLC）/含量/晶型表征是否成熟？

5. **备选化合物系列（SAR多样性）**
   - 是否有≥3个结构骨架的候选化合物系列（Back-up化合物）？
   - 若先导化合物失败，Back-up化合物的启动时间及成本预估
   - 构效关系（SAR）是否清晰，优化空间是否充足？
   - 新型模式药物（ADC/PROTAC）的linker/payload/E3 ligase选择合理性

## 3.8 S-8 里程碑结构与财务边界

**负责部门**：财务 + BD ｜ **终审模板对应**：第四章 财务

### 评估要点

1. **rNPV（风险调整后净现值）建模**
   - 各阶段转化概率假设（临床前→Ⅰ期→Ⅱ期→Ⅲ期→上市，各阶段行业基准成功率）
   - 适应症加权rNPV（多适应症开发时按患者规模/时间加权）
   - 折现率选择（通常15–20%，反映早期创新药的高风险溢价）
   - 峰值销售额预测（上市后5–10年，悲观/基准/乐观三情景）

2. **里程碑付款结构设计**
   - 预付款（Upfront）范围建议：总对价的15–30%，上限不超过rNPV的20%
   - 开发里程碑触发条件（IND获批/Ⅱ期入组完成/POC阳性/Ⅲ期启动）
   - 注册里程碑触发条件（NDA受理/NDA批准/首次商业销售）
   - 销售里程碑触发条件（首年销售额达X亿/累计销售额达Y亿）
   - 合理化里程碑总额：预付款 + 开发里程碑不超过rNPV × 50%

3. **销售提成/分成（Royalty）设计**
   - 提成率区间：净销售额的5–15%（根据引进方开发投入比例确定）
   - 阶梯提成结构（销售额越高，提成率递增）的适用条件
   - 提成计算基础：净销售额（Net Sales）定义的合同确认
   - 亚许可费用分配机制（引进方向第三方再许可时的分成）

4. **总投资预算与现金流测算**
   - 中国开发总投入：临床研究费 + CMC补充 + 注册申报费 + 人员成本
   - 年度现金流预测（投资期：通常8–12年至盈亏平衡）
   - 最大净投资暴露（Max Net Exposure）：高风险场景下的最大累计亏损
   - 各阶段里程碑付款的时间安排与公司现金流承受能力匹配

5. **财务止损机制**
   - POC失败时的合同终止条件及前付款回收安排
   - 选择权（Option）结构设计：先支付Option费用锁定权利，POC后再决策
   - 竞争性授权条款（若竞品优先完成Ⅲ期，引进方是否有对价调整权利）

## 3.10 一票否决检查表

> ⚠️ 以下任意一项成立，项目自动暂停或否决。

- ☐ **V1 — 靶点证据薄弱**：靶点与疾病的致病因果关系缺乏遗传学/功能验证证据，仅有相关性数据，同靶点迄今无任何临床阶段药物达到POC阳性
- ☐ **V2 — 成药性缺陷**：候选化合物存在不可克服的ADMET（吸收/分布/代谢/排泄/毒性）缺陷（hERG IC50/有效浓度 < 5倍，或遗传毒性阳性，或口服生物利用度 < 5%且无注射给药替代方案），且Back-up化合物系列数量不足
- ☐ **V3 — IP争议**：核心化合物专利在中国境内存在无效宣告诉讼或Third-party撤销请求，或权属链条中存在未解决的共有纠纷，导致引进方无法取得清洁授权
- ☐ **V4 — 竞品临床领先3年以上**：同靶点竞品已完成Ⅲ期招募并预计在本品预计上市前3年以上获批上市，且竞品临床数据显示疗效优势明确（ORR差距 > 20%），本品无差异化路径
- ☐ **V5 — 合作方融资断裂**：合作方当前现金跑道不足12个月，且无已确认的新融资计划，导致本品开发计划在关键POC节点前面临中断风险
- ☐ **V6 — rNPV为负**：在基准情景下（行业平均临床成功率、合理峰值销售假设），rNPV（风险调整后净现值）计算结果为负值，且无法通过调整里程碑结构使rNPV转正

# 模板四：生物类似药 / 仿制药引进

## 4.0 产品类型定义

**定义**：该产品为已上市原研药的仿制药（化学药一致性评价品种）或生物类似药（生物制品类似药），已完成或正在开展相似性研究，拟通过授权生产、技术转让或股权合作方式在中国开发和销售。核心任务是评估质量相似性、注册路径可行性、集采竞争策略和盈利能力边界。

**典型特征**：

- 科学不确定性低（活性成分明确），注册路径相对清晰，但技术门槛不可低估（尤其是生物类似药）
- 市场准入（集采/医保）策略直接决定商业成功，财务测算对价格假设高度敏感
- 竞争格局已固化或快速固化：过评家数多意味着价格战不可避免
- 生物类似药有更高的科学要求和更长的开发周期，注册成本通常为化学仿制药的3–10倍
- 差异化策略（剂型/给药装置/超适应症/院外渠道）是逃脱集采绞杀的关键
- 代表场景：单抗生物类似药引进（如贝伐珠单抗/曲妥珠单抗类似药）、专科仿制药技术转让

**框架组合**：主框架——终审模板裁剪版（强化财务+生产+准入，弱化科学差异化）｜辅助——BD投前尽调

**可简化章节**：科学机制（直接引用原研INN机制说明）、医学研究策略（类似药适应症随原研）

## 4.1 G-1 质量相似性评估

**负责部门**：产品中心（质量/CMC） ｜ **优先级**：★★★

### 评估要点

1. **分析相似性研究（化学仿制药）**
   - 溶出曲线对比（多pH值条件下，与原研药的f2相似因子 ≥ 50）
   - 杂质谱比较：仿制药杂质是否在原研药可接受范围内？新增杂质是否已完成安全性评估？
   - 晶型/多晶型分析：固体制剂晶型是否与原研一致？
   - 关键质量属性（CQA）对比表：粒度/溶出度/纯度/水分含量

2. **生物相似性研究（生物类似药）**
   - 结构特征分析：氨基酸序列、二硫键、糖基化图谱比较（高分辨率质谱数据）
   - 理化特性：分子量、等电点、聚集体含量、氧化/脱酰胺分析
   - 生物学活性：体外结合活性（靶点结合/Fc受体结合），细胞功能实验（CDC/ADCC）
   - 免疫原性评估：异质性指标、ADA（抗药抗体）检测方法建立
   - 动物PK/PD相似性研究（大鼠/猴模型）

3. **对照药（RLD，参比制剂）选择**
   - 中国RLD指定情况（是否已被NMPA官方认定为参比制剂？）
   - 境内外参比制剂一致性（原料药来源/批号/生产工艺是否同源）
   - RLD批次变异性评估（是否有≥3批原研产品对照数据）

4. **GMP合规状态**
   - 生产工厂GMP证书状态及有效期
   - FDA/EMA是否已检查且无警告信（Warning Letter）
   - 中国药监局是否已开展或有计划开展境外检查？
   - 偏差/变更管理记录：近3年内的重大变更申报情况

5. **稳定性数据完整性**
   - 加速/长期稳定性数据覆盖期限（至少覆盖申报时间+ 12个月）
   - 运输应力（Transport Stress）稳定性研究（针对温控制剂）
   - 有效期设定依据及申报有效期

## 4.2 G-2 仿制注册路径

**负责部门**：注册 ｜ **优先级**：★★★

### 评估要点

1. **注册分类与申报策略**
   - 化学仿制药：是否已通过一致性评价（BE试验方案、豁免条件）？
   - 生物类似药：按《生物类似药研发与评价技术指导原则》规定的相似性路径
   - 参比制剂资格确认：NMPA参比制剂目录收录情况
   - 申请人资质：申报企业是否已具备相应类别的生产许可证

2. **BE（生物等效性）试验评估（化学仿制药）**
   - BE方案设计：交叉/平行；空腹/餐后；样本量计算
   - BE试验机构资质：GCP认证状态，受试者招募能力
   - 预期BE通过风险：基于体外溶出/体内PK数据的预测
   - BE豁免可行性：BCS（生物药剂学分类系统）分类Ⅰ/Ⅲ类豁免申请条件

3. **临床对比研究（生物类似药）**
   - PK相似性研究设计（健康受试者，与原研比较AUC/Cmax 90% CI）
   - 临床有效性对比研究要求（敏感适应症选择，如RA用于单抗类似药）
   - 免疫原性对比研究（ADA发生率及临床影响对比，随访≥52周）
   - 互换性（Interchangeability）数据要求（如目标市场有此要求）

4. **数据包完整性与时间轴**
   - 质量研究报告（CTD模块2.3/3.2）完成进度
   - 非临床研究：是否有豁免可能或需补充特定研究？
   - 关键申报资料预计就绪时间
   - 总注册时间轴（从申报到获批，化学仿制药通常12–18个月，生物类似药通常18–30个月）

5. **优先审评资格**
   - 临床急需、儿童用药、罕见病等优先审评条件适用性
   - 是否属于"首个仿制药"加速审评范围？
   - 部分自费品种豁免BE的政策适用性评估

### 评估要点

1. **原料药成本**
   - 原料药来源：自产 vs 外购；供应商集中度
   - 原料药当前采购价格及历史波动情况
   - 原料药占总生产成本的比例（化学药通常30–60%，生物制品更高）
   - 原料药产能约束与扩产计划

2. **制剂生产成本分解**
   - 直接材料成本：原料药 + 辅料 + 包装材料
   - 直接人工成本：生产工时及人员效率
   - 制造费用：设备折旧、能耗、质控成本
   - 批次成本及单位成本（元/片/元/支/元/mg）

3. **规模效应与产量规划**
   - 当前设计产能 vs 实际产量
   - 产量翻倍时的边际成本下降预测
   - 集采中选后产量扩张的资本开支（CAPEX，资本性支出）需求及时间

4. **生产成本竞争力评估**
   - 与主要集采竞争对手的成本对比（如可获取信息）
   - 目标：生产成本是否可低于预期集采中选价格的60%（保障合理利润率）
   - 技术改进空间：工艺优化/原料替代/规模化降本路径

5. **质量成本**
   - 历史批次不合格率及质量成本占比
   - GMP维护年度成本
   - 放行检验成本（尤其生物类似药批次检定成本较高）

## 4.6 G-6 财务模型（集采敏感性）

**负责部门**：财务 ｜ **终审模板对应**：第四章 财务

### 评估要点

1. **基准财务模型假设**
   - 集采中选年份预测及采购量基准
   - 集采单价（元/片/元/支）：基于G-3竞品分析的报价策略
   - 非集采渠道（院外/自费）销量及定价假设
   - 医保目录状态及支付比例

2. **集采敏感性分析（核心分析）**
   - 集采中选 vs 未中选两种场景的销量、收入、利润对比
   - 集采价格每下降10%对IRR（内部收益率）的影响
   - 参与家数变化（+2家 vs +5家 vs +10家）对中选价格和份额的影响
   - 未中选时院外渠道的销售保底能力分析

3. **5年财务预测（三情景）**
   - 悲观情景：集采第2轮即参与，价格下降60%，院外份额有限
   - 基准情景：集采第1轮参与，价格下降40%，院外渠道补充15%销量
   - 乐观情景：差异化包装/剂型未纳入集采，保留较高定价空间

4. **投资回报指标**
   - IRR（内部收益率）目标：仿制药/类似药要求 ≥ 20%（低于创新药要求）
   - 投资回收期：目标 ≤ 4年（仿制药风险较低，回收期更短）
   - 5年累计NPV（净现值）
   - 毛利率目标：集采品种 ≥ 35%，非集采品种 ≥ 50%

5. **引进对价谈判边界**
   - 基于NPV（净现值）测算的最高可接受总对价（含技术转让费+预付款+里程碑）
   - 分期支付结构建议（注册里程碑+销售里程碑分期触发）
   - 技术转让费合理区间（占NPV的15–25%）

## 4.7 G-7 数据保护期与市场独占

**负责部门**：法务 + 注册 ｜ **优先级**：★★

### 评估要点

1. **数据保护期评估（仿制药侧）**
   - 化学仿制药：原研新药数据保护期（中国为6年）是否已过期？
   - 生物类似药：原研生物制品数据保护期评估（中国按品种类别有所不同）
   - 数据保护期届满时间点与本品预计上市时间的先后关系

2. **市场独占机会**
   - 若本品为首个过评仿制药：是否可争取"首仿"商业优势（先发上市6–12个月）？
   - 是否有资格申请专利挑战（化学仿制药在原研专利有效期内）？
   - 专利链接制度（4+4制度）下的诉讼等待期规划

3. **孤儿药/罕见病豁免**
   - 若有罕见适应症：孤儿药认定状态及市场独占保护期
   - 儿童用药数据保护期附加（6个月额外保护）适用性

4. **竞品数据保护期障碍**
   - 是否有已上市的类似仿制药享有尚未届满的独占期？（如首仿3年独占期）
   - 该保护期对本品上市时间节点的约束

## 4.8 G-8 差异化包装策略

**负责部门**：市场部 + 产品中心 ｜ **优先级**：★★

### 评估要点

1. **剂型差异化**
   - 原研药/主要仿制药未覆盖的剂型：缓释制剂/微粒/透皮/冻干粉等
   - 差异化剂型的开发成本及是否需要额外临床数据
   - 差异化剂型是否可规避纳入集采（按规格分集采）？

2. **给药装置差异化（注射剂/吸入剂）**
   - 预充针/自动注射笔/吸入器装置便利性对患者依从性的影响
   - 装置专利独立保护：申请给药装置相关专利
   - 装置差异化的商业推广价值（医院自费选择/商保覆盖）

3. **适应症组合差异化**
   - 超适应症申报：是否有机会在原研已上市适应症基础上拓展新适应症？
   - 新适应症注册成本及时间与商业价值匹配评估
   - 儿童适应症/老年患者优化剂量规格开发

4. **院外/DTC（直接面向消费者）市场挖掘**
   - 院外处方药/OTC转化可行性（如有政策支持）
   - 互联网医院 + 电商处方药平台覆盖策略
   - 商业保险覆盖谈判策略（针对集采品种院外自费部分）

## 4.9 一票否决检查表

> ⚠️ 以下任意一项成立，项目自动暂停或否决。

- ☐ **V1 — 注册路径不可行**：同通用名在审ANDA数量超过20家且均已完成BE试验，或NMPA明确不接受现有质量研究包（相似性不充分），注册获批无合理预期
- ☐ **V2 — 集采后经济不可行**：在预计参与家数和历史降幅假设下，集采中选价格低于生产成本的120%（毛利率不足20%），且院外渠道无法提供足够销量补偿
- ☐ **V3 — 生产成本不具竞争力**：引进方生产成本（含原料药）高于竞品主流成本水平30%以上，且工艺改进空间有限，无法在集采竞价中获得合理利润
- ☐ **V4 — 质量相似性不达标**：溶出f2 < 50（化学仿制药）或结构相似性关键指标（糖基化图谱/生物活性）与原研差异显著超出可接受标准（生物类似药），且无可行改进方案
- ☐ **V5 — 财务未达标**：集采基准情景下IRR（内部收益率）< 20%，且非集采院外情景下IRR（内部收益率）< 15%，无战略补偿价值

# 模板五：战略级平台型合作（4.0模式 · 共同开发）

## 5.0 产品类型定义

**定义**：该合作不限于单一产品的引进，而是围绕特定技术平台（如ADC平台/双特异性抗体平台/基因疗法平台/合成生物学平台）、多个管线资产组合或区域联合开发协议，以共同开发（Co-development）、共同商业化（Co-commercialization）或共同出资（Co-investment）为核心机制，建立长期战略伙伴关系。引进方与授出方均承担实质性开发义务及风险共担。

**典型特征**：

- 合作范围超越单一产品，涉及平台技术访问权、多适应症开发权或区域商业化权利包
- 战略匹配度（合作方战略愿景、技术互补性、组织文化）是成败的关键变量
- IP归属设计极为复杂，改良化合物/新适应症/新剂型的IP权属须在协议前明确框架
- 里程碑及对价结构须覆盖平台级和产品级，层次更复杂
- 退出机制（Termination Rights）和分手费设计是长期合作的合规保障
- 组织执行能力（联合指导委员会/项目协调团队）决定合作效率
- 代表场景：与国际Biotech建立中国区共同开发协议；与国内头部药企建立平台技术共享合作

**框架组合**：主框架——麦肯锡8步法（Issue Tree + 战略逻辑验证）｜辅助——BD投前尽调 + 终审模板

---

## 5.1 P-1 战略问题定义

**负责部门**：战略/BD ｜ **优先级**：★★★

### 评估要点

1. **战略问题陈述（Problem Statement）**
   - 此次合作拟解决的核心战略问题是什么？（用一句话明确定义）
   - 示例陈述："我们如何在ADC（抗体偶联药物）领域快速建立技术平台，以在2028年前推出≥2个临床阶段候选化合物，同时避免自建平台所需的5–8年周期和15–20亿元资本投入？"
   - 问题是否可被结构化分解（MECE：相互独立、完全穷尽）？

2. **当前战略缺口识别**
   - 引进方在哪些技术领域、适应症领域或地区市场存在能力缺口？
   - 若不开展此合作，5年后的竞争地位预测（参照竞争对手的布局速度）
   - 内部自建 vs 收购 vs 合作三种路径的成本/时间/风险对比矩阵

3. **战略成功标准（Definition of Success）**
   - 此次合作在3年/5年/10年维度的可量化成功指标（KPI清单）
   - 成功的定性标准：技术能力积累、品牌影响力提升、监管关系建立
   - 失败的早期预警信号清单（便于触发合作退出评估）

4. **关键决策假设**
   - 此次合作成立的3–5个核心假设（列举并排序）
   - 每个假设的可证伪性：何时/以何种证据可验证或推翻该假设？
   - 最脆弱的假设识别：若该假设被推翻，合作价值是否根本性改变？

## 5.2 P-2 战略匹配度分析（Issue Tree）

**负责部门**：战略/BD ｜ **优先级**：★★★

### 评估要点

1. **技术互补性（Technology Complementarity）**
   - 合作方拥有的关键技术能力清单（明确标注我方缺乏的核心能力）
   - 我方拥有的关键能力（临床开发/商业化/政策关系/患者洞察）对合作方的吸引力
   - 技术互补图谱：双方技术能力的交集与互补区域可视化
   - 技术转让深度：合作方愿意分享的技术层次（表面Know-how vs 核心Know-how vs 平台架构）

2. **战略目标一致性（Strategic Alignment）**
   - 双方在目标适应症/目标市场/开发时间表上的一致程度
   - 双方对合作优先级的认知对齐（合作方视此为核心合作 vs 多元化布局之一）
   - 双方退出预期一致性：两方均期望长期合作 vs 一方期望短期套现的隐性冲突

3. **组织文化兼容性**
   - 决策风格：合作方的决策周期/层级/容忍模糊度与我方是否匹配？
   - 项目管理方法论：里程碑驱动 vs 敏捷开发，是否可建立共同语言？
   - 跨文化沟通有效性：语言/时区/工作文化差异的实质影响评估
   - 历史合作参考：合作方与其他中国合作伙伴的合作案例评估（成功/失败/争议记录）

4. **市场战略协同**
   - 双方在中国医院渠道/支付方/KOL体系上的互补资源
   - 双方客户群体（细分市场）的重叠度与互补性
   - 联合商业化是否产生"1+1>2"的协同效应？量化估算协同收益

5. **Issue Tree（议题树）结构**

```
战略合作是否值得推进？
├── 技术层面：双方能力是否形成真正互补？
│   ├── 合作方核心技术是否领先行业？
│   └── 我方能力是否足以支撑共同开发执行？
├── 商业层面：合作是否创造超越独立运营的价值？
│   ├── 协同收益是否量化可信？
│   └── 合作对价结构是否公平反映双方贡献？
├── 风险层面：合作风险是否可管理？
│   ├── IP争议风险是否有预防机制？
│   └── 合作方稳定性是否足以支撑长期承诺？
└── 退出层面：若合作不成功，是否有清洁退出路径？
    ├── 合同终止条款是否保护我方权益？
    └── 资产分割方案是否预先约定？
```

## 5.3 P-3 合作方战略稳定性评估

**负责部门**：BD + 法务 + 财务 ｜ **优先级**：★★★

### 评估要点

1. **财务健康度**
   - 现金储备及Runway（资金跑道）：可支撑的运营月数
   - 年度运营亏损（Burn Rate）及与融资计划的匹配性
   - 债务结构：是否有近期到期的重大债务/可转债/优先股回购义务？
   - 最近一轮融资条款：下行保护条款（Ratchet/Anti-dilution）是否影响未来融资意愿？

2. **管理层稳定性**
   - CEO/CSO/Business Development负责人的任职年限及离职风险
   - 董事会构成：独立董事比例，是否有财务投资人主导的短期退出压力？
   - 近12个月核心高管变动记录及原因分析
   - 创始科学家是否仍深度参与核心项目决策？

3. **股东结构与控制权**
   - 主要股东清单及持股比例
   - 是否有战略投资者（Strategics）持股，其战略意图是否与合作方向一致？
   - 是否有VIE结构或多重股权结构带来的治理复杂性？
   - 被竞争对手收购的风险评估（Change of Control条款的重要性）

4. **历史里程碑履约记录**
   - 与其他合作伙伴已完成的里程碑达成情况（实际完成日期 vs 计划日期）
   - 是否有过合作终止、延迟支付或纠纷记录？
   - 开发进度报告的质量与透明度（历史信息披露评估）

5. **中国市场承诺深度**
   - 合作方在中国是否已有实体（WFOE/合资公司/代表处）？
   - 中国市场在其全球战略中的优先级（核心市场 vs 机会性市场）
   - 合作方高层是否曾多次访华，与监管机构/KOL是否有直接接触经验？

## 5.4 P-4 管线价值评估

**负责部门**：产品中心 + 财务 ｜ **优先级**：★★★

### 评估要点

1. **管线资产清单与分层**
   - 完整管线清单（平台名称/候选化合物/临床阶段/目标适应症/全球开发状态）
   - 优先级分层：将管线分为核心资产（A级）、支撑资产（B级）、期权资产（C级）
   - A级资产的详细评估（参照模板三S-1至S-3章节执行科学评估）
   - 平台技术的长期产出预测：基于历史数据预测平台每年可生成的新候选化合物数量

2. **管线多样性与组合价值**
   - 适应症多样性（是否集中于单一肿瘤/疾病领域，还是跨领域布局）
   - 开发阶段分布（早期/中期/晚期的组合比例，确保管线连续性）
   - 作用机制多样性（单靶点 vs 联合靶点，是否互相竞争或互补）
   - 失败相关性（若一条管线失败，是否导致整个平台价值受损）

3. **平台技术价值评估**
   - 平台技术的独特性评估：业内竞品平台对比（技术壁垒高度）
   - 平台的可扩展性：从单一适应症扩展到10+适应症的技术路径是否成立？
   - 平台IP保护范围：核心平台专利是否覆盖足够宽的技术范围？

4. **整合NPV（净现值）测算**
   - 分别对A级/B级/C级管线资产进行rNPV（风险调整后净现值）测算
   - 按各资产在组合中的权重汇总计算整体管线组合价值
   - 折扣因子：平台合作的协同价值（如共享临床基础设施、共同KOL维护）

5. **非管线价值评估**
   - 技术能力积累价值（员工学习/专利组合增厚/监管经验积累）
   - 品牌合作效应（与知名合作方联合开发对公司融资能力/市值的影响）
   - 未来独家谈判权（Right of First Negotiation/Right of First Refusal）的隐性价值

## 5.5 P-5 知识产权归属设计

**负责部门**：法务（专利） ｜ **优先级**：★★★

### 评估要点

1. **基础IP（Background IP）界定**
   - 合作开始前双方各自拥有的IP清单（附录形式）
   - Background IP的许可范围：合作方向对方授予多大范围的Background IP访问权？
   - Background IP使用的地域限制、适应症限制和时限限制
   - Background IP的不可撤销性条款（保证合作期间Background IP授权不中断）

2. **联合IP（Jointly Developed IP）归属规则**
   - 联合研究成果的IP归属框架（各国法律对共有IP的默认规则差异）
   - 改良化合物（Improvements）的归属：引进方主导改良 vs 授出方主导改良的不同处理
   - 衍生物/同类化合物的IP归属：是否纳入合作范围还是各方独立保留？
   - 新适应症发现的IP归属：在合作平台上发现的新适应症IP如何分配？

3. **商业化IP权利**
   - 中国市场商业化权利的排他性程度（独占 vs 共同销售 vs 非独占）
   - 引进方在中国境内对联合IP的独占使用权期限
   - 亚许可（Sub-licensing）权利：引进方是否可向第三方再许可？条件和分成
   - 海外市场权利：引进方是否有权在东南亚/中华圈其他市场使用合作IP？

4. **竞争性研究限制**
   - 不竞争条款（Non-compete）：合作期间及合作结束后的范围和期限
   - 同类靶点/同类平台技术的合作排他性约束
   - 竞争性研究披露义务：任何一方开展潜在竞争研究的提前告知机制

5. **技术转让深度与保护**
   - Know-how转让范围（书面文件 vs 人员培训 vs 现场技术支持）
   - 逆向工程限制条款
   - 保密协议（NDA）条款：保密期限、保密范围、例外情形
   - 合作结束后的技术留用权（Paid-up License）条款

## 5.6 P-6 里程碑与对价结构

**负责部门**：BD + 财务 ｜ **优先级**：★★★

### 评估要点

1. **平台访问费（Platform Access Fee）**
   - 进入合作的一次性平台授权费
   - 合理范围测算：基于平台技术价值评估（P-4）的折扣率（通常为平台rNPV的5–15%）
   - 分期支付条件：首付款 + 平台验证里程碑付款的组合设计

2. **开发里程碑体系**
   - 平台层里程碑：新候选化合物进入GLP毒理/IND申请/Ⅱ期启动
   - 产品层里程碑：各管线POC阳性/NDA申请/NDA批准/首次商业销售
   - 里程碑金额分布合理性：前期开发阶段不承担过大财务压力，后期商业化里程碑占比更高
   - 条件性里程碑：部分里程碑的触发可附加商业条件（如销售额达标才触发注册里程碑支付）

3. **收益分成（Profit Share）设计**
   - 合作品种中国市场利润分成比例（区别于传统Royalty模式，按利润而非销售额分成）
   - 分成计算基础：净利润（Net Profit）定义的合同准确界定（可扣除费用清单）
   - 阶梯分成：利润越高，合作方分成比例递增（激励合作方支持商业化）
   - 超额利润奖励机制（Upside sharing）

4. **共同开发成本分担**
   - 各临床阶段的开发费用分担比例（引进方：授出方）
   - 成本上限约定：避免开发成本超支导致分担义务无限扩大
   - 独立决策权：费用超出预算时，哪方有权决定是否继续投入？
   - 不均衡投入时的权利再平衡机制（如一方减少投入，另一方可增加投入换取更高分成比例）

5. **对价公平性验证**
   - 以P-4管线价值评估为基础，计算总对价占整体管线rNPV（风险调整后净现值）的比例
   - 可比较交易案例对比（同类平台合作的交易条款公开数据）
   - 独立财务顾问估值建议（如总对价超过10亿元，建议聘请投行出具意见书）

## 5.7 P-7 风险退出机制

**负责部门**：法务 ｜ **优先级**：★★★

### 评估要点

1. **单方面终止权（Termination Rights）**
   - 引进方终止权触发条件：技术平台失败/合作方违约/合作方被竞争对手收购
   - 授出方终止权触发条件：引进方未按约分担开发费用/引进方重大违约/引进方破产
   - 终止通知期限（通常60–180天）
   - 非违约方在对方行使终止权后的资产保留权利

2. **控制权变更（Change of Control）保护**
   - 合作方被收购时：引进方是否有权终止合作并保留已开发资产的商业化权利？
   - "罪犯条款"：若合作方被特定竞争对手收购，引进方自动获得终止权 + 资产买断选择权
   - 收购方承继义务：新股东是否须书面确认承继全部合作义务

3. **资产分割方案**
   - 合作终止时，各方保留资产的清单预设（联合IP如何分割？）
   - 合作期间共同积累的数据包归属（临床数据/真实世界数据所有权）
   - 已支付里程碑款项的不可退还约定 vs 特定违约下的部分退还机制
   - 合作终止后的过渡期安排（供应保障/临床患者继续用药义务）

4. **争议解决机制**
   - 适用法律：中国法 vs 国际仲裁规则（建议优先选择HKIAC/ICC/SIAC）
   - 仲裁地点与语言
   - 强制调解前置程序
   - 裁决的可执行性（对方主要资产所在地的法律认可仲裁裁决）

5. **保护条款完整性**
   - 反稀释条款：合作方增发新股融资时，引进方IP授权条款是否受影响？
   - 优先谈判权（ROFR/ROFN）：若授出方欲出售本合作涵盖资产，引进方享有优先购买权
   - 最惠国待遇（MFN）：授出方若给予其他合作方更优惠的授权条款，引进方可要求同等待遇

## 5.8 P-8 组织执行能力评估

**负责部门**：项目管理/BD ｜ **优先级**：★★

### 评估要点

1. **联合指导委员会（JSC）架构**
   - JSC成员构成（双方各派几人？级别要求？）
   - 决策权层级划分（JSC决策范围 vs 项目层委员会 vs 双方各自独立决策）
   - JSC例会频率（季度/半年）及决议执行跟踪机制
   - 僵局（Deadlock）打破机制：持续分歧时的升级路径

2. **项目执行团队（JPC）**
   - 每个合作项目的项目经理指派机制
   - 跨团队协调：引进方与授出方项目团队的沟通频率和工具
   - 时区/语言障碍的实际管理方案
   - 信息共享平台：用于共享试验数据/进度报告的系统权限设计

3. **引进方内部执行能力**
   - 引进方是否有具备共同开发经验的国际化BD/Research团队？
   - 专职资源承诺：为本次合作投入的FTE（全职员工当量）数量
   - 是否需要外部CRO/CMO的配合？合规管理能力
   - 项目管理软件和风险管理体系的成熟度

4. **里程碑执行跟踪**
   - 里程碑进展的季度汇报标准格式设计
   - 落后预警机制：里程碑延迟≥30天启动自动预警
   - 补救计划（Remediation Plan）触发条件及审批流程
   - 合作成效的年度审视机制（Annual Alliance Review）

5. **文化融合管理**
   - 双方团队的Line Manager层面交流计划（研究员/临床科学家互访）
   - 合作初期的"快赢项目"设计（Early Win）：建立互信的低风险小型合作
   - 合作精神（Alliance Spirit）文件：明确双方对合作价值观的共同承诺

## 5.9 P-9 集团战略标准验证

**负责部门**：战略/最高决策层 ｜ **优先级**：★★★

### 评估要点

1. **集团战略一致性**
   - 本次合作是否明确支持集团3年/5年战略规划中的优先重点领域？
   - 与集团现有管线（已引进/在研/上市）的协同或竞争关系评估
   - 本次合作是否与集团"重点疾病领域专注"原则保持一致？
   - 集团资本配置优先级中，本次合作的相对排名

2. **战略投资标准达标**
   - IRR（内部收益率）≥ 25%（战略级合作，含战略价值溢价加成）
   - 管线组合rNPV（风险调整后净现值）> 10亿元（人民币）
   - 技术能力积累目标：合作结束后，引进方是否具备独立开展同类研究的能力？
   - 集团投委会（IC）批准阈值：超过某金额（如1亿元预付款）须提交集团投委会

3. **机会成本评估**
   - 若资金（假设投入X亿元）用于其他战略路径（自建/收购/多个小项目），预期回报对比
   - 管理层注意力机会成本：本次合作占用的核心团队时间，是否挤占其他优先项目？
   - 短期财务压力：前2–3年的里程碑付款是否影响公司年度利润/现金流目标？

4. **监管与合规标准**
   - 是否需要MOFCOM（商务部）反垄断申报？（双方合并市场份额判断）
   - 是否涉及外商投资安全审查？（战略性技术领域的国家安全考量）
   - 数据安全合规：合作中涉及跨境传输的临床数据是否满足《数据安全法》《个人信息保护法》要求？
   - ESG合规：合作方是否符合集团的供应商ESG准则？

5. **六大战略标准终审**

| 战略标准 | 具体指标 | 目标值 | 当前评估结论 |
|---|---|---|---|
| 临床价值 | 目标适应症未满足需求程度 | 高度未满足需求 | 达标/不达标 |
| 技术壁垒 | 平台技术独特性 + IP护城河 | 行业领先且专利保护充分 | |
| 市场潜力 | 合作管线整体rNPV | > 10亿元人民币 | |
| 执行可行性 | 双方组织能力匹配度 | JSC架构已设计，FTE承诺到位 | |
| 财务回报 | IRR（内部收益率） | ≥ 25% | |
| 退出保障 | 终止条款 + 资产分割条款完整性 | 合同全面保护引进方权益 | |

## 5.10 一票否决检查表

> ⚠️ 以下任意一项成立，项目自动暂停或否决。

- ☐ **V1 — 战略目标根本性不一致**：Issue Tree分析显示，合作方的商业目标（如短期套现、快速出售公司）与引进方的长期战略（如建立自主创新能力、深耕特定疾病领域）存在根本性矛盾，且无法通过合同条款设计弥合
- ☐ **V2 — IP归属无法清洁划定**：联合开发成果的IP归属框架无法在引进决策前达成原则性协议，或合作方在Background IP上存在第三方共有权益导致无法授予引进方清洁许可
- ☐ **V3 — 退出机制不完整**：合同草稿缺乏Change of Control（控制权变更）保护条款，或合作方被竞争对手收购场景下引进方无任何法律手段保护已开发资产的使用权
- ☐ **V4 — 合作方资金断裂风险极高**：合作方现金跑道不足18个月，无已确认融资计划，且平台核心开发里程碑节点在24个月内，合作有很大概率因合作方运营中断而失败
- ☐ **V5 — 管线价值严重低估风险**：独立财务评估显示，合作方要求的总对价（含预付款+全部里程碑付款）超过管线组合rNPV（风险调整后净现值，折现率15%）的80%，引进方承担了与回报不匹配的高额对价
- ☐ **V6 — 不符合集团战略标准**：集团投委会六大战略标准中有≥3项不达标，且无法通过调整合作范围或条款结构使关键指标达标

## 5.11 验收标准分级

| 维度 | 必达标准（不达即暂停） | 参考标准（允许条件性通过） | 否决触发条件 |
|---|---|---|---|
| 战略匹配 | Issue Tree分析证明技术互补性显著（我方缺乏的核心能力由合作方补充），双方战略目标文件一致性确认 | 局部战略不一致，但核心业务协同价值足以覆盖分歧，已约定战略对齐季度会议机制 | 双方商业目标根本性对立（一方套现导向，一方长期建设导向），无合同设计可弥合 |
| IP保护 | 合作IP归属框架协议（Term Sheet）已签署，核心改良化合物IP归引进方所有，联合IP商业化权归引进方独占（中国区） | 改良化合物IP共有，但引进方取得独占商业化授权，且有明确的买断选择权 | IP框架未达成原则性协议，或背景IP存在第三方未披露共有权益 |
| 退出保障 | Change of Control条款（控制权变更保护）明确纳入合同，合作终止时资产分割方案已预设且对引进方公平 | 退出条款仍有3–5处细节待谈判，但核心资产保护原则已书面确认 | 合同缺乏任何终止保护条款，引进方合作终止后无任何资产留存权利 |
| 管线价值 | 合作管线A级资产rNPV（风险调整后净现值）> 5亿元（人民币），平台技术独立价值> 3亿元 | 管线A级资产rNPV 3–5亿元，但平台技术学习价值和战略协同价值可作补充 | 管线rNPV < 2亿元，且平台技术无独立价值，合作仅为财务交易无战略意义 |

## 5.12 快速启动清单（前12周行动优先级）

**第1–3周：战略定义与问题结构化**

1. 完成P-1战略问题陈述，明确合作必须解决的核心战略缺口
2. 内部构建Issue Tree（P-2），识别需要验证的3–5个关键假设
3. 向合作方索取完整管线清单及平台技术白皮书（Non-confidential Overview）

**第4–6周：深度尽调启动**

4. 签署双向保密协议（MNDA），进入数据室（Data Room）审阅详细技术资料
5. 产品中心对A级管线资产开展S-1/S-2级别科学评估（参照模板三执行）
6. 法务团队对Background IP清单开展产权链核查及FTO（自由实施）初步评估

**第7–9周：财务建模与对价设计**

7. 财务团队完成管线组合rNPV（风险调整后净现值）模型（三情景）
8. 基于P-6框架完成里程碑付款结构设计，确定最高可接受总对价上限
9. 对合作方开展详细财务尽调（P-3）：现金跑道、股东结构、债务状况

**第10–12周：谈判准备与集团审批**

10. 完成IP归属框架草稿（P-5），提交法务团队与合作方法律顾问进行Term Sheet谈判
11. 设计JSC架构（P-8）及联合执行方案，确认引进方内部FTE（全职员工当量）资源承诺
12. 完成P-9集团战略标准六维验证，准备投委会汇报材料（不超过30页PPT），提交集团投委会终审

# 模板六：院外/消费健康品种引进（OTC · 健康消费品）

## 6.0 产品类型定义

**定义**：该产品属于非处方药（OTC）、功能性食品、保健品、医疗器械（I/II类）或广义健康消费品，主要通过院外渠道（连锁药店、电商、商超、直销、私域流量）实现商业化，终端消费者为个人自主购买决策者，而非依赖医生处方驱动。

**典型特征**：

- 消费者品牌认知度和口碑是核心壁垒，学术推广不适用或适用范围极窄
- OTC甲/乙类、备案制保健品、功能性食品适用不同注册/备案路径，监管复杂度差异显著
- 毛利率（Gross Margin）目标通常需 ≥ 55%，以支撑高额品牌营销和电商流量成本
- 流量获取成本（CAC，Customer Acquisition Cost，用户获取成本）和复购率（Repurchase Rate）是财务模型核心变量
- 竞品高度分散，差异化依赖品牌故事、成分创新与消费者情感共鸣，而非单纯临床数据
- 供应链通常为境外代工或境内OEM（Original Equipment Manufacturer，原始设备制造商），评估重点为质量标准与配方独特性保护
- 代表案例：某功效护肤品牌境内代理引进、进口维生素矿物质类OTC产品授权运营

**框架组合**：主框架——BD尽调（消费品裁剪版）| 大幅弱化注册/医学研究/供应链深度审查 | 强化品牌资产+消费者调研+电商数据+数字营销ROI（Return on Investment，投资回报率）分析

**省略章节说明**：
- **医学研究策略**：消费品无需独立医学研究团队，学术活动仅限于功效宣传背书，不作单独章节评估
- **准入策略（医保）**：OTC及健康消费品不适用医保谈判和集采，省略医保准入全部子项
- **供应链（境外代工简化）**：境外代工标准化程度高，仅需核查GMP（Good Manufacturing Practice，良好生产规范）证书有效性和独家配方保护条款，不展开完整供应链尽调
- **投资者关系公告**：适用于上市公司重大标的，院外消费品引进通常低于信披门槛，省略

## 6.1 O-1 品牌资产评估

**负责部门**：市场部 ｜ **优先级**：★★★

### 评估要点

1. **品牌知名度与认知度**
   - 目标消费群体自发提及率（TOM，Top of Mind，第一提及率）及辅助提及率
   - 品牌在核心城市（一线/新一线）与下沉市场的认知度差异
   - 海外品牌在国内社交媒体（小红书、抖音、微博）的自然流量与讨论热度
   - 品牌是否有"网红爆品"历史或明星/KOL（Key Opinion Leader，关键意见领袖）背书记录
   - 近3年品牌声量趋势（上升/持平/下降）及驱动因素

2. **品牌情感价值与定位清晰度**
   - 品牌核心卖点是否清晰？消费者能否在5秒内描述出品牌承诺
   - 品牌情感属性（天然、科技感、专业功效、轻奢、平价高效等）与目标人群契合度
   - 品牌视觉系统（Logo、包装、色彩）是否符合中国消费者审美偏好
   - 是否存在文化敏感性风险（品牌名称/视觉/成分在中国市场的本土化障碍）
   - 与竞品的品牌差异化感知度（消费者是否能区分本品与竞品）

3. **已有用户资产与口碑积累**
   - 现有粉丝/会员/用户数量及活跃度（官方渠道+第三方平台）
   - 消费者NPS（Net Promoter Score，净推荐值）或等价口碑评分
   - 用户UGC（User Generated Content，用户原创内容）数量及情感倾向
   - 主要平台（天猫、京东、抖音）已有旗舰店运营状态及历史销售数据
   - 消费者投诉率及主要投诉类型（质量/效果/物流/客服）

4. **品牌授权范围与合规使用**
   - 授权方给予的品牌使用权范围（渠道/地域/产品线/时间）
   - 是否有品牌使用规范手册（Brand Guidelines）？限制条款是否影响本土化运营
   - 品牌名称及相关商标在中国内地的注册状态（参见模板六O-7及O-8联查）
   - 历史合作方是否有过品牌滥用或负面事件，是否已彻底切割

5. **品牌所处赛道的竞争拥挤度**
   - 该细分品类中，是否已有5个以上具有较强品牌认知的竞品？
   - 新进入者依靠纯流量投入能否在12个月内建立品牌记忆点
   - 头部平台（天猫/抖音/小红书）该品类的流量成本趋势（CPC，Cost Per Click，单次点击成本）
   - 品类是否处于增长期/成熟期/衰退期，对品牌建设预算的影响

## 6.2 O-2 产品力与配方评估

**负责部门**：产品中心 + 研发/质量 ｜ **优先级**：★★★

### 评估要点

1. **核心功效成分与科学依据**
   - 主要功效成分的作用机制是否有同行评审文献或品牌专有研究支持
   - 成分浓度/配比是否处于有效剂量范围（对比行业标准和竞品）
   - 功效宣称是否符合中国广告法及相关法规（不得宣称治疗疾病功效）
   - 是否有独家/专利成分作为核心差异化支撑（专利号、有效期、地域覆盖）
   - 消费者可感知的功效体验窗口（使用多久能感受到效果）

2. **配方安全性与法规符合性**
   - 所有成分是否在中国《已使用化妆品原料目录》、卫生部批准保健食品功能等许可清单内
   - 是否含有中国禁限用成分（对照NMPA禁限用物质清单逐一核查）
   - 成品及原料的安全性测试报告（刺激性、致敏性、重金属、微生物）是否有效
   - 不良反应历史：境外上市后是否有安全性事件记录
   - 特殊人群（孕妇、儿童、老年人、过敏体质）的适用性声明

3. **产品配方的独特性保护**
   - 配方是否受到专利保护或技术秘密保护？保护强度如何
   - 竞品是否可以快速仿制核心配方（成分可及性/工艺复杂度）
   - 授权合同中是否有防止配方泄露的保密条款
   - 独家成分的原料来源是否唯一，存在断供风险（单一供应商）

4. **产品形态与使用体验**
   - 剂型/包装形式是否符合中国消费者使用习惯（便携性、用量控制、视觉吸引力）
   - 气味、质地、颜色等感官体验是否需要本土化调整
   - 产品货架寿命（Shelf Life）是否符合跨境物流+本地仓储的时间要求
   - 是否存在产品系列化扩展潜力（延伸品类、不同规格/功效）

5. **产品迭代能力**
   - 原厂是否有持续产品研发投入和新品发布节奏
   - 未来3年是否有升级款/新系列计划，是否纳入本次引进协议
   - 本土化定制配方的可行性（是否允许针对中国市场调整配方）

| 维度 | 评估结果 | 风险等级 | 备注 |
|---|---|---|---|

## 6.3 O-3 院外渠道匹配度

**负责部门**：商业化/渠道 ｜ **优先级**：★★★

### 评估要点

1. **现有渠道资产盘点**
   - 公司现有院外渠道覆盖：连锁药店（DTP，Direct to Patient，直接面向患者药房；全国连锁/区域连锁）、商超、便利店、电商（天猫/京东/抖音/拼多多）、私域（微信社群/小程序）
   - 现有渠道与本产品目标消费者的重合度评估（频次、场景、消费能力匹配度）
   - 渠道铺货效率：新品进场所需时间和条码费（Slotting Fee，上架费）预估
   - 现有渠道对健康消费品品类的运营经验和资源成熟度

2. **电商渠道能力**
   - 公司天猫旗舰店/京东自营/抖音小店的粉丝基础和历史GMV（Gross Merchandise Volume，商品交易总额）
   - 直播带货资源：是否有稳定的达人合作网络和直播运营团队
   - 小红书种草能力：KOL/KOC（Key Opinion Consumer，关键意见消费者）资源储备
   - 搜索广告（SEM，Search Engine Marketing，搜索引擎营销）和信息流广告投放经验
   - 跨境电商（天猫国际、京东国际）渠道的可用性（针对境外品牌）

3. **线下药店渠道协同性**
   - 与国大药房、益丰大药房、老百姓大药房等全国TOP10连锁的合作状态
   - 现有品种在药店的陈列位置与流量带动能力（关联品类是否协同）
   - 药店导购/店员培训资源：是否有成熟的push销售培训机制
   - 本产品在药店的陈列场景定位（OTC货架/健康品专区/收银台旁）

4. **渠道独占性与价格管控**
   - 授权方是否给予中国市场独家代理权或特定渠道独占授权
   - 是否存在其他分销商或平行进口渠道导致价格体系混乱
   - 最低零售价（MAP，Minimum Advertised Price，最低广告价）保护机制是否明确
   - 跨境电商代购灰色渠道的存在规模及其对正价体系的威胁

5. **渠道扩张路径规划**
   - 第1年：聚焦核心电商平台+核心城市TOP5连锁药店，建立基础铺货
   - 第2年：扩展新渠道（抖音电商/私域/商超），下沉至二三线城市
   - 第3年：完成全渠道覆盖，探索线下专柜/品牌形象店
   - 各阶段渠道投入预算及ROI目标

## 6.4 O-4 消费者需求验证

**负责部门**：市场研究/市场部 ｜ **优先级**：★★★

### 评估要点

1. **目标消费者画像**
   - 核心消费人群：年龄/性别/地域/收入水平/消费偏好（从现有用户数据或调研获取）
   - 消费动机：功效驱动型/情感/社交/礼品/悦己等主要购买动机构成比例
   - 消费场景：线上自购、线下冲动购买、礼品、差旅购买等场景权重
   - 消费频次与客单价：复购周期、单次消费金额、年度消费总额（LTV，Lifetime Value，用户终身价值）
   - 消费决策链路：关键信息渠道（小红书/种草内容/医生建议/朋友推荐）

2. **需求未满足度验证**
   - 目标品类是否存在消费者明确的"痛点"或"痒点"？
   - 现有竞品对这些需求的满足程度（消费者评价分析：差评词云/高频吐槽）
   - 本品的差异化卖点是否被消费者感知为真实价值（而非品牌方自我认知）
   - 消费者对价格区间的接受度（支付意愿调研或类比竞品定价验证）
   - 关键功效的消费者自测体验满意度（小范围消费者测试或众测数据）

3. **市场容量与增长趋势**
   - 目标品类近3年线上GMV增长趋势（来源：天猫/京东公开数据、欧睿、艾瑞等第三方数据）
   - 品类搜索量趋势（百度指数/微信指数/小红书话题热度）
   - 用户渗透率提升空间（目标人群中已使用该品类的比例）
   - 是否存在政策利好或生活方式变迁驱动品类扩容（如"颜值经济"、"抗衰"需求增长）

4. **本土化需求适配评估**
   - 境外产品功效诉求是否与中国消费者需求完全匹配（是否需要调整宣传重点）
   - 产品名称/成分名称的中文化方案及消费者接受度
   - 使用习惯差异（境外使用方法是否符合中国消费者日常习惯）
   - 是否需要开发专属中国市场的规格或套装（本土化SKU，Stock Keeping Unit，库存管理单位）

5. **消费者调研数据来源核实**
   - 定量调研：样本量、调研方式（在线问卷/拦截访问）、调研时间及代表性
   - 定性调研：焦点小组（FGD，Focus Group Discussion）或深度访谈（IDI，In-depth Interview）数量
   - 社交聆听工具：是否有第三方（如新红数据、蝉妈妈、飞瓜数据）支持的社媒数据
   - 真实用户测评报告（第三方众测平台：小红书众测/微博超级测评）

## 6.5 O-5 竞品与定位分析

**负责部门**：市场部 ｜ **优先级**：★★

### 评估要点

1. **直接竞品矩阵**
   - 直接竞品（同品类/同功效/同目标人群）列表（至少5个，含国产和进口品牌）
   - 各竞品的年度销售额/增速（来源：欧睿、CBN Data、平台公开数据）
   - 竞品价格带分布及本品拟定价格区间的竞争格局
   - 竞品核心卖点及消费者口碑分析（评论词频/NPS估算）
   - 竞品渠道策略及主力推广平台

2. **间接竞品与替代品威胁**
   - 不同品类但满足相同消费需求的替代品（功能交叉竞争）
   - 医疗机构（皮肤科/营养科）开具的处方产品对本品的替代威胁
   - 消费者是否会在本品与替代品间进行显著价格敏感性比较

3. **差异化定位设计**
   - 基于消费者研究和竞品分析，锁定本品的核心差异化定位（1句话价值主张）
   - 差异化维度选择（成分/功效/品牌故事/包装/价格/渠道/服务中选择1–2个核心维度）
   - 差异化是否可持续（竞品能否在12个月内跟进并消除差异化）
   - 定位在消费者调研中的共鸣度验证分数

4. **价格定位策略**
   - 拟定零售价格区间（RRP，Recommended Retail Price，建议零售价）
   - 与竞品价格的相对位置（溢价/平价/低价进入）及理由
   - 促销价格空间（常规折扣幅度不超过原价的X%）
   - 不同渠道的价格一致性管控方案

5. **市场份额目标与路径**
   - 第1年目标市场份额/销售额（保守/基准/乐观情景）
   - 第3年市场份额目标及达成路径假设（品牌投入、渠道扩张）
   - 关键里程碑：何时进入品类TOP10/TOP5

## 6.6 O-6 数字营销ROI评估

**负责部门**：数字营销/电商运营 ｜ **优先级**：★★★

### 评估要点

1. **流量获取成本基准测算**
   - 目标品类在天猫/京东/抖音的平均CPC（Cost Per Click，单次点击成本）及行业基准数据
   - 小红书品类CPE（Cost Per Engagement，单次互动成本）和内容转化率
   - 各渠道的平均转化率（CVR，Conversion Rate，点击转化率）及本品预估值
   - 首单CAC（Customer Acquisition Cost，用户获取成本）测算：平台广告费 + KOL费用 + 运营人力摊销
   - 新客获取成本与消费者首单毛利润的对比（是否单次购买即可盈利）

2. **复购率与用户终身价值**
   - 目标品类行业平均复购率（3/6/12个月）
   - 本品预估复购率（依据竞品数据、消费者调研、产品消耗周期）
   - 用户LTV（Lifetime Value，用户终身价值）测算：平均客单价 × 年购买频次 × 留存年限
   - LTV/CAC目标比率（健康消费品行业基准通常要求 ≥ 3:1）
   - 私域沉淀比例目标（将公域流量转化为私域会员的比例及成本节省估算）

3. **KOL/KOC推广效能评估**
   - 目标品类头部KOL（粉丝 > 100万）的平均报价及单条种草ROI（Return on Investment，投资回报率）
   - 腰部KOL（粉丝10–100万）和KOC（粉丝 < 10万）的性价比对比
   - 是否有已合作的头部KOL资源可优先复用
   - 小红书搜索SEO（Search Engine Optimization，搜索引擎优化）布局成本（品类关键词占位）
   - 评估"硬广+软文+KOL种草+直播带货"的组合投放比例

4. **内容营销能力评估**
   - 公司内容创作团队能力（图文/短视频/直播脚本）
   - 历史爆款内容案例分析（哪类内容形式转化效率最高）
   - 品牌自媒体账号矩阵的粉丝量级及内容质量（小红书/抖音/微信公众号）
   - UGC激励机制：是否有成熟的用户口碑激励方案（返现/积分/晒单奖励）

5. **数字营销预算规划与ROI目标**
   - 年度数字营销预算总额及分渠道拆分（付费流量/KOL/内容/私域）
   - 整体营销费用率目标（数字营销投入/销售额，健康消费品通常20–40%）
   - 每季度ROI复盘机制和投放策略调整节奏
   - 爆款打造路径：从"0到1冷启动"所需最低预算和时间周期预估

## 6.7 O-7 OTC注册/备案路径

**负责部门**：注册/法规 ｜ **优先级**：★★（简化版，重点核查合规底线）

### 评估要点

1. **产品法规属性界定**
   - 依据成分、功效宣称、使用方式，确认产品在中国的法规属性：
     - 非处方药（OTC甲类/乙类）：需NMPA批文，适用《药品管理法》
     - 保健食品（蓝帽子）：注册制（功能声称）或备案制（维生素矿物质），适用《保健食品注册与备案管理办法》
     - 普通食品/功能性食品：备案或无需备案，但功效宣称受严格限制，适用《食品安全法》
     - 化妆品（特殊/普通）：注册或备案，适用《化妆品监督管理条例》
   - 错误的法规属性界定是最大的注册风险，建议在引进谈判前完成法规分类确认

2. **现有境外注册证明文件**
   - 境外原产地销售证明（FSC，Free Sales Certificate，自由销售证明）是否可获取
   - 原产地GMP（Good Manufacturing Practice，良好生产规范）证书有效性
   - 境外上市批准文件（FDA/CE/TGA等）是否适用于中国注册申请
   - 原料/成分的中国法规许可清单核查（针对化妆品和保健食品尤为关键）

3. **注册/备案时间轴与成本**
   - 保健食品注册制：预计18–24个月，费用约30–50万元人民币（含检验、申报、咨询）
   - 保健食品备案制（维生素矿物质）：预计3–6个月，成本相对较低
   - 化妆品特殊类注册：预计6–12个月
   - 化妆品普通类备案：预计1–3个月，最快路径
   - OTC药品注册（如原有境外批文）：通常24–36个月，整体成本较高

4. **关键合规风险点**
   - 功效宣传合规：是否有宣称治疗/预防疾病的内容（违反广告法，须删除）
   - 成分禁限用风险：依据最新NMPA禁限用物质清单核查
   - 进口备案/注册名义持有人要求：境内需有法定代理人或持证方
   - 跨境电商保税模式的合规边界（是否需要完整国内注册证才能走一般贸易）

5. **备案/注册后的维护合规成本**
   - 年度检验/检测要求（化妆品安全性再评估）
   - 配方变更、包装变更的变更申报流程
   - 不良反应监测上报义务（化妆品/保健食品均适用）

## 6.8 O-8 财务模型（消费品逻辑）

**负责部门**：财务 ｜ **优先级**：★★★

### 评估要点

1. **收入预测（消费品漏斗模型）**
   - 目标可及人群（TAM，Total Addressable Market，总可及市场）→ 目标服务市场（SAM，Serviceable Addressable Market，可服务市场）→ 可获取市场份额（SOM，Serviceable Obtainable Market，实际可获取市场）分层估算
   - 年度GMV预测（电商渠道）+ 线下净销售额（药店/商超）
   - 各渠道销售拆分比例：电商/线下/跨境/私域
   - 第1–5年收入预测（保守/基准/乐观情景）
   - 收入增长驱动假设：新客增长、复购提升、品类扩张

2. **成本结构与毛利率分析**
   - 产品采购成本（来自授权方的转让价格或分成比例）
   - 进口关税、增值税、检验检测费用
   - 毛利率（Gross Margin，毛利润率）计算：（零售价 - 采购成本 - 关税/税费）/ 零售价
   - 平台佣金扣点（天猫：5–8%，抖音：5–10%，京东：3–6%）
   - 物流仓储费用率（通常占销售额5–10%）
   - 综合渠道后净毛利率目标：消费品通常要求 ≥ 55%

3. **营销投入与费用率**
   - 数字营销费用（付费流量+KOL+内容）：年度总额及占销售额比例
   - KOL/KOC合作费用拆分
   - 品牌建设费用（品牌设计本土化、公关、线下体验活动）
   - 销售团队和渠道运营人力成本
   - 首年重点投放期费用率可容忍上限（通常可接受40–50%，确保爆量）

4. **盈亏平衡测算**
   - 月度/年度盈亏平衡点（BEP，Break-Even Point，盈亏平衡点）销售额计算
   - 从引进到首次月度盈利的时间预估
   - 累计投入金额（研发/注册+营销启动资金+首批采购库存）
   - 从累计亏损到累计盈利的转折时间轴（通常消费品 18–36 个月）

5. **核心财务指标目标**
   - 第3年净利润率目标（消费品参考值：10–20%）
   - IRR（Internal Rate of Return，内部收益率）目标（参考值：≥ 25%）
   - 投资回收期：通常要求 ≤ 3年
   - EBITDA（Earnings Before Interest, Taxes, Depreciation and Amortization，息税折旧摊销前利润）利润率目标

## 6.9 一票否决检查表

> ⚠️ 以下任意一项成立，项目自动暂停或否决。

- ☐ **V1 — 品牌认知为零且赛道高度拥挤**：目标消费群体对本品牌TOM（Top of Mind，第一提及率）< 2%，且同品类在天猫/抖音TOP10品牌中无一被消费者替换意愿，重建品牌认知的年度营销投入预算超过公司可承受上限（通常 > 3,000万元），引进ROI（Return on Investment，投资回报率）不可行
- ☐ **V2 — 配方保护严重不足**：核心功效成分无专利保护且原料可从公开渠道低成本获取，同等配方的山寨产品已在市场大量流通，差异化壁垒为零
- ☐ **V3 — 院外渠道无协同，建设成本超预算**：公司现有院外渠道与本品目标消费场景完全不重叠，渠道从零建设的前期投入预估 > 5,000万元，且无法借助现有资源快速启动，第1年难以实现最低销售规模
- ☐ **V4 — OTC/备案注册存在实质性法规障碍**：产品核心成分在中国属于明确禁限用物质，或功效宣称在中国法规下根本无法合法表述，导致产品在中国市场无合规销售路径
- ☐ **V5 — 财务模型未达标**：基准情景下第3年净利润率 < 8%，或IRR（Internal Rate of Return，内部收益率） < 20%，且无品牌战略价值可补偿财务短板

# 模板七：多标的横向比较筛选（快速漏斗 · 对比矩阵）

## 7.0 产品类型定义

**定义**：本模板适用于需要对3个及以上候选引进标的进行系统性横向比较和优先级排序的场景，尤其适合BIO展会/JPM大会后的集中标的筛选、管线规划期的品类布局决策、以及多个潜在合作方同步推进时的快速漏斗分流。

**适用场景**：
- 大型展会（BIO、ASCO、ESMO、CPhI）后需在72小时内对10–30个接触标的完成初步分级
- 公司战略规划期对全市场候选标的进行系统扫描，产出优先引进清单
- 同一治疗领域内多个相近产品需要横向比较确定最优引进对象
- 管线优化时对现有在谈标的重新评分排序，决定资源集中方向

**框架组合**：BD尽调（标准化对比矩阵版）+ 麦肯锡假设驱动（评分假设显性化）

**使用说明**：
- 本模板分为两阶段漏斗结构：第一阶段为Gate Check（硬性否决过滤），第二阶段为量化评分矩阵
- 所有评分人员应在评分前独立完成，然后进行小组校准，避免锚定效应
- 评分假设须在评分表备注栏中明确列出，确保假设可追溯、可复盘
- 不同类型产品（处方药 vs OTC vs 器械）在同一矩阵中比较时，需在备注中标注类型差异对评分的影响

## 7.1 第一阶段：Gate Check（硬性否决过滤）

> 以下7个硬性否决条件，任意一项触发即将标的移出候选池，无需进入第二阶段评分。

**使用规则**：
- 每个标的须逐一核查全部7项，有一项确认触发即记录否决原因后终止该标的评估
- 数据不足无法判断的项目标注"待核实（P）"，给予48小时窗口期补充数据，窗口期结束仍无数据者视同"触发否决"
- Gate Check应在接触标的后72小时内完成

---

### G-1 战略方向符合性

**判断标准**：
- **通过（Pass）**：产品治疗领域、患者人群或商业化模式与公司当前战略重点领域（治疗领域白名单）完全匹配，或经BD负责人确认列为战略机会性引进
- **否决（Fail）**：产品所在治疗领域已明确列入公司战略黑名单（如公司已决定退出的品类），或产品商业化模式（如仅适合自建医院渠道的高度专科化产品）与公司现有能力完全不匹配，且无战略资源投入计划

**数据来源**：
- 公司最新战略规划文件（通常为年度BP，Business Plan，商业计划书）
- BD负责人快速口头确认（5分钟）
- 产品说明书/合作方介绍材料中的适应症和渠道信息

---

### G-2 知识产权（IP）基础合规性

**判断标准**：
- **通过（Pass）**：原研专利存续期内（或保护期剩余 ≥ 5年），或已有清晰的专利战略（FTO，Freedom to Operate，专利自由实施分析已完成），无已知专利侵权诉讼
- **否决（Fail）**：核心化合物专利已在中国到期或即将（< 2年）到期，且已有多家仿制药/生物类似药（Biosimilar）进入NMPA（National Medical Products Administration，国家药品监督管理局）审评，竞争格局将在引进后商业化窗口内显著恶化；或存在已公开的专利无效诉讼/侵权诉讼且胜算不确定

**数据来源**：
- 中国专利数据库（CNIPA，China National Intellectual Property Administration，国家知识产权局）官网查询
- incoPat / 智慧芽 / PatSnap 专利检索平台
- 合作方提供的专利清单及到期时间表

---

### G-3 NMPA注册可行性底线

**判断标准**：
- **通过（Pass）**：产品已在FDA/EMA/PMDA（Pharmaceuticals and Medical Devices Agency，日本药品和医疗器械局）获批，具备申请中国注册的基础条件；或已在中国递交申请且无实质性技术障碍；或按OTC/保健食品/化妆品路径无明显法规障碍
- **否决（Fail）**：核心成分在中国属于明确禁止使用的物质；或产品所需的中国本土桥接研究规模 > III期（即需要重做完整III期），引进后商业化时间轴 > 7年；或产品在境外尚处于I期临床且有效性证据极为有限（适用于早期创新药快速筛选场景例外）

**数据来源**：
- NMPA药品综合查询平台（已申报/已获批产品查询）
- CDE（Center for Drug Evaluation，药品审评中心）官网技术指导原则
- 合作方提供的注册计划文件

---

### G-4 合作方资质与诚信底线

**判断标准**：
- **通过（Pass）**：合作方为上市公司或经核实的知名制药/生物技术企业；或经尽职背调无重大合规问题记录；法定代表人及核心管理层无重大刑事/民事诉讼记录
- **否决（Fail）**：合作方在FDA警告信（Warning Letter）黑名单内；或有已公开的数据造假/临床欺诈记录；或在中国市场有多个已失效批文且无合理解释；或合作方注册地为高风险司法管辖区且无法核实实际控制人

**数据来源**：
- FDA Warning Letter 数据库（公开可查询）
- 中国裁判文书网（企业诉讼记录）
- 企查查/天眼查（企业工商、股权、法律风险信息）
- 标准背景调查（建议委托第三方尽调公司）

---

### G-5 市场竞争格局底线

**判断标准**：
- **通过（Pass）**：目标适应症在中国市场同类药品 < 5个，或本品相对于现有竞品具有明确临床优势（头对头数据证明）或显著价格优势
- **否决（Fail）**：目标适应症已有 ≥ 10个同类药品在市，且本品无明确临床差异化数据，市场高度同质化；或目标品类已经历多轮国家带量采购（VBP，Volume-Based Procurement，带量采购），价格中枢已降至利润上限以下，无论引进结果均无法达到毛利率（Gross Margin，毛利润率）门槛

**数据来源**：
- NMPA药品数据库（同通用名获批品种数量查询）
- IMS/IQVIA中国市场报告
- 国家医保局带量采购中选结果公告

---

### G-6 财务规模基准线

**判断标准**：
- **通过（Pass）**：按最保守情景测算，引进后第5年峰值年销售额预测 ≥ 5,000万元人民币（Renminbi，人民币）（中小型引进）或 ≥ 2亿元（大型引进），且毛利率（Gross Margin）≥ 40%（处方药）/ ≥ 55%（消费品）
- **否决（Fail）**：即使在乐观情景下，市场天花板测算峰值年销售额 < 2,000万元人民币，规模不足以支撑专项引进投入（注册费用+推广启动资金+人力成本）；或毛利率在任何可预见情景下均无法达到最低门槛

**数据来源**：
- 中国患者人群流行病学数据（来源：IQVIA、中国疾控中心报告、文献）
- 竞品价格及市场份额数据（IMS/米内数据）
- 公司财务部预审测算

---

### G-7 合规与伦理底线

**判断标准**：
- **通过（Pass）**：产品临床数据无已公开的数据完整性质疑；合作方无环境/社会/治理（ESG，Environmental, Social and Governance）重大违规记录；产品适应症及目标人群符合公司伦理准则（如无争议的超适应症儿童用药等情形）
- **否决（Fail）**：产品涉及已公开的临床数据造假调查（REMS，Risk Evaluation and Mitigation Strategy，风险评估和缓解策略受损）；或合作方有重大环境污染/强迫劳动等ESG丑闻，引进将显著影响公司品牌声誉；或产品适应症存在明显伦理争议（如非医学必需的增强类应用于儿童）

**数据来源**：
- ClinicalTrials.gov（临床试验数据完整性审查）
- Retraction Watch（已撤稿论文数据库）
- 公司ESG委员会或法务伦理审查意见

## 7.2 第二阶段：10维量化评分矩阵

> **评分规则**：每个维度满分5分，1/3/5分对应明确判断标准（见各维度说明）。禁止使用2分或4分——评分人须明确选边，避免模糊中间态。所有维度权重默认相等（每维度10%），特殊项目可在矩阵顶部注明权重调整及理由。

---

### D-1 临床价值与差异化

**评估维度**：本品相对于现有标准治疗方案（SOC，Standard of Care，标准治疗方案）的临床价值增量及差异化强度

| 分值 | 判断标准 |
|---|---|
| **5分** | 有头对头III期数据证明相对现有SOC的显著优效（P < 0.05，主要终点达标），且在安全性或便利性上无显著劣势；或为该适应症首个可用疗法（Unmet Need完全未满足） |
| **3分** | 无头对头优效数据，但有单臂研究或历史对照数据显示与SOC相当，同时在给药便利性/安全性/患者依从性上有1–2个可量化优势（如从静脉注射改为口服，每周给药改为每月一次） |
| **1分** | Me-too产品，无差异化临床证据，与现有SOC相比无可量化优势，差异化仅依赖价格 |

**评分假设要求**：评分人须在备注中注明所依据的核心临床数据来源（论文DOI/会议摘要编号）及数据检索时间。

---

### D-2 知识产权保护强度

**评估维度**：引进后可预期的专利保护窗口和竞争壁垒持续时间

| 分值 | 判断标准 |
|---|---|
| **5分** | 核心化合物/工艺专利在中国保护期 ≥ 10年，且有多层专利保护（化合物专利+适应症专利+制剂专利），FTO（Freedom to Operate，专利自由实施分析）已完成，无已知侵权风险 |
| **3分** | 核心专利保护期5–10年，专利保护层次单一（仅化合物专利），无已知活跃诉讼，但FTO未完全核查 |
| **1分** | 核心专利保护期 < 5年，或已有1–2家过评仿制药进入NMPA审评，专利悬崖（Patent Cliff）近在眼前 |

---

### D-3 中国注册可行性与时间确定性

**评估维度**：在中国完成注册上市的技术可行性、时间可预测性和监管风险

| 分值 | 判断标准 |
|---|---|
| **5分** | 已在中国获批或递交申请，或属于NMPA优先审评品种（突破性治疗、罕见病、儿童用药），注册路径清晰，预计上市时间确定性高（误差 ≤ 6个月） |
| **3分** | 境外已获FDA/EMA批准，5.1类技术路径可行，但需进行中国人群桥接研究，整体注册周期24–36个月，存在1–2个可识别的技术不确定因素 |
| **1分** | 境外尚在II期临床，中国注册需重做III期或境外数据不被NMPA认可，注册周期 ≥ 5年，时间节点高度不确定 |

---

### D-4 市场规模与增长潜力

**评估维度**：目标适应症的中国患者人群规模和市场增长趋势

| 分值 | 判断标准 |
|---|---|
| **5分** | 目标适应症中国患者人群 ≥ 50万（可治疗人群，非患病率数据），市场年增速 ≥ 15%，且患者确诊率和治疗率仍处于上升阶段，渗透空间大 |
| **3分** | 患者人群10–50万，市场年增速5–15%，市场已进入成熟增长阶段，有稳定但有限的市场扩张空间 |
| **1分** | 患者人群 < 10万（超罕见病需单独评估罕见病溢价逻辑），或市场增速 < 5%甚至负增长，市场天花板低 |

---

### D-5 竞争格局优势度

**评估维度**：进入中国市场时面临的竞争强度及本品的相对竞争地位

| 分值 | 判断标准 |
|---|---|
| **5分** | 目标适应症同类药品 ≤ 3个，且本品在疗效/安全性/给药方式上有明确优势；或本品进入后可快速成为一线治疗方案，获得指南推荐支持 |
| **3分** | 同类药品4–8个，本品与主要竞品无明显优劣势差异，差异化依赖品牌推广和学术教育，需要持续营销投入维持竞争地位 |
| **1分** | 同类药品 ≥ 10个，高度同质化竞争，或已经历国家带量采购（VBP，Volume-Based Procurement，带量采购）价格大幅压低，竞争已进入纯价格战阶段 |

---

### D-6 合作方实力与合作意愿

**评估维度**：授权方/合作方的研发实力、履约能力及推进本次合作的真实意愿

| 分值 | 判断标准 |
|---|---|
| **5分** | 合作方为全球TOP50制药企业或知名Biotech（已上市，市值 > 10亿美元），产品信息充分披露，谈判响应速度快（48小时内回复），已完成过中国合作先例或表达了进入中国市场的明确战略意图 |
| **3分** | 中型Biotech或区域性制药公司，有一定研发实力，合作意愿明确但中国市场经验有限，尽调配合度中等，信息披露基本完整但部分核心数据需要反复催促获取 |
| **1分** | 初创企业（< Series B）或合作方对中国市场了解极为有限，关键数据披露不完整或存在明显信息不对称，谈判响应迟缓（> 72小时不回复），合作意愿不明确 |

---

### D-7 商业化能力匹配度

**评估维度**：公司现有商业化能力（团队/渠道/学术资源）与本产品商业化需求的匹配程度

| 分值 | 判断标准 |
|---|---|
| **5分** | 产品所在科室/适应症是公司现有核心推广领域，已有成熟销售团队、核心KOL（Key Opinion Leader，关键意见领袖）关系网络和院内覆盖；引进后可快速商业化（上市后6个月内达成铺货目标） |
| **3分** | 相邻科室或部分渠道重叠，需新建1–2个推广团队或扩充学术支持能力，但整体商业化能力建设周期 ≤ 12个月，可控 |
| **1分** | 完全新的治疗领域和患者群体，现有团队无任何覆盖，渠道从零建立，KOL关系空白，商业化能力建设预计需要 > 18个月且需要大规模人员招聘 |

---

### D-8 财务回报预期

**评估维度**：基准情景下的财务回报质量（综合考量规模、利润率、回收周期）

| 分值 | 判断标准 |
|---|---|
| **5分** | 基准情景：第5年峰值销售额 ≥ 2亿元人民币，IRR（Internal Rate of Return，内部收益率）≥ 30%，投资回收期 ≤ 3年，峰值净利率 ≥ 20% |
| **3分** | 第5年峰值销售额5,000万–2亿元，IRR 15–30%，投资回收期3–5年，净利率10–20%；或规模较小但有明确战略价值（管线补充/渠道协同价值可量化） |
| **1分** | 第5年峰值销售额 < 5,000万元，或IRR < 15%，或投资回收期 > 5年，且无法识别可量化的战略补偿价值 |

---

### D-9 引进成本与谈判空间

**评估维度**：授权引进成本（首付款+里程碑+版税）的合理性及谈判空间

| 分值 | 判断标准 |
|---|---|
| **5分** | 合作方报价低于同类产品市场参考价（Benchmark）30%以上，或合作方有明显的战略急迫性（资金压力/需要中国合作方背书），谈判空间大；首付款 < 项目NPV（Net Present Value，净现值）的15% |
| **3分** | 报价基本符合市场参考范围（±20%），双方均有合理诉求，存在一定谈判空间，首付款约为项目NPV的15–25% |
| **1分** | 合作方报价显著高于市场参考价（ > 30%溢价），且合作方谈判态度强硬，谈判空间极小；或首付款要求 > 项目NPV的25%，引进经济性存疑 |

---

### D-10 战略协同价值

**评估维度**：超越单产品财务回报的战略价值（管线协同/平台价值/生态位强化）

| 分值 | 判断标准 |
|---|---|
| **5分** | 引进后显著强化公司在核心治疗领域的管线布局（形成1+N协同效应），或获取的技术平台/KOL资源可支撑后续3个以上产品引进；或产品引进能帮助公司进入新的战略高价值赛道（如肿瘤/罕见病/CNS）并带来品牌溢价 |
| **3分** | 对现有管线有补充价值，与已引进产品在患者人群或推广团队上有部分协同，但协同效益难以精确量化，整体战略价值中等 |
| **1分** | 与公司现有管线和战略方向高度独立，无明显协同效应，纯财务投资逻辑，不强化任何现有核心能力 |

## 7.4 验收标准

| 维度 | 必达标准 | 参考标准 | 否决触发条件 |
|---|---|---|---|
| 评分一致性 | 同一标的两位独立评分人的总分差距 ≤ 8分（满分50分），偏差 ≤ 16%；超出须组织校准会议 | 单维度最大评分差距 ≤ 2分；所有主要分歧均有文字解释记录 | 总分差距 > 15分（≥ 30%偏差），说明评分基础信息存在重大理解差异，须暂停评分重新对齐信息口径 |

## 7.5 快速启动清单

1. 在接触标的后24小时内，完成Gate Check的G-1（战略符合性）和G-4（合作方资质）两项快速核查，决定是否投入资源继续评估
2. 建立标的信息收集模板（一页纸，包含10个维度各需要的关键数据点），于首次接触时同步发送给合作方填写
3. 分配两位独立评分人（通常为BD负责人 + 产品中心代表），明确评分校准会议时间（建议在各自独立评分后48小时内召开）
4. 评分前统一一次"评分假设对齐会"（30分钟）：明确本批标的所处治疗领域的判断基准（如市场规模判断标准、竞品认定范围），避免因理解差异导致同一标的评分偏差过大
5. 对总分 ≥ 40分的标的，在评分输出后48小时内提交BD委员会快速立项申请，启动模板一至六的深度尽调程序
6. 将全部评分结果（含否决原因）录入标的管理数据库，建立每季度复盘机制，对"观察跟进"类标的自动触发再评提醒
7. 对总分25–39分的"观察跟进"标的，设置明确的升级触发条件（见7.6再评触发条件设置指引），避免无限期搁置消耗关注资源
