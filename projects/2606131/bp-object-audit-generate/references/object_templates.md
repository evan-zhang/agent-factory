## 6. 完整复盘版模板（v1.5 §4.4.1 / R20/R31）

单目标完整复盘版须包含以下 5 部分，缺任一不得进 `review_ready` 状态：

1. **目标表**：目标名称、目标周期、目标主责
2. **成果层完整表**：成果编号、名称、主责、衡量标准、输出对象/最终验收物、证据路径/规则卡位置
3. **关键举措层完整表**：对应成果、举措、主责、承接方式、下级承接对象/分工、过程证据来源/AI判灯依据
4. **闭环检查表**：目标表述 / 成果层 / 衡量标准 / 举措层 / 主责主体 / 承接方式 / 集团-中心边界 / 规则沉淀 / 未决问题
5. **用户确认入口**："关闭本目标" / "返回修改指定项"

硬性约束：缺衡量标准 / 缺成果输出对象 / 缺下级承接对象/分工 → 不得进 `review_ready`。
