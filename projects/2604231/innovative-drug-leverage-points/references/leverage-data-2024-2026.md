# 杠杆点数据参考（2024-2026）

> 整理于 2026-04-23

## 一、患者旅程关键节点

### 标准流程
症状出现 → 首次就诊 → 明确诊断 → 治疗决策 → 处方开具 → 用药执行 → 疗效评估 → 继续/换药/停药

### 各节点典型转化率（行业参考）
| 节点 | 典型转化率 | 主要流失原因 |
|------|-----------|-------------|
| 症状→就诊 | 40-70% | 症状认知不足/就医门槛 |
| 就诊→确诊 | 60-80% | 误诊/检查不充分 |
| 确诊→治疗 | 70-90% | 患者拒绝/经济原因 |
| 治疗→规范用药 | 50-70% | 依从性差/不良反应 |
| 规范用药→持续治疗 | 40-60% | 疗效不佳/经济负担 |

## 二、处方决策影响因素

### 处方决策者层级
- 核心决策者：主治/主任医师
- 影响者：科室主任/药师/护理
- 最终拍板：MDT团队（肿瘤等复杂疾病）

### 处方驱动因素排序（行业调研参考）
1. 临床疗效数据（最重要）
2. 安全性/耐受性
3. 指南推荐级别
4. 患者经济承受能力
5. 同行推荐/KOL影响
6. 代表拜访/学术推广

## 三、杠杆系数评估矩阵

### 评估维度
- **影响力**（1-10分）：该节点对最终处方的影响程度
- **可变性**（1-10分）：当前行为的分散程度（越分散=可塑性越高）
- **杠杆系数** = 影响力 × 可变性

### 常见高杠杆节点
1. 诊断→治疗决策（高影响力+高可变性）
2. 一线治疗失败后换药（高影响力+中等可变性）
3. 新适应症/新人群的认知建立（中等影响力+高可变性）
