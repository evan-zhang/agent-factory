# 品牌Vision数据参考（2024-2026）

> 整理于 2026-04-23

## 一、医药行业愿景设计趋势

### 从产品到生态
- 趋势：单一产品愿景 → 疾病领域整体解决方案
- 案例：诺华从"高血压药"到"心血管健康管理"
- 驱动因素：患者全生命周期管理需求

### 从中国到全球
- 中国创新药出海成为新常态
- 2025年创新药对外授权交易额超$1300亿
- 品牌愿景需要兼顾国内和国际市场定位

### 从治疗到治愈
- 基因治疗/细胞治疗推动"治愈"叙事
- 愿景从"延长生存"向"功能性治愈"升级

## 二、愿景设计方法论

### Backcasting（逆推法）
1. 描绘10年后的终态（理想状态）
2. 识别当前与终态的差距
3. 规划5年、3年、1年的关键里程碑
4. 识别每个里程碑的前提条件和风险

### 愿景检验标准
- **挑战性**：需要努力才能实现（不是必然发生的事）
- **可理解性**：一句话说清楚
- **可衡量性**：有量化目标可追踪
- **一致性**：与公司战略/价值观一致

## 三、成功案例参考维度

| 维度 | 参考内容 |
|------|----------|
| 愿景陈述 | 简洁有力的一句话 |
| 时间跨度 | 通常5-10年 |
| 里程碑 | 3-5个关键节点 |
| 量化目标 | 市场份额/患者数/适应症数 |
| 对齐机制 | 年度OKR承接 |
