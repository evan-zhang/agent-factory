## Skill 质量原则建立（2026-03-31 22:18）

**文档位置**：`02_guides/SKILL-QUALITY-PRINCIPLES.md`

**六大原则**：
1. 单一职责（一个 Skill 一件事）
2. 三层披露（frontmatter ~100词 / SKILL.md ≤80行 / references/ 按需）
3. description 要"有点激进"（3-5 触发词，动词开头，覆盖口语）
4. 颗粒度决策树（拆/合判断标准）
5. 命令式写法 + 解释原因（不用 ALWAYS/NEVER/CRITICAL）
6. Smoke Test（正常/边界/不触发 三类）

**SOP 变更**：AF-SOP-01 新增 S5.5 质量门控节点（v4.7），发布前强制过 Checklist。

**触发规则**：开发新 Skill 或修改 Skill 时，主动对照原则检查。发布前跑完整 Checklist。
