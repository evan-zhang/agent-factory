# bp-reporting-templates 设计档案

## 产品概述
- **Slug**：bp-reporting-templates
- **当前版本**：v0.5.1
- **定位**：BP 系统绩效报告模板生成工具，支持月报/季报/半年报/年报/四套，强制代码锚点和数字可溯源审查

## 关键设计决策

### D-01：API 优先 + 文件 fallback
BP 系统 API 不稳定时，自动降级读取本地文件，保证生成不中断。

### D-02：强制两步确认（周期 + 模板类型）
防止 AI 直接生成错误周期/类型的报告，生成前必须让用户确认选择。

### D-03：v0.5.0 新增 batch_get_key_position_markdown
对应 BP 4.17 接口（批量获取关键岗位详情），替代已废弃的 4.16 接口。
