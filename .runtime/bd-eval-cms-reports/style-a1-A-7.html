第1章 执行摘要

本报告针对 Klaria Pharma Holding AB 的 Sumatriptan Alginate Film（舒马曲坦藻酸盐薄膜片）进行 CMS BD 投前评估。项目属于成熟 API + 创新薄膜剂型：舒马曲坦作为 5-HT1B/1D 激动剂已有长期临床使用基础，Klaria 的差异化在于藻酸盐薄膜递送平台、便携给药和对偏头痛发作伴恶心/呕吐患者的场景适配。

综合判断，本项目可进入下一轮商务接触与注册路径核验，但不建议直接高额买断或高首付款授权。建议以低首付选择权/独家谈判权方式推进，并将 CDE 预沟通、中国权益完整性、欧洲上市后销售数据作为付款前置条件。

阶段A 商机入池与 Phase 1 Discovery 已完成。 阶段B A-1 路由、Gate 0~5 与 Battle 审查已完成。 阶段C 进入条件式商务接触，不进入无条件立项。

第3章 市场规模与竞争格局（Gate 3）

结论先行：条件通过。 中国偏头痛患者基数大，但规范诊疗率、曲坦类处方渗透率、院外复购和患者支付意愿仍是本项目商业化成败的关键约束。舒马曲坦藻酸盐薄膜片不能按“新机制重磅药”估值，应按“成熟 API + 差异化给药剂型”的谨慎引进逻辑评估。

3.1 市场判断

中国偏头痛急性治疗存在明确未满足需求：发作场景突发、患者对快速给药和便携性有需求，恶心、呕吐或吞咽不便患者对非水吞服剂型存在潜在偏好。但曲坦类药物已是成熟品类，普通片剂、鼻喷、注射剂和新机制药物共同构成竞争压力。因此，本项目的商业机会不来自简单替代全部舒马曲坦片剂，而来自特定场景下的剂型升级与院外复购。

3.2 竞争格局

  • 传统曲坦类：舒马曲坦、佐米曲普坦、利扎曲普坦等已形成成熟竞争格局，价格和可及性对薄膜剂型形成锚定。
  • 新机制药物：CGRP 通路药物与其他新型治疗方案会抬高医生和患者对疗效、价格、支付路径的比较标准。
  • 剂型竞争：鼻喷、口崩/片剂、注射剂在不同发作场景中各有优势，薄膜剂型必须证明“便携 + 快速 + 恶心呕吐适配”的真实增量。

3.3 商业机会

  1. 头痛门诊和神经内科医生教育，可作为急性发作便利给药选择；
  2. 院外药房与线上复购场景，适合围绕便携性和场景化需求做患者教育；
  3. 对吞咽不便、恶心呕吐患者提供差异化卖点；
  4. 若价格控制合理，可作为曲坦类升级剂型，而非高价创新药路径。

3.4 关键商业风险

  • 若医保/集采压力下价格不能体现剂型溢价，峰值销售受限；
  • 若患者对薄膜剂型认知不足，推广成本上升；
  • 若中国上市窗口过晚，竞品和替代治疗进一步拥挤；
  • 若院外复购转化不足，深康主承接与院外业务中心协同价值会被削弱。

3.5 Gate 3 结论

Gate 3 维持“条件通过”。后续进入正式商务谈判前，必须补齐中国曲坦类终端规模、竞品价格带、院外复购数据和渠道可达性验证；在未完成这些数据前，不应按乐观峰值销售额支付高额不可退首付款。

第7章 Battle 对抗审查

审查层异议清单

审查层提出三项反对意见：

1. 欧洲获批不能等同于中国注册低风险；

2. 中国偏头痛药物市场大但规范用药率和复购行为不足，峰值销售可能被高估；

3. 薄膜剂型便利性明确，但商业溢价是否可持续尚未证明。

执行层逐条回应

执行层回应：

1. 将 CDE 预沟通列为付款前置条件；

2. 将峰值销售额基准下调至 1.0–1.5 亿元，并设置低首付交易结构；

3. 要求欧洲上市 6–12 个月真实销售数据作为估值校准。

### Battle 最终结论

A-1 路由成立，但结论应为“条件通过 / 谨慎推进”，不得无条件立项。

第8章 风险评估

风险

主要风险：

  • 中国注册路径不确定；
  • 中国权益完整性不确定；
  • 曲坦类成熟市场价格压力；
  • 薄膜剂型医生教育和患者支付意愿不确定；
  • 欧洲上市数据尚未充分验证商业化表现。
风险缓释：
  • 交易结构采用低首付 + 里程碑 + royalty；
  • CDE 预沟通前不支付高额不可退费用；
  • 先做中国曲坦类终端数据采样；
  • 把欧洲真实销售作为二次估值节点。
结论

综合推荐：条件通过（CONDITIONAL）。

建议 CMS 进入商务接触与注册核验阶段，但交易纪律必须严格：低首付、分阶段、权益清晰、注册可控、数据补强后再做正式立项。

第9章 问题与风险汇总

当前未发现一票否决级信息冲突，但存在三类待核实数据：

  • Klaria / CNX 协议是否影响中国权益；
  • 中国注册路径是否需要大规模临床；
  • 中国曲坦类院内/院外终端规模与价格带。
一票否决项

暂未发现一票否决项。若后续出现以下任一情况，应停止推进：

1. 中国权益已被排他性授权第三方；

2. CDE 要求大规模临床且周期超过 5 年；

3. 总现金对价超过财务纪律上限；

4. 峰值销售额无法证明超过 8,000 万元。
