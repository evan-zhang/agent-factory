# CONTEXT BASELINE（SOP-01 通用完整版）

更新时间：2026-03-28 10:48
项目：AF-20260328-002 / P20260328-002

## 一、项目目标（已确认）
创建一套“非Skill事项通用完整版治理流程”Skill（SOP-01），用于较复杂、跨步骤、跨系统、时长较长任务的规范执行与治理。

## 二、与 SOP-02 Lite 的分工（已确认）
- SOP-02 Lite：10~20分钟中小事项，轻流程。
- SOP-01 完整版：复杂事项、长链路事项、治理要求更高事项。
- 二者并存，不影响 AF-SOP-01（Skill 开发主流程）。

## 三、核心治理原则（已确认）
1. 无记录，不执行
2. 无回执，不收口
3. 无状态，不流转
4. 没记录=没发生

## 四、讨论中形成的完整版要点
1) GRV-Lite/GRV 完整思路必须在执行编排前给出
2) 执行编排必须明确：时长、方式、执行人、模型、影响、回滚
3) >2分钟任务默认子 Agent 执行
4) 支持暂停/阻塞/重启/持续推进
5) 每个实例需有独立目录和台账

## 五、实例化管理要求（已确认）
建议至少覆盖：
- state.json（状态机）
- TASK.md（任务主记录）
- EXEC-LOG.md（执行日志）
- DECISIONS.md（决策）
- HANDOVER.md（交接）
- ARTIFACTS.md（证据索引）
