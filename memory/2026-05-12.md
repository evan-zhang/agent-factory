### 技术细节
- Python 3.9 f-string `$${var}` 行为：`$$` → 字面 `$` + **对 var 求值**（不是 `$` 加字面 var 名）
- Python 3.9 f-string `${{var}}` 行为：`$` + 字面 `{var}`（不求值）
- 所以 JS 模板里需要 `${var}` 的地方，在 Python f-string 里要写 `${{var}}`
