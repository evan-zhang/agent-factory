## 新发现：抖音音频开放 API

来源：Evan 转发的 API 文档（OPEN_API---61870b01-67f3-4207-93a3-91a05b890647.md）

- **Base URL**：`https://hk-al-xg-node.mediportal.com.cn`
- **鉴权**：Bearer Token（向负责人获取）
- **推荐接口**：`POST /api/open/audio/export-with-asr`
  - 只传 `{"url": "抖音视频URL"}`
  - 返回 `aweme_id`、`is_original`、`music_title`、`asr_text`
  - 典型耗时 20-60 秒，超时设置 ≥180 秒
  - 不需要 mcporter/ffmpeg/whisper，比现有方案简单得多

**建议**：Link Archivist 的 douyin_process.py 应将此 API 作为首选，mcporter 作为降级方案。
